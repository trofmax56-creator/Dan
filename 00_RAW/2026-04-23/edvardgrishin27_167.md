ANTHROPIC ПЕРЕДЕЛАЛ ДЕСКТОПНОЕ ПРИЛОЖЕНИЕ CLAUDE CODE. И ВОТ ПОЧЕМУ ЭТО ВАЖНО.

Если вы работали в Claude Code Desktop раньше, вы знаете главную боль — одно окно, одна сессия. Хочешь параллельно вести два проекта? Открывай второе окно и прыгай между ними. Это было неудобно, и все это знали и работали в VS Сode поэтому)

Теперь Anthropic наконец сделал то, что давно напрашивалось: полный редизайн с фокусом на мультисессионность. Новый сайдбар слева показывает все активные сессии — фильтруешь по проекту, по статусу, по окружению. Когда PR мержится, сессия сама уходит в архив. Наконец-то порядок.

Что реально изменилось:
— Терминал, файловый редактор и diff viewer встроены прямо в приложение. Раньше ты постоянно переключался между Claude и VS Code — теперь можно не выходить из окна
— Все панели drag-and-drop — собираешь рабочее пространство под себя как в IDE
— Side Chat (⌘ + ;) — можно задать быстрый вопрос посреди задачи, не ломая основной контекст
— Три режима отображения: Verbose (видишь все tool calls), Normal и Summary (только результат)
— SSH теперь работает и на Mac, а не только Linux

Если сравнивать с конкурентами — Cursor 3 буквально на днях выкатил похожий подход: мультисессии, сайдбар, drag-and-drop панели. Совпадение? Не думаю. Рынок AI-IDE сейчас бежит в одну сторону — к рабочему пространству, где агенты работают параллельно, а ты управляешь ими из одного места. Ты здесь архитектор, то что я давно говорю в роликах и это делается в интерфейсе. 

Но вот что важно понимать: Claude Code Desktop — это не IDE. Это интерфейс для агента. VS Code и Cursor дают тебе редактор кода с AI-помощником. Claude Code Desktop даёт тебе AI-агента с редактором кода. Разница в подходе, и именно поэтому Anthropic пошёл в сторону управления сессиями, а не в сторону подсветки синтаксиса.

Лично я давно ждал этого обновления, потому что вести несколько проектов из одного окна — это то, чего реально не хватало и поэтому я работал в VS Code(хоть мне он и не сильно нравится). Посмотрим, как это будет работать на практике, но направление однозначно правильное. Обновление уже вышло сегодня ночью, обновляйтесь и удивляйтесь как все сильно поменялось.
