## 1. Название
**AI-Ревизор** — Контроль 100% звонков и переписок

## 2. Суть
Claude прослушивает и читает 100% коммуникаций менеджеров с клиентами, выставляет скор по чек-листу продаж, находит возражения без ответа и сигнализирует РОПу на провалы — заменяя отдел контроля качества.

## 3. Бизнес-сценарий
ДО: РОП вручную прослушивает 5–10 звонков в неделю из 200 — это 5% контроля. Менеджер Антон стабильно не закрывает возражение «дорого», но РОП узнаёт об этом через месяц по упавшей конверсии. ИИ делает: каждый звонок расшифровывается через Яндекс SpeechKit, Claude анализирует транскрипт по 12 критериям чек-листа (приветствие, выявление потребности, презентация, работа с возражениями, закрытие), выставляет оценку 1–10 и пишет конкретный комментарий: «Клиент сказал "дорого" на 3:42 — менеджер не использовал технику ценность/цена, сразу согласился на скидку.» Клиент получает: РОП каждое утро видит дашборд: топ-5 лучших и худших звонков, средний балл по каждому менеджеру, динамику за неделю.

## 4. Логика
```
Звонок/переписка → Яндекс SpeechKit (транскрипция) → n8n → Claude (анализ по чек-листу 12 критериев) → Google Sheets/Bitrix24 (дашборд) + Telegram-уведомление РОПу
```

## 6. Связки инструментов
```
UIS (запись звонка) → n8n Webhook → Яндекс SpeechKit API (транскрипция) → Claude (скоринг по промпту-чек-листу) → Bitrix24 (запись в активность сделки) + Google Sheets (сводный дашборд) + Telegram Bot (алерт РОПу при балле < 5)
```

## 7. Алгоритм реализации
1. Подключить UIS или МТС Exolve к n8n через Webhook — триггер на завершение звонка, получить ссылку на аудиофайл
2. Отправить аудио в Яндекс SpeechKit API (модель general), получить транскрипт с таймкодами и разбивкой по спикерам (менеджер / клиент)
3. Передать транскрипт в Claude с системным промптом — чек-лист из 12 критериев продаж, инструкция выставить балл 1–10 по каждому и написать конкретный комментарий с цитатой из разговора
4. Записать результат оценки в активность сделки Bitrix24 через REST API и добавить строку в Google Sheets (дата, менеджер, балл, ключевые комментарии)
5. Отправить Telegram-уведомление РОПу при общем балле < 5: включить имя менеджера, балл, 2–3 конкретных нарушения и ссылку на запись звонка

## 8. Почему купят (ROI)
ОКК из 2 сотрудников стоит 120 000 – 160 000 руб./мес ФОТ + налоги, и всё равно слушает не больше 10% звонков. AI-Ревизор стоит 15 000 руб./мес и контролирует 100% коммуникаций каждый день. Рост конверсии на 15–20% только за счёт выявления системных ошибок отбивает внедрение за первый месяц. Крючок: «Ваш ОКК слушает каждый десятый звонок. ИИ слушает все — и не берёт больничный.»
