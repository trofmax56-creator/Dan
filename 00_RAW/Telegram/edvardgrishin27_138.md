CLAUDE: ОБНОВЛЕНИЯ ЭКОСИСТЕМЫ

У Anthropic вышло сразу несколько обновлений, и если смотреть не по отдельности, а в сумме, картина очень показательная. Они двигают Claude сразу в 3 стороны:

• `чтобы он сам возвращался к задачам по расписанию
`• `чтобы вокруг него появилась экосистема инструментов
`• `чтобы вокруг него росло комьюнити и дистрибуция
`
**1. ****/loop**** в Claude Code**

Теперь в Claude Code можно запускать задачи по циклу.
Формат простой: /loop + интервал + что делать.

По сути ты говоришь Claude:
не просто ответь мне сейчас, а возвращайся к этой задаче каждые 5 минут / 30 минут / час и проверяй, что изменилось.

Типовые сценарии:
— следить за деплоем
— перепроверять тесты
— мониторить PR
— ждать завершения долгой операции
— периодически запускать один и тот же рабочий промпт

Почему это важно?
Потому что раньше Claude Code был в основном реактивным. Ты открыл, написал, получил ответ.

С /loop он уже делает первый шаг к другому поведению, не просто отвечает, а сам возвращается к задаче во времени.

Но тут важно не перепутать с полноценным облачным агентом.
/loop живет внутри текущей сессии Claude Code. Закрыл сессию — все закончилось.

**2. Scheduled Tasks в Claude Code Desktop
**
Если /loop — это цикл внутри терминальной сессии,
то [Scheduled Tasks в Claude Code Desktop](https://code.claude.com/docs/en/scheduled-tasks) — это уже нормальные задачи по расписанию через нативное приложение.

Что это дает:
— можно создать задачу через интерфейс
— выбрать prompt
— указать частоту запуска
— привязать к рабочей папке / проекту
— и Claude будет запускать ее автоматически по расписанию

Но тут тоже есть важный нюанс - это локальные scheduled tasks. Они работают, пока у тебя открыт Claude Code Desktop и компьютер не спит. То есть это не серверная автоматика 24/7, а локальный планировщик на твоей машине.

**3. Claude Marketplace**

Claude Marketplace — это уже история не про одного разработчика, а про рынок.
Anthropic позиционирует его как способ для компаний проще закупать AI-инструменты на базе Claude. Сейчас это идет в формате limited preview. Среди партнеров на старте Anthropic называет GitLab, Harvey, Lovable, Replit, Rogo и Snowflake.

Anthropic хочет быть не только компанией, у которой есть модель. Они хотят быть еще и местом, через которое enterprise покупает AI-решения.

**4. Community Ambassadors**

Anthropic запустил [программу для людей](https://claude.com/community/ambassadors), которые хотят развивать локальные Claude-комьюнити: проводить митапы, воркшопы, хакатоны, обучающие активности. В программе заявлены поддержка комьюнити, monthly API credits, swag и доступ к части внутренних активностей. Cижу вот заполняю заявку и надеюсь что попаду в ряды счастливчиков потому что за время работы с ИИ понял что стал фанатом Claude как компании, я вижу их четкий вектор развития и он мне ближе чем у других компаний. 
Потому что в целом у них читается что они делают и почему они могут выйти победителями из этой гонки. Победит не тот, у кого просто сильнее модель. Победит тот, кто первым станет привычной средой работы.
И Anthropic сейчас очень явно идет именно туда.
