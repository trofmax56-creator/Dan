# 🗂️ Паспорт идеи: ИИ-автоматизация маркетинга в Битрикс24

**Источник:** [AI в маркетинге и продажах. Александр Вартанян, Юрий Николаев. Конференция Битрикс24 «Диалог с AI»](https://youtube.com/watch?v=ocmfYrRMdSM) — Битрикс24. Для бизнеса (2023-10-11)

## 💡 Что это даёт бизнесу
Система автоматически генерирует рекламные тексты, сегментирует аудиторию и запускает кампании через Битрикс24 без участия маркетолога. Клиент сокращает время на подготовку контента с нескольких дней до часов. ИИ анализирует результаты кампаний и предлагает корректировки на основе данных CRM.

---

## ✅ Плюсы
- Прямая интеграция с Битрикс24 — нативная экосистема, знакомая целевой аудитории Студии
- Быстрый ROI: сокращение расходов на штатного маркетолога (50–120 тыс. руб./мес) окупает проект за 2–3 месяца
- Масштабируемость: одно решение можно тиражировать на десятки клиентов с минимальными доработками

## ❌ Минусы / Риски
- Ограниченность нативных AI-инструментов Битрикс24 в 2023–2024 — часть функций требует внешних API и кастомной интеграции
- Качество генерации контента зависит от промптов и обучения под конкретную нишу клиента — нужна настройка под каждый бизнес

## 🧰 Артефакты для реализации

**Инструменты:** Битрикс24 (CRM + маркетинг + автоматизация), n8n или Make.com (оркестрация процессов), OpenAI GPT-4o (генерация текстов и анализ), Битрикс24 REST API
**API:** Битрикс24 REST API (CRM, маркетинг, лиды), OpenAI Chat Completions API, ВКонтакте Ads API / Яндекс.Директ API (запуск кампаний)
