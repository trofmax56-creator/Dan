## 💡 Что это даёт бизнесу
Бот отвечает на вопросы клиентов 24/7 на основе загруженной базы знаний (прайсы, регламенты, FAQ), снижая нагрузку на сотрудников на 60–80%. Интегрируется в Telegram, WhatsApp, сайт через n8n без сложного кодинга. Клиент получает измеримый результат уже через 2–3 недели после запуска.

## 🚀 Шаги запуска
1. Шаг 1: Аудит базы знаний клиента — собрать FAQ, прайсы, регламенты в единый структурированный формат (Markdown/JSON)
2. Шаг 2: Настроить векторную базу в Supabase/Pinecone, загрузить документы через embedding-пайплайн в n8n
3. Шаг 3: Собрать RAG-цепочку в n8n: триггер сообщения → поиск по векторной БД → генерация ответа GPT-4o → отправка в канал
4. Шаг 4: Подключить к каналу клиента (Telegram/WhatsApp/виджет сайта), провести тестирование на 50+ реальных вопросах, сдать с инструкцией по обновлению базы
