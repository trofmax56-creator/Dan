# 💬 WhatsApp AI-агент: автоматические ответы клиентам 24/7

> Meta WhatsApp Cloud API + n8n + Claude/GPT → бот отвечает клиентам в WhatsApp автоматически, передаёт горячие запросы менеджеру.

## Боль клиента

- WhatsApp — основной канал коммуникации с клиентами, но ответы ручные
- В нерабочее время клиенты не получают ответа → уходят к конкурентам
- Типовые вопросы (цена, наличие, статус заказа) занимают 60% времени менеджера

## Решение

AI-агент в WhatsApp:
1. Принимает входящее сообщение через Meta WhatsApp Cloud API
2. Передаёт текст в Claude/GPT с системным промптом (FAQ компании, товары, услуги)
3. Отправляет ответ клиенту обратно в WhatsApp
4. Если вопрос сложный → создаёт задачу менеджеру в Bitrix24/amoCRM

## Технический стек

```
Meta WhatsApp Cloud API (Webhook → POST)
  → n8n Webhook Trigger
  → Code Node: парсинг entry[0].changes[0].value.messages[0]
  → HTTP Request: Claude / GPT API (системный промпт = база знаний компании)
  → HTTP Request: Meta API (отправить ответ клиенту)
  → If Node: нужен менеджер?
    → YES → HTTP Request: Bitrix24/amoCRM (создать задачу/сделку)
    → NO → завершить
```

**Критический момент:**
```js
// Правильный парсинг Meta Webhook
const msg = body.entry[0].changes[0].value.messages[0]
const text = msg.text.body
const from = msg.from  // номер клиента
```

**API токен Meta** имеет срок 60 дней — нужна автоматическая ротация (через Facebook System User).
