# 🗂️ Паспорт идеи: Универсальная n8n-автоматизация для МСБ

**Источник:** [ПОЛНЫЙ ГАЙД ПО N8N](https://youtube.com/watch?v=N4_0SHSUv0o) — MLG Frontier (2025-08-01)
**Категория:** Инфраструктура
**Заменяет:** Операционный менеджер / ручные процессы между сервисами

---

## 💡 Что это даёт бизнесу
Клиент получает единую платформу автоматизации на базе n8n, связывающую CRM, мессенджеры, таблицы и ИИ без постоянного ручного труда. Сокращается время на рутинные операции (лиды, уведомления, отчёты) на 60–80%. Студия продаёт типовые сборки воркфлоу как готовый продукт с поддержкой.

---

## ✅ Плюсы
- n8n — self-hosted, нет ограничений по числу запросов, низкая инфраструктурная стоимость для клиента (~500–2000 руб./мес на VPS)
- Огромная библиотека готовых нод: Telegram, Google Sheets, Bitrix24, amoCRM, OpenAI — быстрая сборка типовых решений
- Легко упаковать в продукт: шаблоны воркфлоу продаются повторно разным клиентам без переработки с нуля

## ❌ Минусы / Риски
- Высокая конкуренция и насыщенность рынка гайдами — клиент может попытаться сделать сам после просмотра видео, не купив у Студии
- Поддержка требует погружения: при обновлениях n8n ноды ломаются, нужен постоянный мониторинг

| Параметр | Значение |

| Поддержка/мес | 5 000 – 15 000 руб./мес |

| Потенциал выручки Студии | При 6–8 клиентах за 6 мес: 600 000 – 1 200 000 руб. (внедрения) + 300 000–500 000 руб. (поддержка накопленная база). Итого 900 000 – 1 700 000 руб. |

## 🧰 Артефакты для реализации

**Инструменты:** n8n (self-hosted или n8n Cloud), Docker / VPS (Timeweb, Reg.ru, Beget), Telegram Bot API, Bitrix24 / amoCRM Webhooks
**API:** OpenAI API (GPT-4o / Claude), Telegram Bot API, Bitrix24 REST API, amoCRM API, Google Sheets API

**Репозитории:**

## 🚀 Шаги запуска
1. Шаг 1: Упаковать 3–5 типовых воркфлоу (лиды из Telegram → CRM, авто-уведомления клиентам, еженедельный отчёт в Google Sheets) как готовые шаблоны
2. Шаг 2: Развернуть демо-стенд n8n с работающими примерами для показа клиентам на sales-звонках
3. Шаг 3: Сформировать прайс: базовый пакет (2–3 воркфлоу) + расширенный (5–7 воркфлоу + ИИ-агент) + поддержка
4. Шаг 4: Запустить лид-магнит (бесплатный аудит процессов) через Telegram-канал, конвертировать в платные внедрения
