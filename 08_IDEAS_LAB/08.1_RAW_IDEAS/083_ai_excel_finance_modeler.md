## 1. Название
**AI-Финансист** — Финансовое моделирование в Excel с Claude

## 2. Суть
Claude анализирует Excel-таблицы клиента (P&L, бюджет, cash flow) и по текстовому запросу строит финансовые модели, прогнозы и сценарии — без участия бухгалтера или финдира.

## 3. Бизнес-сценарий
«У меня есть Excel с продажами за год. Каждый раз когда собственник просит прогноз на следующий квартал — я трачу 3-4 часа на формулы. Хочу просто написать запрос и получить готовую модель.»

Типичный клиент: финансовый менеджер или бухгалтер МСБ, работающий в Excel без финансиста в штате.

## 4. Логика
```
Trigger: пользователь загружает Excel + пишет запрос
  → Claude читает структуру таблицы
  → строит формулы / сводные таблицы / сценарии
  → возвращает готовый файл или инструкцию по заполнению
Action: скачать обновлённый Excel или PDF-отчёт
```

## 5. Технический стек
- Claude claude-sonnet-4-6 (анализ структуры + генерация формул)
- Python / pandas (чтение и запись .xlsx)
- n8n (оркестратор: приём файла → Claude → отдача результата)
- Telegram Bot или веб-форма (интерфейс для клиента)

## 6. Связки инструментов
```
Telegram Bot API → n8n Webhook
  → Python node (pandas читает .xlsx)
  → Claude API (анализ + инструкция по формулам)
  → Python node (записывает результат в .xlsx)
  → Telegram Bot API (отправляет файл обратно)
```

## 7. Алгоритм реализации
1. Настроить n8n workflow: Telegram-бот принимает .xlsx файл
2. Python-нода читает листы, передаёт структуру в Claude
3. Claude строит промпт-инструкцию с готовыми формулами
4. Python-нода применяет формулы и возвращает файл
5. Бот отправляет клиенту готовый Excel

## 8. Почему купят (ROI)
- Финансовый менеджер тратит 3–5 часов в неделю на ручное моделирование
