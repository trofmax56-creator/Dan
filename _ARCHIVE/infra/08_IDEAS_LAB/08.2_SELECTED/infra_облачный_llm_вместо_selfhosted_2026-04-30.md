# 🗂️ Паспорт идеи: Облачный LLM вместо self-hosted

**Источник:** [The HARD Truth About Hosting Your Own LLMs](https://youtube.com/watch?v=EMuBqcO048E) — Cole Medin (2024-09-26)
**Категория:** Инфраструктура | ROI/Экономика
**Заменяет:** —

---

## 💡 Что это даёт бизнесу
Помогает клиентам Студии не тратить 300 000–800 000 руб. на GPU-сервер для self-hosted LLM, а использовать облачные API (OpenAI, Anthropic, YandexGPT) с оплатой за токены. Студия продаёт архитектурный консалтинг: 'что выбрать и почему', избавляя клиента от дорогостоящей ошибки. Результат — быстрый старт ИИ-решения без капитальных затрат на железо.

---

## ✅ Плюсы
- Клиент экономит от 200 000 руб. на старте — не покупает GPU-сервер, который окупается только при >50 запросов/мин
- Студия продаёт консалтинг + интеграцию облачного API как отдельную услугу с высокой маржой
- Актуально для 95% МСБ: у них нет нагрузки, оправдывающей self-hosted

## ❌ Минусы / Риски
- Клиенты с чувствительными данными (медицина, юристы, банки) всё равно потребуют self-hosted — придётся иметь отдельный оффер
- Быстро устаревает: цены на GPU падают, Ollama упрощает деплой — через 1–2 года порог входа снизится

## 💰 Экономика

| Параметр | Значение |
|---|---|
| Стоимость разработки | 15 000 – 35 000 руб. (шаблон сравнительного аудита + типовые схемы интеграции) |
| Инфраструктура/мес | 3 000 – 20 000 руб./мес (облачные API по токенам, зависит от нагрузки клиента) |
| Поддержка/мес | 10 000 – 20 000 руб./мес (мониторинг расходов на токены, обновление моделей) |
| **Цена для клиента** | **60 000 – 120 000 руб. (консалтинг + выбор стека + первичная интеграция API)** |
| Потенциал выручки Студии | За 6 мес: 5–8 аудитов × 25 000 руб. + 3–5 интеграций × 80 000 руб. = 365 000 – 600 000 руб. |

## 🧰 Артефакты для реализации

**Инструменты:** n8n (оркестрация вызовов к LLM API), LiteLLM (прокси для переключения между провайдерами), Ollama (если всё же нужен локальный деплой на скромном железе)
**API:** OpenAI API, Anthropic Claude API, YandexGPT API (для российских клиентов), GigaChat API (Сбер, для data-sensitive)

## 🚀 Шаги запуска
1. Шаг 1: Разработать калькулятор TCO (Total Cost of Ownership) — self-hosted vs облако для типовых нагрузок МСБ (Excel/Google Sheets шаблон)
2. Шаг 2: Создать пакет 'Аудит ИИ-инфраструктуры' — опросник + 2-часовая сессия + рекомендация стека, цена 25 000 руб.
3. Шаг 3: Собрать типовые интеграции (n8n + LiteLLM + amoCRM/Bitrix24) для быстрого старта клиента на облачном API
