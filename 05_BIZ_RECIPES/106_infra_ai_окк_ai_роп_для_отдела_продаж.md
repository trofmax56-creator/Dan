# 🗂️ Паспорт идеи: AI ОКК + AI РОП для отдела продаж

**Источник:** [AI ОКК И AI РОП Связка AI-решений для создания бота прослушки звонков менеджеров отдела продаж](https://youtube.com/watch?v=7lZzodC3PYg) — Маркетинг Ментор AI (2024-10-07)
**Категория:** Замена должности
**Заменяет:** Руководитель отдела продаж (частично) + Отдел контроля качества звонков

---

## 💡 Что это даёт бизнесу
Система автоматически прослушивает и транскрибирует 100% звонков менеджеров, оценивает их по чек-листу (скрипт, возражения, закрытие), формирует отчёт по каждому менеджеру и сводный дашборд для РОПа. Клиент получает объективный контроль без найма ОКК и экономит 60 000–120 000 руб./мес на ФОТ. Менеджеры знают, что каждый звонок анализируется — конверсия растёт на 15–30%.

---

## ✅ Плюсы
- Массовая боль: у каждой компании с отделом продаж (3+ менеджеров) нет сил слушать звонки вручную
- Быстрый ROI для клиента — экономия на ОКК окупает внедрение за 1–2 месяца
- Легко упаковать в типовой продукт с повторяющимися платежами (подписка на инфраструктуру)

## ❌ Минусы / Риски
- Качество транскрипции зависит от качества записи звонков в CRM — у части клиентов интеграция телефонии сырая
- Требует тонкой настройки чек-листов под каждый бизнес, что увеличивает время внедрения

## 🧰 Артефакты для реализации

**Инструменты:** n8n (оркестрация: триггер звонка → транскрипция → анализ → отчёт), Bitrix24 / amoCRM (источник записей звонков), OpenAI Whisper (транскрипция), Claude / GPT-4o (анализ по чек-листу), Google Sheets / Notion (дашборд отчётов)
**API:** OpenAI Whisper API (транскрипция), Anthropic Claude API (анализ качества), Bitrix24 REST API / amoCRM API (получение записей), Telegram Bot API (уведомления РОПу)

## 🚀 Шаги запуска
1. Шаг 1: Подключить CRM клиента через API, настроить автоматическую выгрузку новых записей звонков в n8n-воркфлоу
2. Шаг 2: Пропустить аудио через Whisper API → получить транскрипт, сохранить в структурированном виде
3. Шаг 3: Отправить транскрипт в Claude/GPT с промптом-чек-листом (приветствие, выявление потребности, презентация, работа с возражениями, закрытие) → получить оценку по каждому пункту + итоговый балл
4. Шаг 4: Агрегировать результаты в Google Sheets / дашборд, отправлять ежедневный/еженедельный отчёт РОПу в Telegram с топ-нарушениями и рейтингом менеджеров
