## 1. Название
**AI-Ревизор** — Контроль 100% звонков

## 2. Суть
Агент автоматически получает все записи звонков из телефонии, транскрибирует, оценивает по чек-листу продаж и отправляет РОПу ежедневный дайджест с оценками менеджеров, топ-ошибками и лучшими звонками дня — без ручного прослушивания.

## 3. Бизнес-сценарий
ДО: РОП говорит: «У меня 8 менеджеров, по 15 звонков в день каждый — это 120 звонков. Я физически слушаю 5–6 в неделю, всё остальное мимо. Менеджеры это знают и расслабляются». ИИ делает: каждый вечер в 19:00 n8n забирает все записи за день из UIS, транскрибирует через Яндекс SpeechKit, Claude оценивает каждый звонок по 10 критериям (приветствие, выявление потребности, отработка возражений и т.д.) и выставляет оценку 1–10. РОП получает в Telegram: «Сегодня 118 звонков. Средняя оценка: 6.2. Худший: Петров (4.1) — не отрабатывал возражения по цене в 7 из 12 звонков. Лучший: Иванова (8.7) — послушайте её звонок в 14:32». Клиент получает: 100% покрытие контроля качества без ОКК в штате.

## 4. Логика
```
UIS API (записи звонков за день) → n8n → Яндекс SpeechKit (транскрипция) → Claude (оценка по чек-листу 10 критериев) → агрегация по менеджерам → Telegram дайджест РОПу + запись в Bitrix24
```

## 5. Технический стек
- n8n
- claude-sonnet-4-7

- Яндекс SpeechKit API

- Telegram Bot API

## 6. Связки инструментов
```
UIS API GET recordings (дата = сегодня) → n8n Download audio → Яндекс SpeechKit REST (POST /speech/v1/stt, lang=ru-RU) → Claude (system: 'Ты ОКК, оцени звонок по 10 критериям, верни JSON score 1–10 + комментарий по каждому') → Bitrix24 (обновить поле оценки в сделке) → Telegram sendMessage РОПу
```

## 7. Алгоритм реализации
1. Настроить n8n Cron на 19:00: GET UIS API /calls?date=today&has_record=true — получить список ID звонков с записями, отфильтровать длительность > 60 сек (отсеять сорвавшиеся)
2. Для каждого звонка: скачать аудио (wget по URL из UIS API), отправить в Яндекс SpeechKit POST /speech/v1/stt (формат mp3, lang=ru-RU, profanityFilter=false) — получить текст транскрипта
3. Транскрипт передать в Claude с системным промптом-чек-листом: 10 критериев (приветствие по скрипту, представление компании, выявление потребности open-вопросом, презентация по выгодам, отработка возражений, попытка закрытия, следующий шаг, прощание, запрет стоп-слов, тональность) — Claude возвращает JSON: {scores: {}, total: N, highlights: '', red_flags: ''}
4. Агрегировать результаты по менеджерам: рассчитать среднюю оценку, количество звонков, топ-3 ошибки; обновить кастомное поле 'Оценка ОКК' в карточке сделки Bitrix24 через REST API
5. Сформировать и отправить Telegram-сообщение РОПу: заголовок с датой и общим числом звонков + таблица менеджеров (имя / кол-во звонков / средняя оценка / главная ошибка) + ссылка на лучший звонок дня + ссылка на худший звонок дня; при оценке менеджера < 5.0 — дополнительный алерт с пометкой ⚠️

## 8. Почему купят (ROI)
Штат ОКК из 2 специалистов = 140 000–180 000 руб./мес ФОТ + налоги. При этом они покрывают 15–20% звонков и часто избегают конфликтов с менеджерами. AI-Ревизор покрывает 100% звонков за 8 000–12 000 руб./мес и не боится Петрова. Возврат инвестиций — за 3 недели работы. Крючок: «Менеджер знает, что его слушают все 120 звонков — и сам перестаёт расслабляться».
