# 🏭 Проект: Завод ИИ-решений (AI Factory Setup)

## 🎯 Цель проекта
Превратить Студию Трофимов из интегратора CRM в конвейер комплексных ИИ-продуктов для бизнеса.

---

## 🏗️ Структура "Завода" (4 Цеха)

### 1. Цех Маркетинга (Упаковка и смыслы)
- **Инструменты:**
  - *Claude 3.5:* Генерация экспертных статей из голосовых заметок Максима.
  - *HeyGen API:* Создание рекламных Reels с цифровым аватаром.
  - *Midjourney:* Визуализация ИИ-продуктов.
- **Результат:** Постоянный поток лидов на "CRM + ИИ".

### 2. Цех Продаж (AI-Sales Booster)
- **Инструменты:**
  - *Собственная разработка:* Квалификатор + Реаниматор.
  - *n8n:* Автоматизация партнерской воронки.
- **Результат:** Рост партнерской сети и прямых продаж на автомате.

### 3. Цех Сборки (Low-code разработка)
- **Роль:** n8n Инженер / Архитектор систем.
- **Инструменты:**
  - *Self-hosted n8n:* Для контроля данных и безопасности.
  - *Docker:* Развертывание локальных моделей (Ollama / DeepSeek).
  - *GitHub "Dan":* Библиотека чертежей и готовых воркфлоу для перепродажи.
- **Результат:** Работающий ИИ-продукт у клиента за 1-2 недели.

### 4. Цех Сервиса (LTV и Обучение)
- **Инструменты:**
  - *AI-Наставник по CRM:* Авто-поддержка клиентов.
  - *AI-Ловец Глюков:* Проактивный мониторинг API.
- **Результат:** Нулевой отток клиентов (Churn 0%).

---

## 🛠️ Первоочередная инфраструктура
1. **Dedicated Server:** Для развертывания n8n и локальных БД.
2. **API Keys Wallet:** Централизованное управление ключами (Anthropic, OpenAI, Dadata, Kontur).
3. **Template Library:** Папка `05_BIZ_RECIPES` в Obsidian должна стать "Каталогом продуктов".
