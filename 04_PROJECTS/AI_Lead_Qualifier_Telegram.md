# 🤖 AI-Квалификатор лидов в Telegram

> n8n + Telegram Bot + OpenAI/Claude + Google Sheets → бот автоматически квалифицирует входящие лиды и заполняет таблицу данными без участия менеджера.

## Боль клиента

- Менеджеры тратят 15–30 мин на квалификацию одного лида вручную
- В нерабочее время лиды теряются или остывают
- CRM заполняется неполными данными — менеджеры ленятся

## Решение

Telegram-бот задаёт клиенту 3–5 вопросов (бюджет, задача, срок), отправляет ответы в OpenAI/Claude для оценки качества лида, результат пишет в Google Sheets и уведомляет менеджера только по горячим контактам.

## Технический стек

```
Telegram Bot API (BotFather)
  → n8n Telegram Trigger
  → Code Node: форматирование данных
  → HTTP Request: OpenAI / Claude API (оценка лида, JSON ответ)
  → Google Sheets: запись лида с оценкой
  → Telegram Message: уведомление менеджеру (только HIGH)
```
