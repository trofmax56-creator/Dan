## Суть
Создание AI-агента в Telegram с использованием n8n, OpenAI и Google Sheets для автоматизации продаж и квалификации лидов. Агент автоматически обрабатывает входящие сообщения от потенциальных клиентов, проводит их квалификацию через API OpenAI и сохраняет результаты в Google Sheets для последующей обработки отделом продаж.

## Бизнес-сценарий
Компании, занимающиеся B2B/B2C продажами, используют этот AI-агента для автоматического общения с потенциальными клиентами в Telegram, быстрой их квалификации (определение уровня заинтересованности, бюджета, сроков), и заполнения CRM (Google Sheets) готовыми данными о лидах. Это экономит время менеджеров и ускоряет цикл продаж.

## Алгоритм реализации
1. Шаг 1: Создание Telegram бота через BotFather (@BotFather) и получение токена бота (API token). Сохранить токен для использования в n8n.
2. Шаг 2: Регистрация на n8n.cloud (или развёртывание self-hosted версии) и создание нового workflow. Инициализировать триггер 'Telegram Trigger Node' с использованием полученного токена Telegram бота.
3. Шаг 3: Настройка начального сообщения бота. Добавить ноду 'Telegram Send Message' для отправки приветственного сообщения ('Здравствуйте! Я AI-помощник по продажам. Расскажите о вашем проекте/потребности') при получении первого сообщения от пользователя.
4. Шаг 4: Добавление ноды 'OpenAI' (ChatGPT) для обработки сообщения пользователя. Настроить промпт для квалификации: 'Ты — менеджер по продажам. Проанализируй сообщение клиента и определи: 1) готовность к покупке (0-100%), 2) бюджет (если указан), 3) сроки (если указаны), 4) основную боль/проблему'. Использовать модель 'gpt-4' или 'gpt-3.5-turbo' в зависимости от требований.
5. Шаг 5: Добавление условной логики (IF node или Switch node) для анализа ответа OpenAI. Если уровень готовности > 70%, направить лид в горячие, если 30-70% — в теплые, если < 30% — в холодные.
6. Шаг 6: Интеграция с Google Sheets. Добавить ноду 'Google Sheets' для создания новой строки в таблице с полями: 'Telegram ID', 'Имя', 'Сообщение', 'Уровень готовности', 'Бюджет', 'Сроки', 'Проблема', 'Дата/время', 'Статус лида'.
7. Шаг 7: Отправка автоматического ответа пользователю через 'Telegram Send Message' с сообщением благодарности и информацией о следующих шагах ('Спасибо за информацию! Менеджер свяжется с вами в течение 2 часов').
8. Шаг 8: Тестирование workflow. Активировать нод и протестировать отправкой сообщений в Telegram бот. Проверить, что данные корректно сохраняются в Google Sheets и анализируются OpenAI.
9. Шаг 9 (опционально): Добавить историю диалога. Создать ноду для сохранения всех сообщений чата в отдельной таблице Google Sheets для анализа и улучшения промпта. Использовать 'Execute Query' для получения истории из базы или локального хранилища n8n.
10. Шаг 10: Настройка уведомлений менеджерам. Добавить ноду для отправки горячих лидов (готовность > 70%) в отдельный Telegram канал или на email через SMTP node с приложением ссылки на Google Sheets.

## Технический стек
- n8n — платформа для создания workflow автоматизации (облачная версия n8n.cloud или self-hosted)
- OpenAI API (ChatGPT) — модели gpt-4 или gpt-3.5-turbo для анализа и квалификации лидов
- Telegram Bot API — для отправки и получения сообщений через бота
- Google Sheets API — для сохранения и структурирования данных о лидах
- Webhook или Polling — для получения обновлений от Telegram
- JSON обработка — для парсинга ответов от OpenAI и Telegram
- HTTP запросы — для связи между n8n и внешними API
- JavaScript (в n8n Function nodes) — для преобразования данных и логики

## Связки инструментов
- Telegram Webhook / Polling → n8n Telegram Trigger Node → n8n Switch/IF Node (логика квалификации) → OpenAI API (анализ текста) → n8n Google Sheets Node (сохранение) → Telegram Send Message (ответ клиенту)
- Поток данных: Клиент пишет в Telegram → Webhook уведомляет n8n → n8n передаёт текст в OpenAI → OpenAI возвращает анализ → n8n сохраняет в Google Sheets → n8n отправляет ответ в Telegram
- Архитектура: Telegram Bot ←→ n8n Workflow ←→ OpenAI LLM + Google Sheets DB ←→ Email/Telegram уведомления менеджерам
