❌ **Конечно легко во всём запутаться...** столько кругом этих ИИ Агентов выходит. Разбираемся:

🗣 **ClaudeClaw бот –** тот самый, о котором я снимаю свои видео. По сути это обычный Claude Code, размещённый на сервере, но подключенный к Telegram. Важно! Это не модный OpenClaw бот!

**Работает через подписку Claude на тарифах Pro и Max.**

🗣 **OpenClaw –** это нашумевший ИИ бот, с которого началась истерия по ИИ агентам. Работает через API различных LLM моделей (Gemini, ChatGPT, Claude или Kimi). Не работает через подписки.

Можно установить на свой компьютер и использовать локальные LLM модели, но делать это не рекомендуется. Он получает доступ ко всему и хрен знает как себя поведёт.

❌ Использовать через **API выходит** капец как дорого на нормальных моделях.

🗣 **Claude CoWork –** живёт на Вашем компьютере, запускается вручную после **предоставления доступа** к папке. Нужен ВИПИтот самый Н, чтобы работал на компьютере, а не браузере.

Работает через подписку Claude на тарифах Pro и Max.

🗣 **Claude Code –** это обычный Claude, которому дали доступ к терминалу.

То есть он может: читать файлы, редактировать код, ходить по сайтам, запускать команды, подключать инструменты. А уже сверху люди **прикручивают Telegram** (и называют его ClaudeClaw, чтобы всех запутать), интерфейсы, автоматизации и делают из него ИИ-агентов.

Ещё есть **Codex от ChatGPT** – аналогичная идея: дать модели доступ к коду, файлам и командам. Правда работает он внутри экосистемы OpenAI и не так легко превращается в Telegram-ботов и всякие кастомные агенты.

**Иногда кажется, что Codex ребята из OpenAI держат в клетке и не выпускают в люди.**

🗣 **Gemini CLI от Google –** по сути это попытка Google сделать свой аналог Claude Code. Только с сильным уклоном в интеграцию с Gmail, Drive, Docs, Sheets и другими сервисами Google.

✅ Но при этом ребята из **Google сделали** интересный ход — открыли свою **экосистему Workspace** как набор инструментов для AI-агентов. Во вчерашнем видео как раз подробно рассказал, как подключить эту историю к Claude Code.
