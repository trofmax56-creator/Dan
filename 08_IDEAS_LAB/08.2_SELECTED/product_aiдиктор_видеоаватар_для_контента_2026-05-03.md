## 1. Название
**AI-Диктор** — Видео-аватар для контента

## 2. Суть
Система автоматически превращает текстовый контент (статья, новость, описание товара) в видеоролик с говорящим аватаром — без съёмки, без диктора, без монтажёра. Интегрируется в контент-конвейер и публикует видео на автопилоте.

## 3. Бизнес-сценарий
ДО: Владелец сети автосервисов говорит: «Хочу вести YouTube и Reels, но на съёмку нет ни времени, ни бюджета на оператора». Конкуренты уже ведут каналы — он теряет аудиторию. ИИ делает: менеджер пишет в таблицу тему «Как проверить тормозные колодки самому» — через 15 минут n8n вызывает HeyGen CLI, генерирует 90-секундный ролик с аватаром директора, накладывает субтитры, публикует в VK Видео и Telegram. Клиент получает: 20 видео в месяц без единого съёмочного дня.

## 4. Логика
```
Google Sheets (тема) → n8n → Claude (сценарий + текст для аватара) → HeyGen CLI (рендер видео) → VK Video API + Telegram → уведомление в Telegram
```

## 6. Связки инструментов
```
Google Sheets триггер → n8n → Claude (генерация сценария 90 сек) → HeyGen CLI JSON → polling статуса рендера → скачать mp4 → VK Video API upload + Telegram sendVideo
```

## 7. Алгоритм реализации
1. Развернуть HeyGen CLI на VPS (Ubuntu 22.04): npm install -g heygen-cli, прописать API-ключ HeyGen v3, проверить командой heygen avatar list — получить avatar_id клиента
2. В n8n создать триггер на новую строку в Google Sheets (статус = 'В работу'), передать тему в Claude с промптом: 'Напиши сценарий 90 секунд, 180–200 слов, разговорный стиль, без списков'
3. n8n Execute Command: heygen video create --avatar {avatar_id} --text '{script}' --voice ru-RU --output json — получить video_id, запустить polling каждые 30 сек до статуса 'completed'
4. Скачать mp4 через wget по ссылке из JSON-ответа HeyGen, загрузить в VK Video API (метод video.save + video.upload) и отправить в Telegram-канал через sendVideo
5. Обновить статус в Google Sheets на 'Опубликовано', отправить директору уведомление в Telegram с ссылками на VK и Telegram-канал

## 8. Почему купят (ROI)
Видеопродакшн руками: оператор + монтаж + диктор = от 15 000 руб./ролик × 20 роликов = 300 000 руб./мес. AI-Диктор производит те же 20 роликов за стоимость API ~12 000–18 000 руб./мес. Разница — 282 000 руб. ежемесячно. Крючок: «Ваш директор снимается 24/7 и никогда не устаёт».
