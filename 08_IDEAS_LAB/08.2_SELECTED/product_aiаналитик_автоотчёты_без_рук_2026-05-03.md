## 1. Название
**AI-Аналитик** — Автоотчёты без рук

## 2. Суть
Агент автоматически собирает данные из Яндекс.Метрики, CRM и рекламных кабинетов, формирует еженедельный отчёт на русском языке с интерпретацией цифр и рекомендациями — и отправляет директору в Telegram без участия маркетолога.

## 3. Бизнес-сценарий
ДО: Собственник говорит аналитику: «Покажи, как отработала прошлая неделя». Аналитик тратит 3–4 часа: выгружает из Метрики, сводит с amoCRM, считает CPL, пишет текст. К пятнице отчёт устаревает. ИИ делает: каждый понедельник в 08:00 n8n сам вытаскивает данные из всех источников, Claude пишет: «На этой неделе конверсия выросла на 12% — лучше всего отработала кампания X, рекомендую увеличить бюджет на 20%». Клиент получает: готовое решение в Telegram до первого кофе.

## 4. Логика
```
Cron (понедельник 08:00) → n8n параллельно вызывает Яндекс.Метрика API + amoCRM API + Яндекс.Директ API → Claude (интерпретация + рекомендации) → Telegram сообщение директору
```

## 6. Связки инструментов
```
n8n Cron → параллельные HTTP-запросы (Метрика + amoCRM + Директ) → Merge node → Claude (промпт: аналитик-маркетолог, интерпретируй данные, дай 3 рекомендации) → Telegram sendMessage + запись в Google Sheets
```

## 7. Алгоритм реализации
1. Настроить 3 параллельных HTTP-узла в n8n: Яндекс.Метрика API (визиты, конверсии, источники за 7 дней), amoCRM API (новые лиды, сделки, сумма за 7 дней), Яндекс.Директ API (расходы, клики, CTR по кампаниям)
2. Merge-узел объединяет все данные в единый JSON-объект, передать в Claude с системным промптом: 'Ты маркетинговый аналитик. Проанализируй данные, найди 3 главных инсайта, дай 3 конкретных рекомендации на следующую неделю. Пиши кратко, по-русски, для директора без технического background'
3. Если Claude выявил аномалию (падение трафика >20% или рост CPL >30%) — дополнительный Telegram-алерт с пометкой 🚨 вне расписания, через отдельную ветку IF-узла
4. Форматировать итоговое сообщение в Telegram: заголовок недели + 5 ключевых метрик с дельтой (▲/▼%) + блок инсайтов + блок рекомендаций + ссылка на Google Sheets с деталями
5. Записать сырые данные и текст отчёта в Google Sheets (новая строка каждую неделю) для накопления истории и последующего сравнения периодов

## 8. Почему купят (ROI)
Аналитик на аутсорсе или штатный junior — от 50 000 руб./мес, при этом отчёт раз в неделю, без алертов и всегда с задержкой. AI-Аналитик даёт отчёт каждый понедельник в 08:00 + мгновенные алерты за 4 000–6 000 руб./мес. Директор принимает решения на свежих данных, а не на прошлонедельных. Крючок: «Данные — это не Excel, данные — это решение в Telegram до кофе».
