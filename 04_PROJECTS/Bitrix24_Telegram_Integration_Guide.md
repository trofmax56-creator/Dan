# 📱 Telegram + Bitrix24: выбор и настройка интеграции

> Практический гайд + услуга внедрения: встроенная интеграция vs Wazzup vs ChatApp — подбор оптимального решения под задачи клиента.

## Боль клиента

- Клиенты пишут в Telegram, менеджеры отвечают вручную без фиксации в CRM
- История переписки не попадает в сделку — теряется контекст
- Непонятно, что выбрать: бесплатная встроенная vs платные Wazzup/ChatApp

## Продукт: консалтинг + внедрение

Три варианта для клиента:

| Вариант | Стоимость | Когда выбрать |
|---|---|---|
| Встроенная интеграция Bitrix24 | Бесплатно (наш труд) | Малый бизнес, минимум функций |
| Wazzup | ₽3 990/мес (+ наш труд) | Нужны шаблоны, аналитика |
| ChatApp | ₽4 500/мес (+ наш труд) | Группы, маркетинг, рассылки |

**Наша услуга:** аудит задач клиента + рекомендация + внедрение выбранного решения.

## Технический стек

**Встроенная (бесплатная):**
```
BotFather → Telegram Bot Token
  → Bitrix24 Администратор → Интеграции → Telegram Bot
  → Webhook → Bitrix24 открытые линии
```

**Wazzup / ChatApp:**
```
Wazzup/ChatApp аккаунт
  → API ключ → Bitrix24 маркетплейс-приложение
  → Настройка маппинга: чаты → сделки/контакты
  → Шаблоны сообщений + аналитика
```
