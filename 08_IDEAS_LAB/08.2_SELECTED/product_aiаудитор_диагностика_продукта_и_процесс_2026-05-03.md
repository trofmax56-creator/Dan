## 1. Название
**AI-Аудитор** — Диагностика Продукта и Процессов за 30 Минут

## 2. Суть
Claude-агент по структурированному опроснику собирает данные о продукте, процессах продаж и клиентском пути, анализирует слабые места по 4 категориям и выдаёт приоритизированный отчёт с конкретными рекомендациями — заменяет дорогостоящий консалтинг на первичном этапе.

## 3. Бизнес-сценарий
ДО: Владелец оптовой компании Дмитрий платит консультанту 80 000 руб. за «аудит бизнеса». Консультант неделю собирает информацию через интервью, ещё неделю пишет отчёт на 40 страниц — половина из которых общие слова. Итог: 3 недели и 80 000 руб. за 5 реально полезных страниц. ИИ ДЕЛАЕТ: Дмитрий открывает Telegram-бот и отвечает на 25 вопросов (занимает 20–25 минут): о воронке продаж, конверсиях, среднем чеке, текучке, узких местах. Агент запрашивает 2–3 ключевых документа (прайс, скрипт, оргсхема). Claude анализирует ответы и документы по 4 блокам: стратегия роста, операционные узкие места, риски, быстрые победы. КЛИЕНТ ПОЛУЧАЕТ через 30 минут: PDF-отчёт на 8–12 страниц с топ-5 проблемами, ранжированными по ROI потенциалу, и конкретными шагами по каждой.

## 4. Логика
```
Telegram-бот (25 вопросов + загрузка документов) → n8n (сборка контекста) → Claude (анализ по 4 категориям: стратегия / операции / риски / быстрые победы) → генерация PDF-отчёта → доставка клиенту в Telegram + email
```

## 6. Связки инструментов
```
Telegram Bot (опросник 25 вопросов + file upload) → n8n (агрегация ответов в structured JSON) → Claude Opus (4-блоковый анализ, ~3000 токенов вход, ~2000 выход) → n8n (генерация HTML-отчёта) → pdfmonkey API (рендер PDF) → Telegram Bot (отправка PDF) + email через SMTP
```

## 7. Алгоритм реализации
1. Создать Telegram-бота с пошаговым опросником: 25 вопросов разбить на 5 блоков по 5 вопросов (воронка / команда / продукт / конкуренты / цели), прогресс-бар «3/5», отмена и возврат к предыдущему вопросу; поддержка загрузки до 3 файлов (PDF, DOCX)
2. n8n агрегирует все ответы в структурированный JSON: {company_name, revenue_range, team_size, funnel_stages: [...], key_problems: [...], docs: [...]} и передаёт в Claude
3. Claude-промпт (роль: партнёр McKinsey с опытом МСБ России): анализ по 4 категориям, для каждой — 2–3 проблемы с оценкой критичности (1–5) и потенциальным ROI в рублях; промпт содержит 15 эталонных паттернов проблем МСБ
4. n8n собирает HTML-шаблон отчёта: обложка с названием компании, 4 раздела с таблицами и графиками, итоговая страница «Топ-5 шагов на ближайшие 30 дней» → pdfmonkey рендерит PDF ≤ 3 минуты
5. Отправить PDF клиенту в Telegram + на email (Dadata валидирует email при вводе); через 3 дня автоматическое сообщение: «Дмитрий, посмотрели отчёт? Есть вопросы по пункту 3 про воронку?» — разогрев к продаже следующего продукта Студии

## 8. Почему купят (ROI)
Первичный консалтинговый аудит у агентств — 50 000–150 000 руб. и 2–3 недели ожидания. AI-Аудитор даёт результат за 30 минут. Для Студии Трофимов это идеальный продукт-входная точка: клиент платит 15 000–25 000 руб. за аудит → получает отчёт с 5 проблемами → 3 из 5 решаются продуктами Студии → средний апсейл 120 000–250 000 руб. Крючок: «Лучший способ продать автоматизацию — сначала показать клиенту, сколько он теряет без неё.»
