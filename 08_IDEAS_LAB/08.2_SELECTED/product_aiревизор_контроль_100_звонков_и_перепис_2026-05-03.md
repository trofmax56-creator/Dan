## 1. Название
**AI-Ревизор** — Контроль 100% звонков и переписки

## 2. Суть
Агент автоматически анализирует все записи звонков и переписку менеджеров через Claude, оценивает качество по чек-листу продаж, выявляет потерянные возможности и еженедельно формирует отчёт РОПу с конкретными кейсами для разбора.

## 3. Бизнес-сценарий
ДО: РОП слушает 5 звонков из 200 в неделю — наугад. «Ну вроде всё нормально говорят». В это время 3 менеджера не делают повторное предложение после возражения «дорого», теряя 30% сделок. Узнаёт об этом раз в квартал на отчёте. ИИ делает: UIS/МТС Exolve передаёт транскрипт звонка в n8n через webhook. Claude читает текст и оценивает по 12 пунктам чек-листа (приветствие, выявление потребности, презентация, работа с возражением «дорого», призыв к действию и т.д.), ставит оценку 1–5, фиксирует конкретную цитату-нарушение. Менеджер с оценкой < 3 получает уведомление в Telegram с цитатой и рекомендацией. РОП в пятницу получает сводный отчёт: топ-3 проблемы недели, лучший и худший звонок. Клиент получает: системный контроль качества без найма тренера.

## 4. Логика
```
UIS/МТС Exolve Webhook (транскрипт звонка) → n8n → Claude (оценка по 12 пунктам чек-листа) → Bitrix24 API (запись оценки в карточку звонка) → Telegram Bot (алерт при оценке < 3) → Google Sheets (накопление для еженедельного отчёта)
```

```
UIS Webhook → n8n → Claude API (scoring) → Bitrix24 REST API (activity update) → Telegram Bot → Google Sheets API (weekly report)
```

## 7. Алгоритм реализации
1. Подключить UIS или МТС Exolve к n8n через webhook: получать транскрипт звонка, ID менеджера, длительность, телефон клиента после каждого завершённого звонка
2. Передать транскрипт в Claude с системным промптом чек-листа из 12 пунктов (адаптированного под скрипт продаж клиента); получить JSON: {point_id, score_1_to_5, quote, recommendation} для каждого пункта
3. Вычислить итоговый скор звонка (среднее по 12 пунктам), записать в кастомное поле активности Bitrix24 через REST API crm.activity.update
4. При итоговом скоре < 3: отправить менеджеру Telegram-сообщение с цитатой нарушения и конкретной рекомендацией (текст генерирует Claude)
5. Записать строку в Google Sheets: дата, менеджер, скор, топ-3 проблемы, ссылка на звонок в Bitrix24
6. Настроить n8n Schedule-триггер на каждую пятницу 17:00: агрегировать данные из Google Sheets за неделю, Claude формирует текстовый отчёт РОПу (топ-3 системные проблемы, лучший звонок, худший звонок), отправить в Telegram РОПа

## 8. Почему купят (ROI)
Штат ОКК из 2 человек для прослушки звонков — 120 000–160 000 руб./мес ФОТ. При этом они физически охватывают 10–15% звонков. AI-Ревизор анализирует 100% звонков за 8 000 руб./мес инфраструктуры. Компании, внедрившие системный контроль звонков, фиксируют рост конверсии на 15–25% за 2 месяца. Крючок: «ОКК из двух человек видит каждый десятый звонок. AI-Ревизор видит все двести.»
