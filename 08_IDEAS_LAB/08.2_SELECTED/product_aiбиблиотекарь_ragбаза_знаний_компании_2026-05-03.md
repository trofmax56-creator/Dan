## 1. Название
**AI-Библиотекарь** — RAG-база знаний компании

## 2. Суть
Корпоративный чат-бот на базе RAG (retrieval-augmented generation) отвечает на вопросы сотрудников и клиентов, опираясь строго на документы компании — прайсы, регламенты, договоры, инструкции — без галлюцинаций.

## 3. Бизнес-сценарий
ДО: Менеджер новой строительной компании «ПромСтрой» 10 раз в день спрашивает у коллег: «А какая скидка действует для оптовиков от 500 000?», «Какой срок гарантии на монтаж?», «Есть ли сертификат на этот материал?». Старожилы отвлекаются, новички боятся ошибиться с клиентом. ИИ делает: менеджер пишет в корпоративный Telegram-бот: «Какая скидка для оптовиков от 500 000 рублей?». AI-Библиотекарь за 4 секунды находит нужный раздел в прайс-листе и регламенте, отвечает: «Для оптовых клиентов от 500 000 руб. действует скидка 12% согласно прайсу v2.3 от 01.03.2025. Источник: файл price_wholesale.pdf, стр. 4». Клиент получает: точный ответ со ссылкой на источник, менеджер не боится ошибиться.

## 4. Логика
```
Telegram-бот (вопрос) → n8n → vector search (Qdrant/Supabase) → Claude (формулировка ответа с цитатой) → Telegram-бот (ответ со ссылкой на документ)
```

## 5. Технический стек
- n8n
- claude-sonnet
- Qdrant
- Supabase
- Telegram Bot API

## 6. Связки инструментов
```
Telegram Bot → n8n Webhook → Qdrant vector search (top-3 chunks) → Claude API (ответ на основе контекста) → Telegram Bot (ответ + источник)
```

## 7. Алгоритм реализации
1. Загрузить документы компании (прайсы, регламенты, FAQ) в n8n, разбить на чанки по 500 токенов с перекрытием 50 токенов
2. Создать векторную базу в Qdrant (self-hosted) или Supabase pgvector, проиндексировать все документы через embeddings-модель
3. Настроить Telegram-бот через BotFather, подключить к n8n Webhook на входящие сообщения
4. При каждом вопросе выполнить vector search: извлечь top-3 наиболее релевантных чанка с метаданными (имя файла, страница)
5. Передать вопрос + 3 чанка в Claude с промптом: отвечай только на основе предоставленных документов, цитируй источник, если ответа нет — сообщи об этом честно
6. Логировать все вопросы без ответа в Google Sheets для еженедельного обновления базы знаний

## 8. Почему купят (ROI)
Адаптация нового менеджера занимает 2–4 недели, в это время он ошибается с клиентами и переспрашивает коллег ~15 раз в день. Каждая ошибка в цене или условиях — потенциальный конфликт или потеря сделки. Компания из 5 менеджеров тратит около 10 часов в неделю на внутренние справки. AI-Библиотекарь даёт точный ответ за 4 секунды, новый сотрудник выходит в полную силу за 3 дня вместо 3 недель. Крючок: «Ваши регламенты написаны — пусть их хоть кто-то читает.»
