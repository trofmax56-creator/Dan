# 🗂️ Паспорт идеи: AI-квалификатор лидов в Битрикс24

**Источник:** [Как интегратор использует Битрикс24 для автоматизации бизнеса](https://youtube.com/watch?v=utYd37i7OiI) — Пуд Соли. Интервью, бизнес и автоматизация (2026-02-10)
**Категория:** Замена должности
**Заменяет:** Менеджер по первичной квалификации лидов / SDR

---

## 💡 Что это даёт бизнесу
ИИ-агент автоматически обрабатывает входящие заявки в Битрикс24: задаёт уточняющие вопросы через чат/звонок, присваивает скор лиду и переводит сделку на нужный этап воронки без участия человека. Клиент экономит 1–2 ставки менеджеров и сокращает время реакции на лид с часов до секунд. Отдел продаж получает только тёплые, уже квалифицированные контакты с заполненными полями CRM.

---

## ✅ Плюсы
- Мгновенная реакция 24/7 — лид не остывает, конверсия растёт на 20–40%
- Нативная интеграция с Битрикс24 через REST API и роботов — минимум кастомной инфраструктуры
- Легко тиражируется: один шаблон продаётся десяткам клиентов из разных ниш

## ❌ Минусы / Риски
- Качество квалификации зависит от чёткости скриптов — клиент должен формализовать критерии ICP, иначе точность низкая
- Битрикс24 периодически меняет API и логику роботов — нужна поддержка и регрессионное тестирование

## 🧰 Артефакты для реализации

**Инструменты:** Битрикс24 (роботы, REST API, открытые линии), n8n или Make.com, Claude API / OpenAI GPT-4o, Битрикс24 Бизнес-процессы
**API:** Битрикс24 REST API (crm.lead.*, crm.deal.*), Anthropic Claude API, OpenAI Chat Completions API, Битрикс24 Webhook API

## 🚀 Шаги запуска
1. Шаг 1: Провести CustDev с 3–5 клиентами на Битрикс24 — зафиксировать критерии квалификации лида (бюджет, срок, ЛПР, ниша)
2. Шаг 2: Собрать прототип на n8n — триггер на новый лид → запрос к Claude с промптом квалификации → запись скора и тегов обратно в CRM
3. Шаг 3: Подключить диалоговый сценарий (чат через открытые линии Битрикс24 или WhatsApp) для дозапроса недостающих данных
4. Шаг 4: Упаковать в тиражный продукт — документация, видео-онбординг, шаблон промпта под 3 типовые ниши (недвижимость, B2B-услуги, e-commerce)
