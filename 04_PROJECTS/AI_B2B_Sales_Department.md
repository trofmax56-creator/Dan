# 🚀 AI B2B Отдел продаж: автоматическая генерация и квалификация лидов

> Google Sheets (ЦА) → n8n → RocketReach/Hunter.io → Claude API → персональные письма → CRM — без живых SDR.

## Боль клиента

- Найм SDR/sales-менеджера стоит ₽80–150k/мес + 2–4 месяца адаптации
- Холодный аутрич через людей — медленный, непоследовательный, дорогой
- Масштабирование ограничено: нанял 10 SDR — получил 10x расходов

## Решение

Автоматизированная система:
1. Берёт список целевых компаний из Google Sheets
2. Через RocketReach/Hunter.io находит контакт Decision Maker
3. Claude генерирует персонализированное письмо под компанию
4. Отправляет через Email / LinkedIn
5. Горячие ответы автоматически создаются как сделки в CRM (amoCRM/Bitrix24)

Результат: 100+ персонализированных касаний в день без человека.

## Технический стек

```
Google Sheets (список ЦА-компаний)
  → n8n Schedule Trigger (утро)
  → HTTP Request: RocketReach API (найти email Decision Maker)
  → HTTP Request: Claude API (сгенерировать персональное письмо)
  → HTTP Request: SendGrid / Gmail API (отправить письмо)
  → Webhook (отслеживание открытий + ответов)
  → If Node: ответ получен?
    → YES → HTTP Request: amoCRM / Bitrix24 (создать сделку)
    → NO → Schedule: follow-up через 3 дня
```

## Важные ограничения

- Шаблоны писем в Gmail/SendGrid должны быть верифицированы во избежание спам-фильтра
