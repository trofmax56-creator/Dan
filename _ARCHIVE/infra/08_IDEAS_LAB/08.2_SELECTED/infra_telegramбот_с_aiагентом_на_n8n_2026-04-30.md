## 💡 Что это даёт бизнесу
Бизнес получает круглосуточного AI-ассистента в Telegram, который отвечает на вопросы клиентов, квалифицирует лиды и передаёт сложные кейсы живому менеджеру. Сокращает нагрузку на поддержку на 60–80% без найма дополнительного персонала. Интегрируется с CRM (Bitrix24/amoCRM) для автоматической фиксации обращений.

---

## ✅ Плюсы
- Telegram — самый популярный канал коммуникации МСБ в России, низкий порог входа для клиента
- n8n позволяет быстро собрать MVP за 1–2 недели без написания кода с нуля
- Легко масштабируется: добавляются новые сценарии, интеграции с базами знаний, голосом

## ❌ Минусы / Риски
- Качество ответов зависит от качества базы знаний клиента — нужна подготовка данных, что удлиняет проект
- Клиенты МСБ часто недооценивают продукт и торгуются по цене, сравнивая с готовыми дешёвыми чат-ботами

## 🚀 Шаги запуска
1. Шаг 1: Собрать базу знаний клиента (FAQ, скрипты, прайс) и загрузить в векторное хранилище (Pinecone / Supabase pgvector)
2. Шаг 2: Настроить n8n-воркфлоу: Telegram Trigger → AI Agent (LangChain-нода) → поиск по базе знаний → ответ пользователю
3. Шаг 3: Подключить CRM-интеграцию — автоматически создавать сделки/контакты при квалификации лида
4. Шаг 4: Настроить эскалацию на живого менеджера при ключевых триггерах, протестировать и сдать клиенту
