## 1. Название
**AI-Завод** — Контент-Машина

## 2. Суть
n8n-агент автоматически генерирует, форматирует и публикует контент во все каналы бизнеса (Telegram, ВКонтакте, сайт) по расписанию — без участия маркетолога. Тема берётся из списка, Claude пишет текст, изображение генерируется через API.

## 3. Бизнес-сценарий
ДО: Владелец кофейни Антон каждую неделю говорит SMM-фрилансеру: «Напиши что-нибудь про латте и скидку». Фрилансер берёт 15 000 руб./мес, задерживает, публикует 2 поста вместо 5. Антон злится, конкуренты постят каждый день. ИИ делает: каждое утро в 9:00 n8n берёт тему из Google Sheets («Акция на капучино»), Claude пишет пост в стиле бренда, Fusion Brain генерирует картинку с кофе, пост уходит в Telegram-канал и ВКонтакте автоматически. Клиент получает: 5 постов в неделю без единого звонка фрилансеру, стиль выдержан, акции выходят вовремя.

## 4. Логика
```
Cron (расписание) → Google Sheets (тема дня) → Claude (генерация текста) → Fusion Brain API (изображение) → Telegram Bot API + VK API (публикация)
```

## 5. Технический стек
- n8n
- claude-haiku
- Fusion Brain API

- Telegram Bot API

## 6. Связки инструментов
```
n8n Cron → Google Sheets Node (тема) → Claude API (текст) → HTTP Request Fusion Brain (картинка) → Telegram/VK Nodes (публикация)
```

## 7. Алгоритм реализации
1. Создать Google Sheets с колонками: дата, тема, статус — клиент заполняет темы на месяц вперёд (30 строк)
2. Настроить n8n Cron-триггер на 09:00 ежедневно, читать строку с сегодняшней датой из Sheets
3. Передать тему + системный промпт бренда в Claude Haiku, получить текст поста до 800 символов с эмодзи и CTA
4. Отправить POST-запрос в Fusion Brain API с описанием изображения из поста, получить картинку в base64
5. Опубликовать пост через Telegram Bot API и VK API, записать статус 'опубликовано' обратно в Sheets
6. Отправить уведомление владельцу в Telegram с превью поста и ссылкой на публикацию

## 8. Почему купят (ROI)
Фрилансер-SMM стоит 15 000–25 000 руб./мес и всё равно делает 2–3 поста в неделю вместо 5. Бизнес платит за присутствие, а получает тишину. AI-Завод производит 5 постов/нед = 20/мес стабильно, без опозданий. Экономия 15 000–25 000 руб./мес от первого месяца. Крючок: «Фрилансер берёт деньги за присутствие, ИИ берёт деньги за результат.»
