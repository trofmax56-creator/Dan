## 1. Название
**AI-Ревизор** — Контролёр 100% Звонков

## 2. Суть
Агент автоматически транскрибирует все звонки менеджеров, оценивает каждый по чек-листу продаж (приветствие, выявление потребности, отработка возражений, закрытие), выставляет балл 1-10 и отправляет РОПу сводный отчёт с флагами проблемных разговоров — без ОКК в штате.

## 3. Бизнес-сценарий
ДО: РОП Алексей слушает вручную 5-10 звонков из 200 в месяц — это 5% контроля, остальные 95% в слепой зоне. Менеджер Дима стабильно «забывает» называть цену и уходит от закрытия. Это видно только когда план провален. ИИ делает: каждый звонок автоматически попадает в транскрипцию, Claude проверяет по 8 критериям чек-листа, фиксирует временну́ю метку где Дима ушёл от закрытия, выставляет оценку 4/10 и помечает тегом «пропущено закрытие». РОП получает: утром Telegram-дайджест «Вчера 23 звонка, средний балл 6.8, 3 звонка ниже 4 — ссылки и временны́е метки приложены», открывает конкретный момент разговора за 30 секунд.

## 4. Логика
```
Звонок завершён → UIS/МТС Exolve webhook (запись MP3) → n8n → Whisper/SaluteSpeech (транскрипция) → Claude (оценка по чек-листу 8 критериев) → Bitrix24 (запись в карточку сделки) → Telegram РОПу (дайджест при оценке < 4)
```

## 5. Технический стек

## 6. Связки инструментов
```
UIS webhook → n8n → SaluteSpeech (транскрипция) → Claude (скоринг 1-10 по 8 критериям) → Bitrix24 REST API (поле оценки в сделке) → Telegram Bot (алерт РОПу при балле < 4)
```

## 7. Алгоритм реализации
1. Подключить UIS или МТС Exolve к n8n через webhook: после каждого завершённого звонка передавать URL записи MP3 и ID менеджера
2. Отправить аудио в SaluteSpeech API (Сбер, РФ-совместим) для транскрипции, получить текст с тайм-кодами реплик
3. Передать транскрипт в Claude с системным промптом: оценить по 8 критериям чек-листа (приветствие, имя клиента, выявление потребности, презентация продукта, работа с возражениями, озвучивание цены, попытка закрытия, договорённость о следующем шаге) — каждый критерий 0 или 1, итог из 8
4. Записать оценку, расшифровку и временны́е метки нарушений в кастомные поля карточки сделки Bitrix24 через REST API
5. Ежедневно в 9:00 отправлять РОПу Telegram-дайджест: топ-3 худших звонка за вчера с прямыми ссылками на временны́е метки проблемных моментов; немедленный алерт при оценке ≤ 3

## 8. Почему купят (ROI)
2 специалиста ОКК = 160 000 руб./мес ФОТ + налоги = ~210 000 руб./мес, при этом они физически прослушивают максимум 15-20% звонков. AI-Ревизор контролирует 100% звонков за 15 000 руб./мес инфраструктуры. Окупаемость — первый месяц. Крючок: «ОКК из двух человек за 15 тысяч в месяц, который не устаёт и не договаривается с менеджерами.»
