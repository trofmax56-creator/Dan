ANTHROPIC ЗАПУСТИЛА CHANNELS — CLAUDE CODE ТЕПЕРЬ РАБОТАЕТ ЧЕРЕЗ TELEGRAM И DISCORD. 

Помните, я писал про Dispatch? Так вот, Anthropic не остановилась и пошла дальше — теперь Claude Code можно дёргать прямо из Telegram и Discord. Функция называется [Channels, и она в бете](https://code.claude.com/docs/en/channels).

Суть простая: ты ставишь MCP-плагин в Claude Code, подключаешь Telegram-бота через BotFather — и всё. Пишешь боту в Телеграм задачу, Claude выполняет её в терминале на твоём компьютере и отвечает тебе в чат. Прямо как OpenClaw, только от самой Anthropic.

Как настроить:

Обновить Claude Code до версии 2.1.80+
В терминале: /plugin install telegram@claude-plugins-official
Создать бота в BotFather, получить токен
Настроить: /telegram:configure <токен>
Работает на подписках Pro ($20), Max ($100/$200). Для Enterprise нужно включать вручную.

Теперь про честное мнение.

Это ровно то, что делает OpenClaw. Только OpenClaw это умел уже давно, работает с любыми моделями и мессенджерами, и главное — это open-source, который ты контролируешь сам.

Channels — это Anthropic, которая посмотрела на то, что делает сообщество, и сказала: "Мы тоже так можем. И сделаем это нативно." И это правильный ход — большинству людей не нужна гибкость OpenClaw, им нужно чтобы работало из коробки.

Но есть нюанс: твой компьютер должен быть включён и Claude Code должен быть запущен. То есть это не облачный агент, который работает 24/7 — это всё ещё привязка к твоей машине. Dispatch решал это частично, Channels — дополняет, но до полной автономии ещё далеко.

Вывод: Если вы уже на OpenClaw и вам нравится свобода выбора — Channels пока не повод переезжать. Если вы целиком в экосистеме Claude и хотите просто писать задачи боту в Телеграм без настройки серверов — это именно для вас. Anthropic продолжает закрывать разрыв с open-source решениями, и это хороший знак для всех.
