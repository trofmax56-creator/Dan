## Суть
Создание мульти-канального AI-агента на n8n, который интегрирует Telegram, веб-сайт и Instagram с Bitrix24, используя Claude для обработки сообщений клиентов через Открытые Линии. Система автоматизирует получение входящих сообщений, их обработку искусственным интеллектом и направление ответов обратно клиентам через соответствующие каналы.

## Бизнес-сценарий
Компании, работающие с клиентами через несколько каналов коммуникации (мессенджеры и веб-сайт), используют эту систему для автоматизации обработки входящих запросов. AI-агент получает сообщения из Telegram, Instagram и веб-формы сайта, отправляет их в Bitrix24 как сообщения в Открытых Линиях, обрабатывает ответы через Claude и отправляет их обратно клиентам в исходном канале.

## Алгоритм реализации
1. 1. Настройка Telegram-бота в n8n: создание веб-хука для получения входящих сообщений от пользователей, сохранение chat_id для отправки ответов
2. 2. Интеграция Instagram: подключение Instagram Business Account API через n8n для получения Direct Messages с идентификацией отправителя
3. 3. Веб-форма на сайте: создание HTML-формы с webhook'ом в n8n для перехвата информации (имя, телефон, сообщение) из посетителей
4. 4. Создание контактов в Bitrix24: проверка существует ли контакт в Bitrix24 по телефону/email, если нет — создание нового контакта с сохранением идентификатора канала источника
5. 5. Отправка сообщения в Открытые Линии Bitrix24: использование API Bitrix24 для создания сообщения в чате Открытой Линии с привязкой к контакту и сохранением текста оригинального сообщения
6. 6. Обработка сообщения Claude AI: передача входящего сообщения и контекста в Claude API для генерации интеллектуального ответа на основе контекста беседы
7. 7. Отправка ответа обратно в исходный канал: использование сохраненного идентификатора канала для маршрутизации ответа Claude в Telegram, Instagram DM или email уведомление
8. 8. Логирование и отслеживание: создание истории всех взаимодействий с указанием временных меток, идентификаторов транзакций и статусов обработки для аудита

## Технический стек
- n8n — оркестратор рабочих процессов и создание автоматизаций
- Bitrix24 REST API — работа с контактами, сообщениями и Открытыми Линиями
- Telegram Bot API — получение и отправка сообщений через веб-хуки
- Instagram Graph API / Instagram Business Account API — получение Direct Messages
- Claude API (Anthropic) — обработка и генерация ответов на основе AI
- Webhook — прием данных из веб-формы сайта
- HTTP Request ноды в n8n — выполнение API запросов
- JSON — формат данных для обмена между сервисами

## Связки инструментов
- Webhook n8n → Telegram Bot API (получение сообщений)
- Webhook n8n → веб-форма сайта (получение данных посетителей)
- Instagram Graph API → n8n (получение DM)
- n8n → Bitrix24 REST API (создание/обновление контактов)
- n8n → Bitrix24 API (отправка в Открытые Линии)
- n8n → Claude API (обработка сообщения)
- n8n → Telegram Bot API (отправка ответа)
- n8n → Instagram API (отправка ответа в DM)
- n8n → Email/SMTP (отправка уведомлений веб-посетителям)

## Конфигурация и параметры
- Telegram Bot Token: вставляется в HTTP Request ноде для аутентификации (формат: Credentials или напрямую в URL параметры)
- Bitrix24 Webhook URL: вида https://your-domain.bitrix24.ru/rest/{userId}/{webhookToken}/ — используется для всех операций с Bitrix24 API
- Claude API Key: сохраняется в Credentials n8n для безопасного хранения и использования в HTTP Request ноде
- Instagram Business Account ID и Access Token: получаются из Facebook Business Suite для работы с Graph API
- Имена полей Bitrix24 контактов: PHONE, EMAIL, NAME — указываются в теле POST запроса при создании контакта
- ID Открытой Линии в Bitrix24: используется при отправке сообщения (параметр OL_ID или LINE_ID в зависимости от версии API)
- Хранение session_id чата: сохраняется в переменной n8n для отправки следующего сообщения в тот же диалог
- Параметры Claude запроса: model (claude-3-sonnet или claude-2), max_tokens, temperature, system_prompt для контекста
