# 🗂️ Паспорт идеи: ИИ-ассистент на Claude с памятью

**Источник:** [How to Use Claude Cowork Better Than 99% of People](https://youtube.com/watch?v=f95-O8C88uw) — Brock Mesarich | AI for Non Techies (2026-03-05)
**Категория:** Замена должности
**Заменяет:** Персональный ассистент / офис-менеджер

---

## 💡 Что это даёт бизнесу
Бизнес получает цифрового ассистента на базе Claude, который помнит контекст компании, задачи и предпочтения руководителя — без найма живого сотрудника. Ассистент обрабатывает входящие запросы, готовит черновики писем/документов, ведёт список задач и отвечает в нужном стиле. Экономия 40 000–80 000 руб./мес на зарплате ассистента при работе 24/7.

---

## ✅ Плюсы
- Низкий порог входа: Claude API + n8n/Make.com, нет сложной инфраструктуры
- Легко тиражируется — одна архитектура продаётся десяткам клиентов
- Высокая воспринимаемая ценность: клиент видит живой диалог и экономию

## ❌ Минусы / Риски
- Claude API недоступен напрямую в РФ — нужен прокси/VPN-прослойка, что добавляет нестабильность
- Конкуренция высокая: похожие решения предлагают фрилансеры за 20–30 тыс., нужно отстроиться качеством памяти и интеграций

## 🚀 Шаги запуска
1. Шаг 1: Собрать шаблон 'памяти' — файл/база с контекстом компании, задачами, стилем общения руководителя (Notion или Google Sheets)
2. Шаг 2: Настроить триггер входящих сообщений (Telegram или email) → n8n/Make.com подтягивает память → отправляет в Claude API → возвращает ответ
3. Шаг 3: Добавить инструменты-действия: создание задач в Bitrix24/amoCRM, черновики писем, напоминания — через API интеграции
4. Шаг 4: Упаковать в продукт: онбординг-анкета клиента → автонастройка памяти → передача доступа + 1 мес поддержки включено
