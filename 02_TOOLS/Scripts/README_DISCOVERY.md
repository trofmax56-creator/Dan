# Discovery-Dan System Guide

Automated script package for finding engineering content.

## Dependencies
```bash
pip install telethon google-api-python-client requests
```

## Scripts

1. tg_discovery.py: Search for TG channels based on technical keywords.
2. yt_discovery.py: Search for YouTube videos with technical filters.
3. article_discovery.py: Scrape Habr/VC full articles using Firecrawl.

Check individual scripts for configuration details.
