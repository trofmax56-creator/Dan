# 🗂️ Паспорт идеи: AI-разработчик на Claude Code

**Источник:** [Claude Code: полный гайд по AI-кодингу (хаки, техники и секреты)](https://youtube.com/watch?v=6NK4Pona2fY) — Олег Стефанов (2025-10-15)
**Категория:** Замена должности
**Заменяет:** Junior/Middle разработчик автоматизаций

---

## 💡 Что это даёт бизнесу
Студия может ускорить разработку интеграций и автоматизаций в 3-5 раз, используя Claude Code как внутренний инструмент вместо найма дополнительных разработчиков. Клиентам малого бизнеса можно предложить услугу 'AI-разработчик под ключ' — быстрое создание скриптов, ботов и интеграций без дорогостоящего штата. Снижение себестоимости проектов Студии при сохранении маржинальности.

---

## ✅ Плюсы
- Сокращает время разработки типовых интеграций (n8n, Make.com, API-связки) с недель до дней
- Позволяет Студии браться за больше проектов параллельно без расширения команды
- Claude Code хорошо работает с контекстом кодовой базы — подходит для доработки существующих автоматизаций клиентов

## ❌ Минусы / Риски
- Видео ориентировано на IT-разработчиков, а не на бизнес-автоматизацию — прямой бизнес-кейс для клиентов Студии слабый
- Высокий порог входа для клиентов МСБ: они сами не будут использовать Claude Code, это инструмент только для команды Студии

## 🚀 Шаги запуска
1. Шаг 1: Установить Claude Code CLI внутри команды Студии, протестировать на реальном проекте интеграции (например, Bitrix24 + n8n)
2. Шаг 2: Создать библиотеку промптов для типовых задач Студии: парсинг API, написание webhook-хэндлеров, скрипты для amoCRM
3. Шаг 3: Внедрить в рабочий процесс как обязательный инструмент разработчиков — замерить экономию времени на 3 проектах
4. Шаг 4: Упаковать как конкурентное преимущество в продажах ('разрабатываем в 3x быстрее, поэтому дешевле конкурентов')
