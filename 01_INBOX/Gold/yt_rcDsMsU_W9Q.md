## Суть
Видео демонстрирует создание полнофункционального лендинга с нуля за 7 минут, используя Claude Design для проектирования интерфейса и Perplexity для поиска информации и генерации контента. Результат — готовый к использованию посадочный сайт без написания кода с нуля.

## Бизнес-сценарий
Веб-разработчики, маркетологи и предприниматели, которым нужно быстро создать лендинг без глубоких навыков программирования. Используется для создания посадочных страниц, минисайтов, портфолио. Обрабатываются: дизайн-макеты, текстовый контент, изображения, структура страницы.

## Алгоритм реализации
1. Шаг 1: Открыть Claude Design и начать новый проект. Определить целевую аудиторию и назначение лендинга (например, продвижение сервиса, сбор контактов). Выбрать шаблон или начать с пустой страницы.
2. Шаг 2: Использовать Perplexity для поиска трендов в дизайне, конкурентных решений и текстовых идей для лендинга. Генерировать UX-копирайтинг и структуру разделов (Hero, Features, Call-to-Action, Footer).
3. Шаг 3: В Claude Design добавить основные блоки: заголовок, описание, визуальные элементы (иконки, изображения), форму для сбора данных. Выбрать цветовую схему и типографику.
4. Шаг 4: Копировать текстовый контент из Perplexity в соответствующие секции лендинга. Оптимизировать текст для конверсии, добавить призывы к действию (CTA).
5. Шаг 5: Настроить интерактивные элементы: кнопки, ссылки навигации, форма обратной связи. Протестировать функциональность на разных устройствах (мобильная и десктопная версии).
6. Шаг 6: Экспортировать готовый лендинг. Выбрать формат экспорта (HTML, или публикация на встроенном хостинге Claude Design). Убедиться в доступности и скорости загрузки.
7. Шаг 7: Опубликовать лендинг и настроить аналитику (если доступно). Протестировать форму сбора данных, убедиться в корректной отправке информации на указанный email или webhook.

## Технический стек
- Claude Design (инструмент для дизайна интерфейсов)
- Perplexity AI (поисковая и генеративная система)
- HTML/CSS (базовая верстка при необходимости)
- Встроенный хостинг Claude Design
- JavaScript (для интерактивных элементов, если нужно)
- Стандартные веб-технологии (HTTP, REST)

## Связки инструментов
- Perplexity Search → идеи контента/дизайна → Claude Design → создание макета и экспорт → опубликованный лендинг

## Конфигурация и параметры
- Выбор размера экрана в Claude Design (обычно 1440x900 пиксель для десктопа, 375x667 для мобильной версии)
- Установка цветовой палитры (основной цвет, вспомогательные цвета, цвета текста и фона)
- Выбор шрифтов (Google Fonts или системные шрифты)
- Настройка отступов и выравнивания элементов (padding, margin, flexbox или grid)
- Конфигурация формы обратной связи: поля (имя, email, сообщение), тип отправки (email webhook или встроенная система)
- Установка параметров кнопок CTA: текст кнопки, цвет, размер шрифта, функция действия (ссылка или форма)
- Добавление favicon и заголовка страницы (title, meta description)

## Ключевые инсайты
- Claude Design позволяет создавать лендинги без написания кода вручную — всё делается визуально в конструкторе
- Perplexity ускоряет генерацию идей и контента: можно за несколько запросов получить полный концепт и текст
- Средний процесс создания: поиск идеи (Perplexity) → дизайн макета (Claude Design) → наполнение контентом → экспорт = 5-10 минут
- Мобильная оптимизация в Claude Design происходит автоматически через адаптивный дизайн, но нужна проверка на реальных устройствах
- Использование встроенной формы сбора данных в Claude Design удаляет необходимость интеграции с внешними сервисами (Zapier, n8n)
- Экспорт HTML позволяет дальше размещать лендинг на любом хостинге (Netlify, GitHub Pages, собственный сервер) без привязки к платформе
- Для повышения конверсии нужно: чёткий заголовок (сразу объяснить ценность), минимум 2-3 CTA кнопки, социальное доказательство (если есть отзывы)
- Perplexity может генерировать не только текст, но и подсказывать дизайн-решения: 'какие элементы должны быть на лендинге SaaS-продукта'

## Подводные камни
- Если в Claude Design экспортировать HTML без подключенного хостинга, статические файлы нужно будет самостоятельно загружать на сервер
- Perplexity может выдать устаревшую или неточную информацию — всегда проверяй факты перед добавлением на лендинг
- Форма в Claude Design работает только если к ней привязана email-адрес или webhook для получения данных
- Мобильная версия может выглядеть неправильно если не установлены правильные breakpoints в дизайне (например, при ширине < 768px)
- Изображения с высоким разрешением замедляют загрузку лендинга — нужна оптимизация размера файлов
- Встроенный хостинг Claude Design может иметь ограничения на трафик и время жизни проекта (уточнить в документации)
- Если скопировать контент прямо из Perplexity без редакции, текст может быть слишком длинным или не адаптированным под целевую аудиторию
