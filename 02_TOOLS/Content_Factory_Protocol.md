# 🏭 Протокол: Контент-Фабрика Trofimov (v1.1 - EXPERIMENTAL)

Этот протокол описывает логику адаптации контента. ВНИМАНИЕ: Стратегия в режиме тестирования («пристрелка»).

## 📥 Входящие данные
- **Место поиска:** Папка `11_CONTENT_FACTORY/11.1_DRAFTS` в Obsidian.
- **Формат:** .docx файлы с кейсами и постами.

## 🏗️ Схема «Якорь + Эхо»
1. **Якорь (vc.ru / Habr):** Самый сильный технический или бизнес-кейс.
2. **Эхо:** Адаптация якоря для Дзена, TenChat, ВК.

## 📝 Площадки и Правила (Гибкие)

### 1. vc.ru (Бизнес-лонгрид)
- Провокационные заголовки, цифры, графики. Поиск клиентов.

### 2. Habr (Технический лонгрид) - NEW 🚀
- **Цель:** Пиар продуктов через экспертизу.
- **Стиль:** Глубокое погружение, код, схемы n8n, API нюансы. Минимум маркетинговой воды.

### 3. Яндекс Дзен (Сторителлинг)
- Эмоциональные истории, короткие абзацы.

### 4. TenChat (Экспертный тезис)
- Коротко, по делу, ссылки в комментариях.

### 5. ВКонтакте (Разговорный)
- Короткие посты, призыв к обсуждению.

## 🚀 Алгоритм Данилы
1. Мониторить `11.1_DRAFTS` на наличие новых .docx.
2. Извлекать суть (без жестких рамок по знакам на этапе теста).
3. Предлагать Максиму варианты адаптации здесь, в чате.
4. **Пушить в GitHub только после одобрения текста.**
