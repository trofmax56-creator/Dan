# 🗂️ Паспорт идеи: ИИ-агент для малого бизнеса

**Источник:** [9 бизнес-идей для заработка с AI](https://youtube.com/watch?v=xXrCFQxlHZA) — Marina Mogilko (2025-06-13)
**Категория:** Автоматизация отдела
**Заменяет:** Операционный директор / Ассистент руководителя

---

## 💡 Что это даёт бизнесу
Клиент получает автоматизированного ИИ-агента, который берёт на себя рутинные операционные задачи: обработку заявок, постановку задач, контроль дедлайнов и базовую аналитику. Это позволяет владельцу малого бизнеса избавиться от найма дорогостоящего операционного менеджера и сократить ручной труд на 40–60%. Студия Трофимов упаковывает решение в готовый продукт на базе n8n/Make.com + Claude и продаёт как «ИИ-директор под ключ».

---

## ✅ Плюсы
- Высокий спрос: малый бизнес массово ищет способы сократить ФОТ без потери качества управления
- Повторяющаяся модель: после первого внедрения легко тиражировать на схожие ниши (кафе, салоны, логистика)
- Понятная экономика для клиента: ИИ-агент стоит 15–25к/мес против 80–120к/мес живого сотрудника

## ❌ Минусы / Риски
- Видео рекламное и поверхностное — конкретных технических решений не раскрыто, придётся самостоятельно проектировать архитектуру
- Высокие ожидания клиентов при низкой AI-грамотности: риск разочарования, если агент не покрывает 100% кейсов с первого дня

## 🧰 Артефакты для реализации

**Инструменты:** n8n (оркестрация воркфлоу), Make.com (альтернатива для простых сценариев), Claude API (мышление и генерация ответов), Bitrix24 / amoCRM (CRM-ядро), Telegram Bot API (интерфейс для клиента)
**API:** Anthropic Claude API (claude-3-5-sonnet), Bitrix24 REST API, amoCRM API, Telegram Bot API

**Репозитории:**
- n8n-io/n8n (github.com/n8n-io/n8n)
- Поиск по тегу 'ai-agent-crm' на GitHub

**Документация:**
- docs.anthropic.com
- dev.bitrix24.ru/rest_help
- docs.n8n.io

## 🚀 Шаги запуска
1. Шаг 1: Выбрать целевую нишу для пилота (например, салоны красоты или малая логистика) и провести 5 custdev-интервью для выявления топ-3 операционных болей
2. Шаг 2: Спроектировать MVP-агента в n8n: обработка входящих заявок → постановка задач в Bitrix24 → уведомления в Telegram → еженедельный отчёт через Claude
3. Шаг 3: Запустить пилот с 2–3 клиентами по льготной цене (50–80к), собрать метрики экономии времени и задокументировать кейс
4. Шаг 4: Упаковать в тиражируемый продукт с фиксированным прайсом, лендингом и договором на подписку; запустить продажи через LinkedIn/Telegram-канал Студии
