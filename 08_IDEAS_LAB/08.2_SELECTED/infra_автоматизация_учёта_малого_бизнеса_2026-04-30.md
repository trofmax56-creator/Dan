# 🗂️ Паспорт идеи: Автоматизация учёта малого бизнеса

**Источник:** [ПОЛНЫЙ гайд по автоматизации бизнеса - С НУЛЯ до рабочей системы](https://youtube.com/watch?v=nAcPcZO1pec) — Карташов | Учётник (2025-01-03)
**Категория:** Автоматизация отдела
**Заменяет:** Бухгалтер/учётник на первичке + операционный менеджер

---

## 💡 Что это даёт бизнесу
Клиент получает автоматический сбор первичных данных (продажи, расходы, остатки) без ручного ввода. ИИ-слой категоризирует операции и формирует управленческие отчёты по запросу. Владелец видит P&L, кассовые разрывы и аномалии в реальном времени без найма отдельного специалиста.

---

## ✅ Плюсы
- Массовая боль: 80% малого бизнеса в России ведёт учёт в Excel или не ведёт вообще
- Короткий цикл сделки — боль очевидна, ROI считается за 5 минут (замена учётника 40–60 тыс./мес.)
- Легко тиражируется: один раз настроенный шаблон под нишу (ритейл, услуги, общепит) продаётся повторно

## ❌ Минусы / Риски
- Высокая конкуренция с готовыми SaaS (МойСклад, 1С:Фреш, Финтабло) — нужно чёткое отличие через ИИ-аналитику
- Клиенты МСБ часто не готовы к дисциплине ввода данных, что ломает автоматику и создаёт саппорт-нагрузку

## 🧰 Артефакты для реализации

**Инструменты:** n8n, Make.com, Bitrix24 / amoCRM, Google Sheets / Notion, Финтабло API или МойСклад API
**API:** Claude API (Anthropic), OpenAI API, МойСклад REST API, Tinkoff / Сбер бизнес API (выписки)

**Репозитории:**
- n8n-io/n8n (self-hosted)
- github.com/makecom — нет публичных шаблонов, кастом

**Документация:**
- https://moisklad.ru/api/remap/1.2

## 🚀 Шаги запуска
1. Шаг 1: Провести Discovery (2–3 дня) — выгрузить текущие источники данных клиента: касса, банк, таблицы, мессенджеры
2. Шаг 2: Настроить сборщик данных через n8n/Make — подключить банковские выписки, кассу, CRM → единая БД (Airtable или Google Sheets)
3. Шаг 3: Добавить ИИ-слой (Claude) для категоризации операций, детекции аномалий и генерации управленческого резюме по расписанию
4. Шаг 4: Сдать дашборд (Looker Studio или Bitrix24 виджет), обучить клиента, подписать на абонемент поддержки
