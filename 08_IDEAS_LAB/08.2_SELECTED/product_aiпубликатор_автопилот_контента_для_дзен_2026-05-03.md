## 1. Название
**AI-Публикатор** — Автопилот Контента для Дзен и Соцсетей

## 2. Суть
Агент на n8n + Claude берёт сырые тезисы или голосовые заметки владельца бизнеса, разворачивает в SEO-оптимизированные статьи для Яндекс.Дзен и посты для ВКонтакте/Telegram, публикует по расписанию и отчитывается о просмотрах — контент-план закрывается без редактора и копирайтера.

## 3. Бизнес-сценарий
ДО: Владелец кофейни Марина знает, что контент в Дзен даёт клиентов, но нанимать копирайтера за 40 000 руб./мес не хочет. Сама пишет раз в 2 недели, когда «есть время», — 3 статьи в месяц, охваты мизерные. ИИ ДЕЛАЕТ: Марина записывает 2-минутную голосовую заметку в Telegram-бот: «Хочу статью про то, почему осенью лучше пить флэт уайт, а не американо — там про температуру и вкус». Через 8 минут бот присылает готовую статью на 1 800 слов с заголовком, подзаголовками, ключевыми словами и рекомендацией к публикации в четверг в 09:00. Марина нажимает «Одобрить» — агент сам публикует в Дзен и ВКонтакте. КЛИЕНТ ПОЛУЧАЕТ: 15–20 публикаций в месяц, органический трафик из рекомендательной системы Дзена начинает накапливаться — контент не умирает через сутки, а набирает охваты месяцами.

## 4. Логика
```
Голосовая/текстовая заметка в Telegram-бот → n8n (транскрибация Deepgram) → Claude (статья 1500–2000 слов + SEO-теги + адаптация для ВК) → превью владельцу в Telegram → кнопка «Одобрить» → публикация в Дзен API + ВКонтакте API по расписанию → сбор статистики через 7 дней → отчёт в Telegram
```

## 6. Связки инструментов
```
Telegram Bot (голосовая/текст) → n8n Webhook → Deepgram (транскрибация) → Claude (генерация статьи + 5 SEO-ключей + адаптация под ВК 800 слов) → Telegram Bot (превью + кнопки Одобрить/Редактировать) → Яндекс.Дзен API (публикация по cron) → ВКонтакте API (кросспостинг) → Яндекс.Дзен API (stats через 7д) → Telegram отчёт
```

## 7. Алгоритм реализации
1. Создать Telegram-бота для приёма контент-заявок: поддержка голосовых (→ Deepgram транскрибация), текстовых сообщений и файлов .txt; бот принимает до 5 заявок в день
2. Claude-промпт системный: роль SEO-копирайтера для Дзена, структура статьи (заголовок H1 + лид 100 слов + 4–5 подзаголовков H2 + заключение + CTA), плотность ключей 1,5–2%, длина 1500–2000 слов — параметры настраиваются под нишу клиента при онбординге
3. n8n формирует превью: первые 300 слов статьи + предложенный заголовок + 5 ключевых слов → отправить владельцу в Telegram с кнопками «✅ Одобрить», «✏️ Правка», «🗑 Отклонить»; при «Правке» — текстовое поле с комментарием → Claude дорабатывает
4. При нажатии «Одобрить» n8n ставит публикацию в очередь по контент-плану (не более 1 публикации в день, оптимальное время 09:00 или 19:00 по часовому поясу клиента) → публикует через Дзен API и ВКонтакте API
5. Cron через 7 дней после публикации: запрос статистики Дзена (просмотры, дочитывания, переходы) → Claude формирует вывод «лучший материал недели» и рекомендацию темы → отчёт владельцу в Telegram

## 8. Почему купят (ROI)
Копирайтер на аутсорсе для Дзена — 35 000–55 000 руб./мес за 12–15 статей. AI-Публикатор даёт 20 статей в месяц за 3 500 руб. инфраструктуры. Дзен работает по накопительной модели: статья, написанная сегодня, будет приносить трафик через 6 месяцев — это актив, а не расход. Крючок: «Копирайтер пишет 15 статей и уходит в отпуск. Ваши статьи не уходят никуда.»
