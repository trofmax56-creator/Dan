# 🗂️ Паспорт идеи: ИИ-автоматизация продаж в Bitrix24

**Источник:** [Битрикс24: полный обзор ВСЕХ функций в 2025 (воронки продаж, ии для бизнеса, корпоративный портал)](https://youtube.com/watch?v=3Jw2Pw4xhwY) — Константин Киселев (2025-01-06)

---

## 💡 Что это даёт бизнесу
Клиент получает готовую настройку Bitrix24 с ИИ-ассистентом, который квалифицирует лиды, заполняет карточки сделок и двигает их по воронке без участия менеджера. Сокращается время обработки заявки с часов до минут, снижается человеческий фактор. Руководитель видит прозрачную аналитику и контролирует продажи в режиме реального времени.

---

## ✅ Плюсы
- Bitrix24 массово используется в российском МСБ — рынок огромный, знакомый инструмент клиентам
- Быстрый старт: базовая автоматизация воронки + ИИ-скрипты реализуются за 2–4 недели
- Легко тиражировать по отраслям (недвижимость, стройка, услуги) с минимальными доработками

## ❌ Минусы / Риски
- Bitrix24 часто уже есть у клиента, но настроен криво — придётся разбирать чужой хаос перед интеграцией
- ИИ-функции Bitrix24 (CoPilot) ограничены и требуют дополнительных интеграций для серьёзной автоматизации

## 🧰 Артефакты для реализации

**Инструменты:** Bitrix24 (CRM + роботы + бизнес-процессы), n8n / Make.com (внешняя автоматизация), Claude API / OpenAI API (ИИ-обработка лидов), Bitrix24 REST API
**API:** Bitrix24 REST API (crm.deal, crm.lead, im.message), Claude API (Anthropic), OpenAI API (GPT-4o)

## 🚀 Шаги запуска
1. Шаг 1: Аудит текущей воронки клиента — выявить узкие места, где теряются лиды и тратится ручной труд
2. Шаг 2: Настройка Bitrix24 — воронки, роботы, триггеры, обязательные поля карточек
3. Шаг 3: Подключение ИИ через n8n/Make — автоквалификация лида по входящему сообщению, авtozаполнение карточки, генерация первого ответа клиенту
4. Шаг 4: Обучение команды (2–4 часа), передача регламентов, настройка дашборда для руководителя
