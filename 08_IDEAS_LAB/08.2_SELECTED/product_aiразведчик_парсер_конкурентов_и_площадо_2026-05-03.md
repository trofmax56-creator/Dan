## 1. Название
**AI-Разведчик** — Парсер конкурентов и площадок

## 2. Суть
Автоматически парсит Авито, маркетплейсы и сайты конкурентов через Firecrawl + Claude, собирает данные в таблицу и уведомляет владельца об изменениях цен и новых объявлениях.

## 3. Бизнес-сценарий
ДО: Менеджер интернет-магазина вручную мониторит 3 конкурента на Авито и Wildberries — открывает каждый сайт, сравнивает цены, записывает в Excel. Уходит 2 часа ежедневно, всё равно пропускает изменения. ИИ делает: каждое утро в 7:00 агент обходит 20 URL конкурентов, вытаскивает цены, остатки, новые позиции. Claude сравнивает с вчерашними данными и пишет в Telegram владельцу: «Конкурент "ОптТорг" снизил цену на артикул 4521 с 3 200 до 2 900 руб. Ваша цена сейчас выше на 12%.» Клиент получает: готовую таблицу с кликабельными ссылками и дельтой цен — без единого клика вручную.

## 4. Логика
```
Cron (7:00) → n8n запускает Firecrawl-парсинг URL-листа → Claude анализирует дельту цен → Google Sheets обновление + Telegram-уведомление владельцу
```

## 5. Технический стек
- n8n
- claude-sonnet-4-5
- Firecrawl
- Google Sheets
- Telegram Bot API

## 6. Связки инструментов
```
n8n Cron → Firecrawl API (парсинг страниц) → Claude (сравнение и интерпретация дельты) → Google Sheets REST API (запись) → Telegram Bot (уведомление)
```

## 7. Алгоритм реализации
1. Загрузить в n8n Google Sheet с 20–100 URL для мониторинга (конкуренты, Авито, маркетплейсы)
2. Настроить Cron-триггер на запуск в 7:00 и 19:00 ежедневно, передавать URL-список в Firecrawl API пакетами по 10 штук
3. Claude получает HTML-контент страниц и извлекает: цену, наличие, заголовок, дату обновления — структурированный JSON
4. n8n сравнивает текущие данные с предыдущим снимком в Google Sheets, вычисляет дельту по каждой позиции
5. Если дельта цены > 5% или появилось новое объявление — Claude формирует человекочитаемый дайджест и отправляет в Telegram-чат владельца с таблицей изменений

## 8. Почему купят (ROI)
Менеджер тратит 2 часа в день × 22 рабочих дня = 44 часа/мес на ручной мониторинг (≈ 26 000 руб. по ставке 600 руб./час) и всё равно реагирует на изменения цен с опозданием в 1–2 дня, теряя заказы. AI-Разведчик работает дважды в сутки и сигналит за 3 минуты. Крючок: «Пока конкурент снизил цену вчера — вы узнали сегодня. Теперь наоборот.»
