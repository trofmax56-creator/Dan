# 🗂️ Паспорт идеи: Автоматизация обработки заказов интернет-магазина

**Источник:** [Автоматизация заказов в интернет магазине. 23.01.2018](https://youtube.com/watch?v=Pc8VC4xEXaE) — Вебинары 1С-Битрикс: Управление Сайтом (2018-01-23)
**Категория:** Автоматизация отдела
**Заменяет:** Менеджер по обработке заказов / оператор колл-центра

---

## 💡 Что это даёт бизнесу
Заказы из интернет-магазина автоматически попадают в CRM/1С без ручного переноса, статусы обновляются автоматически, клиент получает уведомления без участия менеджера. Снижается количество ошибок при обработке и время от заказа до отгрузки сокращается в 2–3 раза.

---

## ✅ Плюсы
- Массовая боль: каждый интернет-магазин вручную обрабатывает заказы — прямая экономия на ФОТ
- Видео 2018 года — технология давно зрелая, риск сырых инструментов минимален
- Легко упаковать в типовой продукт и продавать повторно разным клиентам

## ❌ Минусы / Риски
- Видео посвящено нативному функционалу 1С-Битрикс (CMS), а не Bitrix24 CRM — стек Студии частично не совпадает, потребуется адаптация
- Видео 2018 года: конкретные API и интеграции могли устареть, нужна актуализация под текущие версии платформ

## 🚀 Шаги запуска
1. Шаг 1: Аудит текущего процесса клиента — откуда приходят заказы, куда попадают, кто обрабатывает вручную
2. Шаг 2: Настройка интеграции интернет-магазин → Bitrix24/amoCRM через REST API или n8n (автосоздание сделки/заказа)
3. Шаг 3: Автоматизация статусной модели — триггеры на смену статуса, автоуведомления клиенту (email/SMS/WhatsApp)
4. Шаг 4: Подключение склада/1С для синхронизации остатков и передачи заказа на сборку без ручного ввода
