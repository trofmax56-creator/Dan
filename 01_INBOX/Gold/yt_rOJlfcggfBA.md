## Главная мысль
Создание WhatsApp-бота для строительной компании с автоматической интеграцией в Bitrix24 для управления повторными продажами и клиентскими контактами.

## Схема / Workflow
Входящее сообщение в WhatsApp → Бот обрабатывает запрос → Автоматическая передача данных в Bitrix24 → Создание сделки/контакта → Уведомление менеджера о повторной продаже

## Технические связки
- WhatsApp → Bitrix24

## Мясо
- WhatsApp-бот автоматизирует первичный контакт и квалификацию лидов из строительной сферы
- Интеграция с Bitrix24 позволяет синхронизировать контакты клиентов без ручного ввода
- Система автоматически отслеживает повторные покупки по истории коммуникаций
- Бот обрабатывает типовые вопросы (стоимость, сроки, услуги) без участия менеджера
