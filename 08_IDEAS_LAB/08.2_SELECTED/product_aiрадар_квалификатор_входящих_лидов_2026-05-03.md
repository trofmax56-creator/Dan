## 1. Название
**AI-Радар** — Квалификатор входящих лидов

## 2. Суть
ИИ-агент перехватывает все входящие сообщения в WhatsApp/Telegram, задаёт 4–5 квалификационных вопроса, скорит лид от 1 до 5 и сам создаёт карточку в CRM с заполненными полями — менеджер подключается только к горячим.

## 3. Бизнес-сценарий
ДО: Оконная компания «СтройОкно» получает 80 заявок в месяц через WhatsApp. Менеджер Дима отвечает каждому: «Здравствуйте, что вас интересует?» — уточняет этаж, тип остекления, срок, бюджет. На каждый диалог 20–40 минут. 30% клиентов — «просто узнать цену», и Дима тратит на них столько же времени, сколько на реальные сделки. ИИ делает: клиент пишет в WhatsApp «Хочу застеклить балкон». AI-Радар отвечает: «Добрый день! Подскажите: 1) какой этаж? 2) размер балкона? 3) когда планируете? 4) есть ли примерный бюджет?» За 5 минут переписки Claude анализирует ответы, присваивает скор 4/5, создаёт карточку в Битрикс24 с полями «Балкон 3м, 5 этаж, срок — март, бюджет 80 000». Дима получает уведомление: «Горячий лид, скор 4/5 — позвони». Клиент получает: быстрый отклик, ощущение профессионализма.

## 4. Логика
```
WhatsApp (Wazzup) → n8n Webhook → Claude (квалификация по 5 параметрам, скоринг) → Bitrix24 REST API (создание сделки) → уведомление менеджеру
```

## 5. Технический стек
- n8n
- claude-sonnet

## 6. Связки инструментов
```
Wazzup Webhook → n8n → Claude API (диалог + скоринг) → Bitrix24 REST API (создание сделки/контакта) → Bitrix24 уведомление менеджеру
```

## 7. Алгоритм реализации
1. Подключить Wazzup к n8n через Webhook, настроить фильтр на первое сообщение в новом чате
2. Запустить Claude-агента с системным промптом: последовательно задать 5 вопросов квалификации (бюджет, срок, объём, ЛПР, геолокация), хранить контекст в n8n memory
3. После получения всех 5 ответов Claude выставляет скор 1–5 и формирует JSON с параметрами сделки
4. При скоре ≥ 3 создать сделку в Bitrix24 через REST API с заполненными кастомными полями и тегом источника 'WhatsApp-AI'
5. При скоре < 3 отправить клиенту автоответ с предложением оставить email для рассылки, записать в отдельную воронку 'холодные'
6. Отправить РОПу Telegram-уведомление: скор, имя, краткий профиль лида, ссылка на карточку в Битрикс24

## 8. Почему купят (ROI)
80 лидов/мес × 30 мин квалификации = 40 часов менеджерского времени = 1 рабочая неделя в месяц тратится только на вопросы «что вас интересует». При ФОТ менеджера 60 000 руб. это 15 000 руб. потерь на рутине ежемесячно. Плюс 30% лидов — нецелевые, но Дима не знает это до конца разговора. AI-Радар фильтрует до первого звонка. Крючок: «Менеджер должен закрывать, а не анкетировать.»
