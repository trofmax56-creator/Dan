# 🗂️ Паспорт идеи: Локальный LLM-сервер для бизнеса

**Источник:** [Everything in Ollama is Local, Right?? #llm #localai #ollama](https://youtube.com/watch?v=2PBMs8Z67sk) — Matt Williams (2024-02-20)
**Категория:** Инфраструктура
**Заменяет:** —

---

## 💡 Что это даёт бизнесу
Компания получает корпоративный ИИ-сервер на базе Ollama, где все данные остаются внутри периметра — никакой утечки в OpenAI/Anthropic. Сотрудники пользуются LLM как ChatGPT, но через внутренний endpoint, что снимает юридические и безопасностные риски. Студия разворачивает инфраструктуру под ключ и подключает к CRM/n8n.

---

## ✅ Плюсы
- Данные клиента не покидают сервер — критично для юристов, медицины, финансов, госконтрактов
- Один раз настроил — нет абонентской платы за токены OpenAI, экономия при высоком объёме запросов
- Легко интегрируется с n8n, Make.com, Bitrix24 через REST API — стандартный стек Студии

## ❌ Минусы / Риски
- Видео прямо указывает на 3 исключения когда данные всё же уходят наружу (обновления моделей, телеметрия, внешние registry) — нужно донастраивать и объяснять клиенту
- Требует нормального железа у клиента (GPU или мощный CPU-сервер) — не у всех МСБ есть бюджет или облачный VPS нужного класса

## 🧰 Артефакты для реализации

**Инструменты:** Ollama, Docker / Docker Compose, n8n, Open WebUI, Nginx reverse proxy
**API:** Ollama REST API (localhost:11434), OpenAI-compatible endpoint Ollama

**Репозитории:**
- ollama/ollama (github.com/ollama/ollama)
- open-webui/open-webui (github.com/open-webui/open-webui)

**Документация:**

## 🚀 Шаги запуска
1. Шаг 1: Аудит инфраструктуры клиента — есть ли сервер/VPS с нужными ресурсами, выбор модели (Llama 3, Mistral, Gemma)
2. Шаг 2: Развернуть Ollama в Docker, закрыть внешние исключения (отключить телеметрию, настроить firewall, зеркало для моделей)
3. Шаг 3: Подключить Open WebUI для сотрудников + настроить OpenAI-совместимый endpoint для интеграций
4. Шаг 4: Интегрировать с n8n/Make.com и Bitrix24/amoCRM — автоматизация задач: суммаризация, классификация заявок, генерация ответов
