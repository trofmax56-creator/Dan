## 💡 Что это даёт бизнесу
Клиент получает собственный ИИ-сервер внутри периметра компании без утечки данных в облако — критично для юристов, медицины, финансов. Через MCP-протокол локальная LLM подключается к 8000+ инструментам (CRM, почта, таблицы) и автоматизирует рутину без абонентской платы OpenAI. Студия продаёт разовую установку + поддержку, получая предсказуемый доход.

---

## ✅ Плюсы
- Нет абонентской платы за API — клиент платит только за железо, экономия 30 000–150 000 руб./год по сравнению с GPT-4
- Данные не покидают контур компании — закрывает возражение по 152-ФЗ и корпоративной безопасности
- MCP даёт готовые коннекторы к Bitrix24, amoCRM, Google Sheets — не нужно писать интеграции с нуля

## ❌ Минусы / Риски
- Требует сервер с GPU (минимум RTX 3090 / 24 ГБ VRAM) — не все клиенты готовы к капексу 80 000–200 000 руб. на железо
- Локальные модели (Llama 3, Mistral) уступают GPT-4o по качеству на русском языке — клиент должен быть готов к компромиссу

## 🚀 Шаги запуска
1. Шаг 1: Аудит инфраструктуры клиента — выбор между on-premise GPU-сервером или гибридным VPS; подбор модели (Llama 3.1 8B / 70B, Mistral) под задачи
2. Шаг 2: Развёртывание Ollama в Docker на сервере клиента, настройка MCP-сервера, подключение Zapier MCP для доступа к внешним сервисам
3. Шаг 3: Интеграция с Bitrix24/amoCRM через n8n — настройка триггеров (новая сделка → ИИ пишет письмо, входящий лид → квалификация)
4. Шаг 4: Тестирование на реальных данных клиента, обучение команды, сдача под ключ + договор поддержки
