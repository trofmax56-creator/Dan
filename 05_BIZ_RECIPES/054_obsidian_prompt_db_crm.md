### 1. Описание идеи
В `02_TOOLS/Bitrix_Prompts.md` хранятся протестированные промпты по категориям (возражения, счёт, рекламация). Claude Code перед отправкой ответа в CRM читает этот файл, выбирает подходящий шаблон и адаптирует его под конкретного клиента. Качество ответов стабильно, менеджеры не изобретают текст каждый раз.

### 2. Технический стек
- AI: Claude Code (claude-sonnet-4-6)
- CRM/ERP: Bitrix24
- Middleware: n8n / прямой вызов Claude API

### 3. Пошаговый план внедрения
1. Создать файл `02_TOOLS/Bitrix_Prompts.md` (структура ниже)
2. В CLAUDE.md добавить правило: перед генерацией CRM-ответа читать `02_TOOLS/Bitrix_Prompts.md`
3. n8n: при входящем сообщении в Bitrix24 (вебхук `onCrmActivityAdd`) — вызвать Claude API
4. В запросе к Claude: system = содержимое `Bitrix_Prompts.md`; user = текст клиента + данные сделки
5. Ответ Claude → `crm.activity.add` в Битрикс (тип EMAIL или IM) как черновик для менеджера

### 4. Промпты и Код
```markdown
<!-- 02_TOOLS/Bitrix_Prompts.md -->
# Библиотека промптов Bitrix24

## Категория: Возражение по цене
Ты — менеджер по продажам. Клиент считает цену высокой. Ответь кратко (2-3 предложения),
подчеркни ценность, предложи сравнение с альтернативами. Не давай скидку без согласования.

## Категория: Запрос счёта
Ты — менеджер. Клиент просит счёт. Уточни реквизиты если их нет в карточке,
пообещай выставить в течение 30 минут. Тон: деловой, дружелюбный.

## Категория: Рекламация
Ты — менеджер по работе с клиентами. Клиент недоволен качеством/сроками.
Прими ответственность, не оправдывайся, предложи конкретное решение (замена/возврат/скидка).
```

```python
# Вызов Claude API с промптом из файла
import anthropic, pathlib

prompts_db = pathlib.Path("02_TOOLS/Bitrix_Prompts.md").read_text()
client_message = "Почему так дорого? У конкурентов дешевле."

client = anthropic.Anthropic()
response = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=300,
    system=f"База промптов для CRM-ответов:\n\n{prompts_db}\n\nВыбери подходящую категорию и ответь клиенту.",
    messages=[{"role": "user", "content": client_message}]
)
print(response.content[0].text)
```
