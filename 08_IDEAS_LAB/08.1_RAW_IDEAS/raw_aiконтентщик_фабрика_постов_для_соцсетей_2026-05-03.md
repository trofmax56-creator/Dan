## 1. Название
**AI-Контентщик** — Фабрика Постов для Соцсетей

## 2. Суть
Агент еженедельно генерирует контент-план и готовые тексты постов для ВКонтакте и Telegram на основе ниши клиента, его продуктов и текущих трендов — SMM-менеджер больше не нужен для производства базового контента, только для стратегии.

## 3. Бизнес-сценарий
ДО: Владелец сети автосервисов Роман платит SMM-щику 35 000 руб./мес. Тот делает 12-15 постов в месяц, половина — «поздравляем с праздником» и стоковые фото. Контент не продаёт. ИИ делает: каждый понедельник в 9:00 агент смотрит на прошлую неделю бизнеса (какие услуги оказывались, есть ли акции в МоёмСкладе), забирает 3 актуальных тренда по автотематике, генерирует 5 постов на неделю с конкретными заголовками: «Почему замена масла раз в 10 000 км — это миф, который стоит вам двигателя» — и загружает черновики в отложенный постинг. Роман получает: на согласование уходит 15 минут вместо 3 часов переписки с SMM.

## 4. Логика
```
Cron (понедельник 9:00) → n8n → МойСклад API (данные об услугах за неделю) + HTTP (парсинг трендов) → Claude (генерация 5 постов по шаблонам ниши) → ВКонтакте API / Telegram Bot (черновики на согласование) → уведомление владельцу
```

## 5. Технический стек
- n8n

## 6. Связки инструментов
```
Cron → n8n → МойСклад REST API (топ услуг за 7 дней) → HTTP GET (RSS трендов) → Claude (5 постов: заголовок + тело + CTA + хэштеги) → ВКонтакте API (черновик) → Telegram бот владельцу (превью + кнопка «Опубликовать»)
```

## 7. Алгоритм реализации
1. Настроить Cron-триггер в n8n каждый понедельник 09:00, запрашивать МойСклад API: топ-5 оказанных услуг за прошлые 7 дней и текущие акционные позиции
2. Парсить 2-3 RSS-источника по тематике бизнеса клиента для получения актуальных инфоповодов; передавать в Claude: ниша + топ услуг + инфоповоды + бренд-голос (задаётся один раз при онбординге)
3. Claude генерирует 5 постов по структуре: крючок-заголовок (проблема/факт/вопрос) + тело (180-220 слов) + призыв к действию + 5 релевантных хэштегов; каждый пост в отдельном JSON-объекте
4. Загрузить посты как черновики в ВКонтакте через API и/или отправить в Telegram-бот владельца в виде карточек с кнопками «Опубликовать сейчас», «Отложить на +1 день», «Отклонить»
5. Логировать статистику одобренных/отклонённых постов в Google Sheets; раз в месяц Claude анализирует паттерн отклонений и корректирует промпт генерации под предпочтения клиента

## 8. Почему купят (ROI)
SMM-менеджер на аутсорсе стоит 25 000–45 000 руб./мес за 15-20 постов. AI-Контентщик генерирует 20 постов в месяц за 3 000-5 000 руб. инфраструктуры. Экономия 20 000–40 000 руб./мес с первого месяца. При этом 78% малого бизнеса вообще не ведут соцсети регулярно — не из-за нехватки денег, а из-за нехватки времени на контент. Крючок: «Контент-план на неделю за время утреннего кофе.»
