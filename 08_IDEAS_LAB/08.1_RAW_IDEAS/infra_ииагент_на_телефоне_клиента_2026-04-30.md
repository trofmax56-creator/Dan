# 🗂️ Паспорт идеи: ИИ-агент на телефоне клиента

## 💡 Что это даёт бизнесу
Клиент получает автономного ИИ-агента прямо на смартфоне без абонентской платы за API и без утечки данных в облако. Агент выполняет задачи автономно: парсит, пишет, анализирует, интегрируется с локальными данными. Для малого бизнеса это способ попробовать ИИ-автоматизацию с нулевыми операционными расходами.

---

## ✅ Плюсы
- Нулевая стоимость инфраструктуры для клиента — модель работает локально, без платных API
- Полная конфиденциальность данных — ничего не уходит в облако, актуально для юристов, медицины, финансов
- Низкий порог входа — можно быстро сделать MVP-демо и показать клиенту результат за 1-2 дня

## ❌ Минусы / Риски
- Производительность ограничена железом смартфона — сложные задачи (длинные документы, мультиагентность) нереальны на мобиле
- Узкая ниша применения в B2B: серьёзная автоматизация требует серверной инфраструктуры, а не телефона
