## Мясо
- **Anthropic выпустили постмортем о деградациях Claude Code**
- Компания признала три проблемы с начала весны которые влияли на качество кода. Первая была не очевидным (__лол__) изменением дефолтной reasoning effort с high на medium, что длилось больше месяца. Второй баг удалял из истории сессий которые простаивали больше часа размышления модели после каждого сообщения, что убивало качество (пофиксили за две недели). А третья проблема была просто в системном промпте — модель просто попросили генерить меньше токенов, из-за чего она потупела.
- Вместе с выходом постмортема всем пользователям Claude Code ресетнули лимиты (постмортем вчерашний). Кроме этого Anthropic пообещали чаще использовать публичные версии Claude Code внутри компании и тестить все изменения промпта.
