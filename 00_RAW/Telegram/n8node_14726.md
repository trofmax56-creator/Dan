Ну чат говорит :

**1. “Telegram запрещает собирать ID” - не совсем так**
Telegram **сам по себе отдаёт user_id боту** при любом взаимодействии (start, кнопки, join request и т.д.).
👉 То есть факт получения ID = это **нормальная работа API**, не нарушение.

Проблема начинается не в сборе, а в **использовании**.

⸻
