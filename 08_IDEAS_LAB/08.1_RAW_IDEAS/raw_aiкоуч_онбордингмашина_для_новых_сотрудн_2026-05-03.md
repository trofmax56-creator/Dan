## 1. Название
**AI-Коуч** — Онбординг-машина для новых сотрудников

## 2. Суть
Telegram-бот на базе Claude, обученный на внутренних регламентах и скриптах компании, который сопровождает нового сотрудника в первые 30 дней: отвечает на вопросы, проверяет знания тестами и фиксирует прогресс руководителю.

## 3. Бизнес-сценарий
ДО: Вышел новый менеджер по продажам. РОП тратит 2 часа в день первые 2 недели на объяснения: 'Как работать с CRM?', 'Что говорить клиенту, если он говорит дорого?', 'Куда вносить заявку с сайта?' Через неделю новичок всё равно делает ошибки, потому что не может запомнить 40-страничный регламент. ИИ делает: Новый менеджер Артём в первый день получает ссылку на Telegram-бота. Пишет: 'Клиент говорит, что у конкурентов дешевле, что отвечать?' Бот отвечает готовым скриптом из базы знаний компании + объясняет логику. Раз в 3 дня бот присылает Артёму мини-тест: 5 вопросов по пройденному материалу. Результаты тестов РОП видит в еженедельном отчёте в Telegram: 'Артём: 4/5 тестов сдал, слабое место — работа с возражением по срокам.' Клиент получает: новый сотрудник выходит на плановые показатели за 2 недели вместо 4.

## 4. Логика
```
Telegram-сообщение сотрудника → n8n Webhook → Claude (поиск по базе знаний компании + генерация ответа) → Telegram-ответ; Cron каждые 3 дня → Claude генерирует тест → сотрудник отвечает → результат в Bitrix24 + отчёт РОПу
```

## 5. Технический стек
- n8n
- claude-sonnet-4-5
- Telegram Bot API
- Notion API (база знаний)
- Bitrix24 REST API
- VPS-сервер

## 6. Связки инструментов
```
Telegram → n8n Webhook → Claude (RAG по Notion-базе регламентов) → Telegram ответ сотруднику; n8n Cron (каждые 3 дня) → Claude (генерация теста) → Telegram → ответы сотрудника → Bitrix24 (прогресс) → Telegram РОПу (отчёт)
```

## 7. Алгоритм реализации
1. Загрузить регламенты, скрипты и FAQ компании в Notion (10–50 страниц markdown); подключить Notion API к n8n для поиска релевантных страниц по ключевым словам вопроса
2. Написать Claude-промпт: роль — опытный сотрудник компании, контекст — найденные страницы Notion, задача — ответить на вопрос новичка простым языком + указать номер регламента для самостоятельного изучения
3. Реализовать генератор тестов: Claude раз в 3 дня выбирает 5 вопросов по темам, которые сотрудник изучал (по истории диалогов), формирует тест с 3 вариантами ответов, отправляет в Telegram
4. Записывать результаты тестов в Bitrix24 (кастомные поля карточки сотрудника): дата теста, процент правильных ответов, слабые темы; при результате ниже 60% — автоматическое уведомление РОПу
5. Еженедельно в пятницу в 17:00 формировать и отправлять РОПу в Telegram сводный отчёт по всем новым сотрудникам: прогресс, количество вопросов, результаты тестов, рекомендации по точкам усиления

## 8. Почему купят (ROI)
РОП тратит на онбординг одного менеджера 20–30 часов в первый месяц = при ФОТ РОПа 100 000 руб. — это 12 500–18 750 руб. потраченного дорогого времени на одного новичка. Текучка в продажах 30–50% в год: 3–5 онбордингов ежегодно = 37 500–93 750 руб. в год на 'объяснялки'. AI-Коуч освобождает РОПа от повторяющихся вопросов и сокращает срок выхода на плановые показатели с 4 до 2 недель. Крючок: 'РОП управляет, а не объясняет одно и то же в пятый раз.'
