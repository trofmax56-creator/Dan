## Главная мысль
Автоматизация сохранения данных участников встреч Zoom в Airtable с помощью n8n для упрощения управления контактами и аналитикой мероприятий.

## Схема / Workflow
Zoom webhook триггер на завершение встречи → Получение списка участников через Zoom API → Трансформация данных → Сохранение в Airtable базу

## Технические связки
- n8n → Zoom API
- Zoom → Airtable
- webhook → автоматический workflow

## Мясо
- Использование Zoom webhook для автоматического срабатывания при завершении встречи
- Zoom API предоставляет полный список участников с метаданными (время подключения, длительность присутствия)
- Airtable используется как централизованная база для управления контактами и анализа посещаемости
- Автоматизация исключает ручной экспорт и обработку данных участников
- n8n позволяет трансформировать формат данных перед загрузкой в таблицу
