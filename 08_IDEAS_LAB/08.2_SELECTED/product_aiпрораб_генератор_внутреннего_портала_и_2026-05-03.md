## 1. Название
**AI-Прораб** — Генератор внутреннего портала из переписки

## 2. Суть
Агент еженедельно собирает данные из Telegram-чатов, CRM и мессенджеров, Claude синтезирует из них структурированный отчёт по проектам — что сделано, что зависло, кто виноват — и публикует в виде интерактивного HTML-дашборда без покупки дорогих BI-систем.

## 3. Бизнес-сценарий
ДО: Руководитель проектного офиса в пятницу вечером вручную собирает статусы с 5 менеджеров, копипастит в Excel, делает сводку для акционеров — 3 часа каждую пятницу. Данные неполные, менеджеры забывают ответить. ИИ делает: в пятницу в 17:00 агент сам собирает данные из Telegram-чатов команды, Bitrix24 и Google Sheets. Claude формирует отчёт: «Проект А: 78% выполнен, дедлайн 20 мая — в норме. Проект Б: 3 задачи просрочены на 5 дней, ответственный — Козлов, причина не указана. Риск: высокий.» Генерирует HTML-страницу с цветовой индикацией. Клиент получает: отчёт без единого вопроса к команде — прямо в Telegram-боте.

## 4. Логика
```
Cron (пятница 17:00) → n8n собирает данные из Bitrix24 + Telegram + Google Sheets → Claude анализирует и пишет статус-отчёт → n8n генерирует HTML-дашборд → публикует на VPS + ссылка руководителю
```

## 5. Технический стек
- n8n
- claude-sonnet-4-5
- Bitrix24 REST API
- Google Sheets API
- Telegram Bot API
- Nginx (VPS для HTML)

## 6. Связки инструментов
```
n8n Cron → Bitrix24 REST (задачи, сделки, просрочки) + Google Sheets (план/факт) + Pyrogram (чаты) → Claude (синтез отчёта + цветовая оценка рисков) → n8n рендерит HTML → загружает на VPS → Telegram-ссылка руководителю
```

## 7. Алгоритм реализации
1. Подключить Bitrix24 REST API: забирать все задачи со статусами, дедлайнами и ответственными за прошедшие 7 дней через метод tasks.task.list
2. Параллельно забрать данные из Google Sheets (план/факт по проектам) и последние 500 сообщений из 3–5 ключевых Telegram-чатов через Pyrogram
3. Передать всё в Claude с промптом: для каждого проекта определить статус (зелёный/жёлтый/красный), выявить блокеры, назвать конкретного ответственного за просрочки, написать 1–2 предложения прогноза
4. n8n принимает JSON от Claude и рендерит HTML-шаблон с таблицами, индикаторами RAG (red-amber-green) и графиком прогресса — загружает файл на VPS через SSH
5. Отправить руководителю в Telegram сообщение с кнопкой «Открыть дашборд» (ссылка на HTML) и текстовым саммари на 5 строк прямо в чате

## 8. Почему купят (ROI)
3 часа еженедельного сбора статусов × 4 недели = 12 часов/мес руководителя (стоимость от 30 000 до 120 000 руб./мес в зависимости от ставки) — и всё равно картина неполная, потому что менеджеры врут или молчат. AI-Прораб агрегирует факты из систем, а не из слов людей. Крючок: «Отчёт без совещания — данные не врут, люди врут.»
