# 🗂️ Паспорт идеи: ИИ-агент продаж для локального бизнеса

**Источник:** [Sell AI Automation to Local Businesses (Beginner Step-by-Step Guide)](https://youtube.com/watch?v=iUNzRKOC25w) — Jason Wardrop (2024-08-30)
**Категория:** Замена должности
**Заменяет:** Менеджер по продажам / SDR (первичная квалификация лидов)

---

## 💡 Что это даёт бизнесу
Автоматизирует первичный контакт с входящими заявками: отвечает на запросы, квалифицирует лидов и записывает на встречу без участия человека. Малый бизнес получает круглосуточную обработку лидов без найма менеджера. Студия Трофимов упаковывает это как белый продукт на базе HighLevel/amoCRM + ИИ.

---

## ✅ Плюсы
- Массовый спрос — каждый локальный бизнес (стоматология, автосервис, юристы) теряет лиды из-за медленного ответа
- Быстрая упаковка в повторяемый продукт — один раз настроил, продаёшь разным нишам
- Понятный ROI для клиента: замена 1 менеджера (~60 000 руб./мес) окупает внедрение за 2–3 месяца

## ❌ Минусы / Риски
- Видео ориентировано на западный рынок (HighLevel слабо распространён в РФ) — нужна адаптация под amoCRM/Bitrix24
- Высокая конкуренция в нише ИИ-чатботов для малого бизнеса, сложно удерживать маржу

## 🚀 Шаги запуска
1. Шаг 1: Выбрать 2–3 целевые ниши (стоматологии, автосервисы, юридические бюро) и описать типовой сценарий квалификации лида
2. Шаг 2: Собрать MVP на n8n: входящий вебхук из формы/WhatsApp → GPT-4o квалифицирует → создаёт сделку в amoCRM → отправляет ссылку на запись
3. Шаг 3: Упаковать в продукт с брифом, SLA и онбордингом; подготовить коммерческое предложение с калькулятором ROI
4. Шаг 4: Запустить пилот с 2–3 клиентами, собрать кейс, масштабировать через партнёрства с маркетинговыми агентствами
