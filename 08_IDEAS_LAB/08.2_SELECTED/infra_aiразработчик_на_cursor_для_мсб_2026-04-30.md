# 🗂️ Паспорт идеи: AI-разработчик на Cursor для МСБ

**Источник:** [Cursor 2.0 is here... 5 things you didn&#39;t know it can do](https://youtube.com/watch?v=HIp8sFB2GGw) — Fireship (2025-10-30)
**Категория:** Замена должности
**Заменяет:** Junior/Middle разработчик, фрилансер на разовые задачи

---

## 💡 Что это даёт бизнесу
Студия Трофимов может предложить клиентам МСБ быструю разработку и доработку веб-приложений, ботов, интеграций силами 1 специалиста с Cursor вместо найма команды разработчиков. Скорость итераций возрастает в 3–5 раз, стоимость разработки для клиента снижается на 40–60% по сравнению с классическим аутсорсом. Клиент получает работающий MVP или автоматизацию за 2–4 недели вместо 2–3 месяцев.

---

## ✅ Плюсы
- Резкое снижение себестоимости разработки — один специалист Студии закрывает объём работы 2–3 разработчиков
- Быстрый вывод продукта: новые фичи Cursor 2.0 (агентный режим, мультифайловое редактирование, встроенный браузер) сокращают ручной труд до минимума
- Легко продать клиенту как 'разработку с ИИ-ускорением' — понятный маркетинговый аргумент для МСБ

## ❌ Минусы / Риски
- Cursor — инструмент для технических специалистов: если в Студии нет хотя бы одного разработчика, внедрить некому
- Качество результата сильно зависит от промптинга и надзора человека — галлюцинации в коде могут стоить дороже, чем сэкономили

## 🧰 Артефакты для реализации

**Инструменты:** Cursor 2.0 (IDE), Claude 3.5 Sonnet (встроен в Cursor), PostHog (аналитика, упомянута в видео), n8n / Make.com (для интеграций)
**API:** Cursor API (Agent mode), Anthropic Claude API, OpenAI API

**Репозитории:**
- https://github.com/getcursor/cursor

**Документация:**
- https://cursor.sh/docs
- https://www.anthropic.com/docs

## 🚀 Шаги запуска
1. Шаг 1: Установить Cursor 2.0, оформить Pro-подписку, изучить агентный режим и мультифайловое редактирование (1–2 дня)
2. Шаг 2: Собрать внутреннюю библиотеку промптов для типовых задач Студии: интеграции CRM, Telegram-боты, n8n-воркфлоу (1 неделя)
3. Шаг 3: Провести пилот на реальном клиентском проекте — зафиксировать экономию времени и сравнить с предыдущими аналогичными проектами
4. Шаг 4: Упаковать как услугу 'Быстрая разработка с ИИ' — отдельный прайс, кейс, маркетинг для привлечения клиентов с задачами на разработку
