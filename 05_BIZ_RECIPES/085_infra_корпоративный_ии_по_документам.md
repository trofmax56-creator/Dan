---

## 💡 Что это даёт бизнесу
Сотрудники задают вопросы в чат и мгновенно получают ответы из внутренней базы знаний компании (регламенты, инструкции, договоры, FAQ). Исключает звонки «а где найти форму?» и потери времени на поиск документов. Новый сотрудник адаптируется в 2 раза быстрее без нагрузки на HR.

---

## ✅ Плюсы
- Высокий спрос: боль есть у любой компании от 10 человек с накопленной документацией
- Стек n8n + Pinecone + OpenAI хорошо известен Студии, быстрая сборка MVP за 1–2 недели
- Легко масштабируется: добавляются новые источники (Notion, Google Drive, Confluence, PDF)

## ❌ Минусы / Риски
- Pinecone и OpenAI API — зарубежные сервисы, клиент может требовать российский аналог (риск замены на YandexGPT + локальный векторный поиск)
- Качество ответов зависит от структуры документов клиента: если база хаотичная — потребуется предобработка, что увеличивает Dev

**Инструменты:** n8n, Pinecone, OpenAI API (GPT-4o / text-embedding-3-small), Telegram Bot или веб-чат

## 🚀 Шаги запуска
1. Шаг 1: Собрать документы клиента (PDF, Word, Notion-экспорт) → загрузить в n8n, разбить на чанки, создать эмбеддинги и залить в Pinecone
2. Шаг 2: Построить RAG-цепочку в n8n: входящий вопрос → векторный поиск по Pinecone → формирование промпта с контекстом → ответ GPT-4o
3. Шаг 3: Подключить интерфейс — Telegram-бот или встраиваемый веб-чат (n8n Chat Widget)
4. Шаг 4: Настроить пайплайн обновления базы знаний (автоматическая переиндексация при добавлении новых файлов) + сдать клиенту с инструкцией
