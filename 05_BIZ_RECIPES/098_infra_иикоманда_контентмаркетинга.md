# 🗂️ Паспорт идеи: ИИ-команда контент-маркетинга

**Источник:** [I Built An Entire AI Marketing Team With Claude Code In 16 Minutes](https://youtube.com/watch?v=eorc3jLBqIA) — Zubair Trabzada | AI Workshop (2026-03-03)
**Категория:** Замена должности
**Заменяет:** Контент-менеджер, SMM-специалист, копирайтер (частично маркетолог)

---

## 💡 Что это даёт бизнесу
Клиент получает автономную систему на Claude+n8n, которая генерирует тексты для соцсетей, email-рассылок и блога по расписанию без участия человека. Снижает ФОТ на 60–120 тыс. руб./мес за счёт отказа от 1–2 контент-специалистов. Система адаптирует тон под бренд и публикует материалы напрямую в Bitrix24/amoCRM или через webhook в соцсети.

---

## ✅ Плюсы
- Высокий и понятный ROI для клиента — экономия на зарплате окупает за 1–2 месяца
- Claude API даёт качественный русскоязычный контент без дообучения
- Легко упаковать в тиражируемый продукт для ниш: ритейл, недвижимость, медклиники

## ❌ Минусы / Риски
- Клиент должен потратить 3–5 часов на онбординг: описать бренд-войс, целевую аудиторию, запрещённые темы
- Высокий риск «контентного мусора» без регулярного аудита промптов — нужна поддержка

## 🚀 Шаги запуска
1. Шаг 1: Провести аудит текущего контент-процесса клиента — какие форматы, частота, каналы, кто делает вручную
2. Шаг 2: Разработать систему промптов под бренд клиента (тон, рубрики, длина) и собрать n8n-пайплайн: триггер по расписанию → Claude → публикация
3. Шаг 3: Интегрировать с CRM (Bitrix24/amoCRM) для персонализации контента под сегменты клиентской базы
4. Шаг 4: Запустить пилот на 2 недели, собрать обратную связь, передать на абонентское обслуживание
