# 🗂️ Паспорт идеи: ИИ-колл-центр для отдела продаж

**Источник:** [AI Call-центр 2026: Как ИИ-звонки экономят 70% бюджета на отдел продаж](https://youtube.com/watch?v=lQFtNoPYbm4) — Александр Прошин | IT и AI для бизнеса (2026-01-11)
**Категория:** Замена должности
**Заменяет:** Операторы входящих/исходящих звонков, менеджеры первичной квалификации лидов

---

## 💡 Что это даёт бизнесу
Голосовой ИИ-агент обрабатывает входящие и совершает исходящие звонки 24/7 без найма операторов, экономя клиенту 60–70% ФОТ отдела продаж. Бот квалифицирует лиды, отвечает на типовые вопросы и передаёт горячих клиентов живому менеджеру или сразу пишет в CRM. Внедрение окупается за 1–3 месяца даже при команде из 3–5 операторов.

---

## ✅ Плюсы
- Острая боль: колл-центры и отделы продаж — самая дорогая статья ФОТ у МСБ
- Быстрая окупаемость для клиента — легко считается ROI и продаётся через калькулятор
- Масштабируемый продукт: один шаблон адаптируется под разные ниши (недвижимость, клиники, авто)

## ❌ Минусы / Риски
- Высокая конкуренция: рынок голосовых ботов перегрет, много дешёвых SaaS-решений (Звонобот, Томору и др.)
- Качество русскоязычного TTS/STT ещё нестабильно — клиенты жалуются на 'роботость', нужна тщательная настройка сценариев

## 🧰 Артефакты для реализации

**Инструменты:** Voiceflow / Vapi.ai — конструктор голосовых агентов, n8n — оркестрация сценариев и интеграция с CRM, Bitrix24 / amoCRM — приём данных от бота, Twilio / Манго Телеком / Zadarma — SIP-телефония
**API:** Vapi.ai API (голосовой агент), OpenAI Realtime API / GPT-4o (логика диалога), Yandex SpeechKit (альтернатива для русского языка), Bitrix24 REST API / amoCRM API

**Репозитории:**
- vapi-ai/vapi-python — официальный SDK Vapi
- n8n-io/n8n — self-hosted автоматизация

**Документация:**
- https://docs.vapi.ai
- https://platform.openai.com/docs/guides/realtime
- https://cloud.yandex.ru/docs/speechkit/

## 🚀 Шаги запуска
1. Шаг 1: Выбрать пилотную нишу (например, стоматологии или агентства недвижимости), собрать типовые сценарии звонков и возражений
2. Шаг 2: Собрать MVP на Vapi.ai + n8n: входящий звонок → квалификация → запись в CRM → уведомление менеджеру
3. Шаг 3: Провести пилот с 1–2 клиентами на реальном трафике, замерить конверсию и % отказов, докрутить сценарии
4. Шаг 4: Упаковать в тиражируемый продукт с фиксированной ценой внедрения по нишам, запустить лидген через кейс с цифрами ROI
