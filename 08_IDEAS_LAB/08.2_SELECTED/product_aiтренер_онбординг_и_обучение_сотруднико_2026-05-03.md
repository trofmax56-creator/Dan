## 1. Название
**AI-Тренер** — Онбординг и обучение сотрудников

## 2. Суть
Интерактивный Telegram-бот, который проводит нового сотрудника через программу онбординга: задаёт вопросы, объясняет регламенты, принимает тесты и фиксирует прогресс в Bitrix24 — без участия HR и руководителя.

## 3. Бизнес-сценарий
ДО: Новый агент выходит — руководитель бросает текущие задачи на 3 дня, объясняет одно и то же в сотый раз. → ИИ делает: Бот шаг за шагом ведёт через 8 модулей, задаёт вопросы, проверяет, адаптирует объяснение при ошибке, фиксирует прогресс. Агент пишет: «Не понял, как работает воронка в amoCRM» — бот отвечает: «Объясню иначе: представь воронку как трубу...» → Клиент получает: Руководитель освобождает 3 дня × 4 онбординга в год = 12 рабочих дней; новый сотрудник приносит первую сделку на 10 дней раньше.

## 4. Логика
```
HR добавляет нового сотрудника в систему → n8n создаёт персональный трек обучения → Telegram-бот ведёт по модулям → Claude оценивает тесты → прогресс в Bitrix24 → итоговый отчёт руководителю
```

## 5. Технический стек
- n8n
- claude-sonnet-4-6
- Telegram Bot API
- Bitrix24
- Google Sheets (контент модулей)

## 6. Связки инструментов
```
Bitrix24 (новый сотрудник) → n8n webhook → Telegram Bot (старт онбординга) → Claude API (оценка ответов + адаптивное объяснение) → Bitrix24 REST (обновление прогресса) → Telegram (уведомление руководителю)
```

## 7. Алгоритм реализации
1. Создать контент 8 модулей в Google Sheets: модуль, название, текст объяснения, ссылка на материал, 3 тестовых вопроса с правильными ответами; n8n читает этот справочник как источник правды
2. При создании нового контакта в Bitrix24 (тег 'новый сотрудник') — n8n стартует онбординг: отправляет первое сообщение в Telegram с приветствием и модулем 1
3. Бот последовательно: отправляет материал → ждёт 'Готов к тесту' → задаёт 3 вопроса по одному → ответ передаёт в Claude с эталонным ответом из Sheets → Claude оценивает (правильно/нет + объяснение если неверно)
4. При прохождении теста ≥70% — разблокировать следующий модуль; при <70% — повторное объяснение другими словами (Claude перефразирует материал), повторный тест через 30 минут; 3 провала подряд — уведомление руководителю
5. После завершения всех модулей: n8n обновляет статус в Bitrix24, отправляет руководителю PDF-отчёт с результатами по каждому модулю, средним баллом и выявленными слабыми местами

## 8. Почему купят (ROI)
Боль: каждый онбординг стоит руководителю 3 рабочих дня = 24 часа × 3 000 руб./час = 72 000 руб. упущенного времени. При текучке 30% в год для команды 10 человек = 3 онбординга = 216 000 руб./год только потерянного времени руководителя. Плюс агент без нормального онбординга теряет первые 2 сделки из-за незнания скриптов = ещё 100 000–200 000 руб. Крючок: «Первые 5 дней сотрудника определяют следующие 5 месяцев — пусть их проведёт AI-Тренер, а не ваш вечер.»
