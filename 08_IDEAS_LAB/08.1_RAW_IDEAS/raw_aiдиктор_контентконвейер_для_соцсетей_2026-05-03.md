## 1. Название
**AI-Диктор** — Контент-Конвейер для соцсетей

## 2. Суть
n8n-автомат забирает RSS/темы из таблицы, прогоняет через Claude, генерирует готовые посты с картинками и публикует в Telegram-канал бизнеса по расписанию — без участия маркетолога.

## 3. Бизнес-сценарий
ДО: Владелец кофейни каждый день в 9 утра открывает телефон и думает: 'Что бы сегодня запостить?' Через час либо ничего не выходит, либо выходит скучное фото с подписью 'Доброе утро'. Клиенты не реагируют, охваты падают. ИИ делает: В 8:00 n8n берёт из Google Sheets список тем недели ('акция на капучино', 'новинка — матча латте'), отправляет в Claude промпт 'напиши живой пост для Instagram кофейни в Казани, 150 слов, с призывом прийти сегодня', параллельно генерирует обложку через Kandinsky API. В 9:00 пост уходит в Telegram с картинкой. Клиент получает: 7 постов в неделю без единого действия, единый tone-of-voice, охваты растут потому что регулярность появилась.

## 4. Логика
```
Google Sheets (темы) + Cron 08:00 → n8n (сборка промпта) → Claude (текст поста) + Kandinsky API (картинка) → Telegram Bot API (публикация) → Google Sheets (отметка 'опубликовано')
```

## 5. Технический стек
- n8n
- claude-sonnet-4-5
- Kandinsky API (Сбер)
- Telegram Bot API
- Google Sheets

## 6. Связки инструментов
```
Google Sheets → n8n Cron → Claude API (генерация текста) → Kandinsky 6.0 API (генерация картинки) → Telegram Bot API (публикация)
```

## 7. Алгоритм реализации
1. Создать Google Sheets с 3 колонками: дата, тема поста, статус (пусто/опубликовано) — заполнять клиент будет 1 раз в неделю на 7 дней вперёд
2. Настроить Cron-триггер в n8n на 08:00 ежедневно, который берёт строку с сегодняшней датой из таблицы и передаёт тему в следующий узел
3. Написать системный промпт для Claude: роль SMM-редактора конкретной ниши, ограничение 120–160 слов, обязательный призыв к действию, запрет штампов — передать тему как user-сообщение
4. Параллельно отправить HTTP-запрос в Kandinsky API с кратким описанием визуала (извлекает Claude отдельным вызовом), получить URL картинки
5. Отправить пост + картинку в Telegram через sendPhoto, после успеха обновить статус строки в Google Sheets на 'опубликовано', при ошибке — уведомление владельцу в личный Telegram

## 8. Почему купят (ROI)
Владелец МСБ тратит 1–2 часа в день на контент или платит SMM-специалисту 30 000–50 000 руб./мес — и всё равно выходит нерегулярно. AI-Диктор даёт 30 постов в месяц за 3 000 руб. инфраструктуры. Крючок: 'Ваш SMM никогда не болеет, не уходит в отпуск и не просит повышения.'
