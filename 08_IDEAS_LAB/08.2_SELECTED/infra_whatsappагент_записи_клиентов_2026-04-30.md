# 🗂️ Паспорт идеи: WhatsApp-агент записи клиентов

**Источник:** [Build a Whatsapp AI Agent for appointment handling (n8n Tutorial) #n8n #aiautomation](https://youtube.com/watch?v=vcvRVlc_VFg) — Ed Hill | AI Automation (2025-06-04)
**Категория:** Замена должности
**Заменяет:** Администратор / оператор записи

---

## 💡 Что это даёт бизнесу
ИИ-агент в WhatsApp принимает заявки на запись 24/7, проверяет доступные слоты, подтверждает и переносит записи без участия человека. Клиент Студии сокращает минимум 1 ставку администратора и перестаёт терять заявки в нерабочее время. Скорость ответа — секунды, конверсия из обращения в запись растёт за счёт мгновенной реакции.

---

## ✅ Плюсы
- WhatsApp — основной мессенджер в РФ, барьер входа для клиентов нулевой
- n8n разворачивается on-premise или в облаке, данные остаются у клиента
- Универсальный сценарий: клиники, салоны, фитнес, юристы, автосервисы — широкий рынок

## ❌ Минусы / Риски
- Официальный WhatsApp Business API платный (через провайдеров: Waba360, Green API и др.) и требует верификации — у некоторых МСБ есть сопротивление
- Интеграция с calendar/CRM у каждого клиента своя — кастомизация съедает время и усложняет тиражирование

## 🚀 Шаги запуска
1. Шаг 1: Выбрать провайдера WhatsApp API (Green API для старта — дешевле), зарегистрировать номер клиента, получить токен
2. Шаг 2: Собрать n8n-воркфлоу: триггер входящего сообщения → парсинг намерения через LLM (запись / перенос / отмена / вопрос) → проверка слотов в календаре/CRM
3. Шаг 3: Реализовать логику подтверждения: запись в календарь, отправка подтверждения клиенту, уведомление мастера/менеджера
4. Шаг 4: Тест на пилотном клиенте (салон/клиника), снять метрику конверсии и сэкономленных часов, упаковать в кейс для продаж
