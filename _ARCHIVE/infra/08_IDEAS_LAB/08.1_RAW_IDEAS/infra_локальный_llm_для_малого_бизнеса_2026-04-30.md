# 🗂️ Паспорт идеи: Локальный LLM для малого бизнеса

**Источник:** [How to Run LLMs Locally - Full Guide](https://youtube.com/watch?v=km5-0jhv0JI) — Tech With Tim (2025-12-19)
**Категория:** Инфраструктура
**Заменяет:** —

---

## 💡 Что это даёт бизнесу
Клиент получает собственный языковой ИИ без передачи данных в облако — подходит для работы с конфиденциальными документами, договорами, клиентской базой. Снижает зависимость от OpenAI и валютных рисков: инфраструктура оплачивается в рублях. Студия может упаковать это как 'ИИ-ассистент внутри компании' и продавать как инфраструктурный проект с поддержкой.

---

## ✅ Плюсы
- Нет утечки данных в облако — критично для юристов, медицины, госконтрактов
- Нет валютной зависимости от OpenAI/Anthropic — предсказуемые рублёвые расходы
- Повторяемый продукт: одна инсталляция → шаблон для следующего клиента

## ❌ Минусы / Риски
- Локальные LLM (Llama 3, Mistral) заметно слабее GPT-4o — клиент может быть разочарован качеством
- Требует железа на стороне клиента (GPU-сервер) или аренды мощного VPS — порог входа высокий для МСБ

## 🧰 Артефакты для реализации

**Инструменты:** Ollama, LM Studio, Open WebUI, n8n (для интеграции в бизнес-процессы), AnythingLLM
**API:** Ollama REST API (localhost:11434), OpenAI-совместимый API через Ollama

**Репозитории:**
- ollama/ollama (github.com/ollama/ollama)
- open-webui/open-webui (github.com/open-webui/open-webui)
- Mintplex-Labs/anything-llm (github.com/Mintplex-Labs/anything-llm)

**Документация:**
- ollama.com/docs
- docs.openwebui.com
- hub.docker.com/r/ollama/ollama

## 🚀 Шаги запуска
1. Шаг 1: Аудит клиента — какие данные нельзя в облако, какие задачи нужно автоматизировать (чат с документами, саппорт, генерация текстов)
2. Шаг 2: Развернуть Ollama + Open WebUI на GPU VPS (Selectel/Timeweb Cloud) или локальном сервере клиента, выбрать модель (Llama 3.1 8B / Mistral 7B / GigaChat-совместимая)
3. Шаг 3: Подключить RAG через AnythingLLM или самописный пайплайн — загрузить документы клиента (регламенты, договоры, база знаний)
4. Шаг 4: Интегрировать с Bitrix24/amoCRM через n8n — чтобы менеджеры использовали ИИ прямо в CRM, передать и обучить сотрудников
