## 1. Название
**AI-Контент-Завод** — Автопилот публикаций

## 2. Суть
ИИ-система, которая по расписанию сама генерирует, адаптирует и публикует SEO-статьи, посты в соцсети и рассылки — без участия маркетолога. Контент-план, написание, верстка и дистрибуция — полный цикл на автопилоте.

## 3. Бизнес-сценарий
ДО: Владелец интернет-магазина говорит: «Мы два месяца не публиковали статьи — просто не до этого, вся команда на заказах». SEO проседает, трафик падает. ИИ делает: каждое утро в 9:00 n8n запускает сценарий — Claude берёт ключевые слова из таблицы, пишет статью 1500–3000 слов, адаптирует в 3 варианта (блог / ВКонтакте / Telegram), публикует через API. Клиент получает: 30 единиц контента в месяц без единого редактора в штате. Директор заходит в Яндекс.Вебмастер и видит: позиции растут, а он ничего не делал.

## 4. Логика
```
Cron (расписание) → n8n читает ключи из Google Sheets → Claude генерирует статью + 3 адаптации → публикация в CMS/VK/Telegram → отчёт в Telegram директору
```

```
Google Sheets (контент-план) → n8n Cron → Claude (генерация + адаптация) → WordPress REST API + VK API + Telegram Bot → Яндекс.Метрика API (отчёт)
```

## 7. Алгоритм реализации
1. Создать Google Sheets с колонками: ключевое слово / тема / рубрика / целевая страница / статус — клиент заполняет 1 раз
2. Настроить n8n Cron-триггер на 09:00 ежедневно: читать 1 строку со статусом 'Очередь', передать в Claude с промптом SEO-статьи 1500–3000 слов + структурой H1/H2/H3
3. Claude генерирует основной текст, затем второй вызов Claude адаптирует в пост для ВКонтакте (800 знаков) и Telegram (500 знаков) с эмодзи
4. n8n публикует: WordPress REST API (статья) + VK API (пост с отложкой +2 ч) + Telegram sendMessage (канал) + обновляет статус в Sheets на 'Опубликовано'
5. Раз в неделю (понедельник 10:00) n8n вызывает Яндекс.Метрика API, собирает топ-10 страниц по трафику за 7 дней, отправляет PDF-сводку директору в Telegram

## 8. Почему купят (ROI)
Контент-маркетолог в штате — от 80 000 руб./мес, фрилансер — от 3 000 руб./статья × 30 статей = 90 000 руб./мес. AI-Контент-Завод закрывает тот же объём за 8 000–12 000 руб./мес инфраструктуры. Итого экономия от 78 000 руб. ежемесячно. Крючок: «Ваш контент-маркетолог спит — завод работает».
