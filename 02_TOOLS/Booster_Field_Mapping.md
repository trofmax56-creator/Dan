# 🗺️ Маппинг полей: AI-Sales Booster (v1.3)

## 1. Унификация Телефона (Normalization)
Перед поиском в CRM, узел n8n нормализует номер: всегда 11 цифр, начинающихся на 7.

## 2. Структура JSON (Выход из Claude 3.5 Sonnet)
```json
{
  "name": "Имя",
  "phone": "7XXXXXXXXXX",
  "budget": 50000, // Optional
  "crm_interest": "Bitrix24 / amoCRM", // Optional
  "pain": "Суть запроса",
  "next_question": "Вопрос для доквалификации"
}
```

## 3. Соответствие полей в CRM
- Имя: Поле «Имя».
- Телефон: «Рабочий телефон» (Ключ).
- Бюджет: «Бюджет» сделки.
- Интерес к CRM: «Какая CRM (AI)».
- Боль (AI): «Суть запроса (AI)».
- Следующий шаг: Примечание в ленту.
