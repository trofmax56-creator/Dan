## 1. Название
**AI-Сценарист** — Генератор контента бренда

## 2. Суть
Claude изучает стиль, лексику и структуру постов конкретного бренда или канала — и генерирует тексты, рекламные объявления и сценарии «голосом компании», а не безликим корпоративным языком.

## 3. Бизнес-сценарий
ДО: Маркетолог интернет-магазина тратит 4 часа на написание 5 постов для ВКонтакте. Или отдаёт копирайтеру 15 000 руб./мес, а получает тексты, которые «не звучат как мы». Директор читает и правит каждый пост, потому что «не то настроение». ИИ делает: маркетолог один раз выгружает JSON экспорт Telegram-канала или 50 лучших постов бренда и передаёт Claude через промпт: «Проанализируй все тексты, вытащи характерные обороты, частотную лексику, структуру постов, типичные анкоры и эмодзи.» Claude формирует профиль голоса бренда. Дальше: «Напиши 5 постов про осеннюю коллекцию в нашем стиле» — и получает тексты, которые директор принимает с первого раза. Клиент получает: 20 постов за 30 минут вместо 4 часов, стоимость копирайтера падает с 15 000 до 0 руб./мес.

## 4. Логика
```
Выгрузка контента бренда → Claude (анализ стиля + создание профиля голоса) → n8n (шаблонизация запросов) → Claude (генерация контента) → Google Docs / Notion (контент-план)
```

## 5. Технический стек
- n8n
- claude-opus-4-7
- Telegram Bot API
- Google Docs API
- Notion API

## 6. Связки инструментов
```
Google Drive / Telegram JSON Export (исходные тексты) → Claude API (анализ стиля, создание Brand Voice профиля) → n8n (обработка запросов на генерацию) → Claude API (генерация под бриф) → Google Docs (контент-план на месяц)
```

## 7. Алгоритм реализации
1. Собрать 50–100 лучших текстов бренда (посты, письма, рекламные объявления) в один документ; передать в Claude с промптом анализа голоса бренда — на выходе получить профиль: тон, характерные конструкции, запрещённые слова, структура текста, длина
2. Сохранить Brand Voice профиль как системный промпт в n8n — он будет подставляться к каждому запросу на генерацию автоматически
3. Создать Telegram-бота для маркетолога: принимает бриф (тема + ключевые тезисы + площадка), передаёт в n8n, Claude генерирует 3 варианта текста в стиле бренда
4. Настроить еженедельный автоматический контент-план: n8n в понедельник отправляет запрос Claude на 5 тем для постов на неделю (на основе актуального сезона и товарной матрицы из Google Sheets), результат записывает в Google Docs
5. Добавить модуль анализа конкурентов по инсайту из TochkiNadAI_2295: Claude собирает жалобы из отзывов конкурентов и превращает каждое возражение в рекламный крючок для бренда клиента

## 8. Почему купят (ROI)
Штатный копирайтер обходится 40 000 – 80 000 руб./мес, при этом пишет в своём стиле, болеет и уходит. Фриланс-копирайтер — 15 000–30 000 руб. и 3–5 дней ожидания на каждый пакет текстов. AI-Сценарист создаёт 20 постов за 30 минут голосом именно вашего бренда, доступен 24/7 и дешевеет с каждым месяцем использования. Крючок: «Копирайтер пишет своим голосом. ИИ пишет вашим.»
