## 1. Название
**AI-Ревизор** — Контроль 100% звонков

## 2. Суть
Каждый записанный звонок менеджера автоматически транскрибируется, Claude анализирует по чеклисту скрипта продаж и выдаёт оценку с конкретными ошибками — РОП получает отчёт без прослушивания.

## 3. Бизнес-сценарий
ДО: В компании по продаже окон 6 менеджеров делают по 15 звонков в день = 90 звонков. РОП Сергей физически прослушивает 5–7 звонков в неделю — это 1% от потока. Остальные 99% звонков никто не контролирует. Менеджер Вася каждый раз забывает назвать имя, не выявляет боль, сразу называет цену — и РОП об этом не знает. ИИ делает: звонок завершён, UIS автоматически отправляет запись в n8n. Whisper транскрибирует за 40 секунд. Claude анализирует: «Менеджер не представился (нарушение п.1 скрипта), цену назвал на 2-й минуте без выявления потребности (нарушение п.3), не предложил замер (нарушение п.7). Итоговый скор: 3/10». РОП получает в Telegram: сводку по всем 90 звонкам за день, топ-3 нарушения недели, рейтинг менеджеров.

## 4. Логика
```
UIS/МТС Exolve (запись звонка) → n8n Webhook → Whisper API (транскрипция) → Claude (анализ по чеклисту скрипта) → Telegram РОПу (отчёт) + Bitrix24 (оценка к сделке)
```


## 7. Алгоритм реализации
1. Подключить UIS или МТС Exolve к n8n через Webhook: триггер — звонок длиннее 60 секунд завершён, получить URL записи
2. Скачать аудиофайл, отправить в Whisper API (self-hosted или OpenAI), получить текстовый транскрипт с таймкодами
3. Передать транскрипт в Claude с системным промптом — чеклист из 10 пунктов скрипта продаж клиента: проверить каждый пункт, выставить 0/1, сформировать скор и список нарушений
4. При скоре < 5 автоматически поставить задачу в Bitrix24: 'Разобрать звонок с менеджером [имя]' с дедлайном 24 часа и прикрепить транскрипт
5. В 18:00 собирать дневную сводку: количество звонков, средний скор по каждому менеджеру, топ-3 системных нарушения, отправлять РОПу в Telegram

## 8. Почему купят (ROI)
Отдел контроля качества из 2 человек стоит 120 000–160 000 руб./мес ФОТ и всё равно прослушивает максимум 15–20% звонков. AI-Ревизор анализирует 100% звонков за 40 секунд каждый, стоит в 10 раз дешевле. Главное: проблемы скрипта видны через неделю, а не через квартал. Крючок: «РОП думает, что контролирует продажи. На самом деле он контролирует 1%.»
