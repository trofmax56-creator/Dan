# 🗂️ Паспорт идеи: Self-hosted n8n автоматизация на VPS

**Источник:** [n8n will change your life as a developer...](https://youtube.com/watch?v=bS9R6aCVEzw) — Fireship (2025-09-29)
**Категория:** Инфраструктура

---

## 💡 Что это даёт бизнесу
Клиент получает собственный сервер автоматизации на базе n8n без ежемесячной подписки облачного тарифа, полный контроль над данными и процессами. Студия разворачивает готовую инфраструктуру под ключ: VPS + n8n + базовые воркфлоу для CRM, уведомлений, лидогенерации. Экономия клиента на облачном n8n Cloud составляет от 3 000 до 15 000 руб./мес при объёме операций от 10 000 в месяц.

---

## ✅ Плюсы
- Низкий порог входа: n8n open-source, деплой на Hostinger/Timeweb за 1-2 дня
- Высокий повторный спрос: клиент платит за каждый новый воркфлоу и интеграцию
- Полный контроль данных — критично для B2B в России (152-ФЗ)

## ❌ Минусы / Риски
- Видео маркетинговое (sponsorship Hostinger), глубокой бизнес-кейсовой базы нет — придётся формировать свои кейсы с нуля
- Низкий чек разовой продажи инфраструктуры; маржа растёт только на сопровождении и воркфлоу

## 🧰 Артефакты для реализации

**Инструменты:** n8n (self-hosted), Docker + Docker Compose, Nginx reverse proxy, Timeweb Cloud / Hostinger VPS
**API:** Bitrix24 REST API, amoCRM API, Telegram Bot API, OpenAI API

**Репозитории:**
- https://github.com/n8n-io/n8n
- https://github.com/n8n-io/self-hosted-ai-starter-kit

**Документация:**
- https://docs.n8n.io/hosting/installation/docker/
- https://docs.n8n.io/integrations/

## 🚀 Шаги запуска
1. Шаг 1: Подготовить типовой Docker Compose шаблон (n8n + PostgreSQL + Nginx) с документацией — 3 дня
2. Шаг 2: Собрать библиотеку из 10-15 готовых воркфлоу под типовые задачи РФ-бизнеса (лиды из ВК/Telegram → CRM, авто-счета, уведомления) — 5-7 дней
3. Шаг 3: Упаковать в оффер 'n8n под ключ за 3 дня' с фиксированной ценой, продвигать через Telegram-канал и партнёров Bitrix/amoCRM — 1 неделя
4. Шаг 4: Подключить клиента на ежемесячный retainer: мониторинг + 2 новых воркфлоу в месяц — ongoing
