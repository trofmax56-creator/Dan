## Главная мысль
**Anthropic выпустили Claude Design** 📝

Новый продукт от Anthropic Labs - возможность создавать дизайн, прототипы, презентации и визуальные материалы вместе с Claude.

## Схема / Workflow
1. На онбординге Claude строит дизайн-систему из вашего кодa и файлов - цвета, типографика, компоненты применяются автоматически
2. Импорт откуда угодно - текст, изображения, DOCX, PPTX, XLSX, или прямо из кодовой базы
3. Web capture - захватывает элементы прямо с вашего сайта, чтобы прототипы выглядели как реальный продукт
4. Точечные правки - комментарии на элементах, прямое редактирование текста, слайдеры для spacing/color/layout
