# 🏛️ Справочник ГИПа: Объект "Комплекс 20 метров" (v6.1 Final)
**Соответствие стандарту:** ГОСТ Р 21.101-2026 (СПДС)

## 1. Общие данные
- **Местоположение:** РФ, г. Курск.
- **Климатические параметры:** Снеговой район III (180 кг/м²), Ветровой район II.
- **Габаритные размеры:** 20.0 х 10.0 метров (в осях).

## 2. Архитектурно-планировочные решения
Объект разделен на 4 функциональных блока с единым тепловым контуром глубиной 6.0м и интегрированной террасой 4.0м.

### Экспликация помещений:
1. **Навес (6х10м):** Открытая стоянка для 2-х автомобилей.
2. **Котельная (3х4м):** Технический блок, вход с торца (левая стена).
3. **Склад (3х6м):** Холодный склад, вход с торца + выход на террасу.
4. **Беседка (5х6м):** Закрытая витражная зона с раздвижной системой.
5. **Терраса (5х4м):** Центральный рекреационный хаб (углубленный).
6. **Баня (6х10м):** 
   - Зона отдыха (6х4м) с панорамным остеклением.
   - Помывочная (3х6м).
   - Сауна/Парная (3х6м) в дальнем углу.

## 3. Конструктивные решения
- **Фундамент:** Монолитная плита. Деформационный шов по оси "Навес / Бытовка".
- **Стены:** Пеноблок D600 (300мм). Фасадная стена (20м) — ГЛУХАЯ.
- **Кровля:** Двухскатная на МЗП-фермах. Покрытие: Клик-фальц RAL 7024.

## 4. Ведомость материалов (Спецификация)
- Бетон М300: 60.0 м³
- Пеноблок D600: 65.5 м³
- Клик-фальц: 270.0 м²
- Арматура 12мм: 2.5 т
