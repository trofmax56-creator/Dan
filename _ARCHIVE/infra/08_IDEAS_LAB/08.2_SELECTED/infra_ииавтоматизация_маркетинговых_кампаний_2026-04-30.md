# 🗂️ Паспорт идеи: ИИ-автоматизация маркетинговых кампаний

**Источник:** [Marketing Operations AI Demo: Campaign Management, Asset Production &amp; Brand Compliance Automation](https://youtube.com/watch?v=nQcjWUzUk8U) — Airtable (2026-02-05)
**Категория:** Автоматизация отдела
**Заменяет:** Маркетолог-операционист / трафик-менеджер

---

## 💡 Что это даёт бизнесу
Клиент получает единую систему управления маркетинговыми кампаниями: от брифа до готовых материалов с автоматической проверкой брендбука. ИИ генерирует тексты, баннеры и посты по шаблонам, согласует их с правилами бренда без участия человека. Сокращается цикл запуска кампании с 3–5 дней до нескольких часов.

---

## ✅ Плюсы
- Устраняет рутину согласований и правок — экономия 20–40 ч/мес на одного маркетолога
- Контроль brand compliance автоматически: ИИ проверяет цвета, шрифты, тон коммуникации
- Масштабируется без найма: один маркетолог управляет объёмом на 3–4 человека

## ❌ Минусы / Риски
- Требует качественного брифинга и оцифрованного брендбука — у большинства МСБ их нет, нужна подготовительная работа
- Высокий порог доверия: клиенты боятся отдавать маркетинг 'роботу', нужна демонстрация + пилот

## 🧰 Артефакты для реализации

**Инструменты:** n8n (оркестрация воркфлоу), Make.com (альтернатива), Airtable или Notion (база кампаний), Claude API / GPT-4o (генерация текстов), Canva API / Adobe Express API (генерация визуалов), Bitrix24 / amoCRM (интеграция со сделками)
**API:** Claude API (Anthropic) — генерация текстов и проверка brand voice, OpenAI GPT-4o API — альтернатива, Canva Connect API — генерация визуальных материалов, Airtable REST API — управление базой кампаний, Telegram Bot API / VK API — публикация контента

**Репозитории:**
- n8n-io/n8n (github.com/n8n-io/n8n)

## 🚀 Шаги запуска
1. Шаг 1. Аудит клиента: собрать брендбук, список каналов, типовые форматы кампаний, текущий стек инструментов (2–3 дня)
2. Шаг 2. Построить базу данных кампаний в Airtable/Notion + настроить шаблоны брифов и чеклисты brand compliance (3–5 дней)
3. Шаг 3. Собрать воркфлоу в n8n: триггер → Claude генерирует тексты → Canva API создаёт визуалы → автопроверка по правилам бренда → уведомление маркетологу на согласование (5–10 дней)
4. Шаг 4. Интегрировать с CRM (Bitrix24/amoCRM) для привязки кампаний к сделкам/сегментам, провести обучение команды клиента, запустить пилот на одном канале (3–5 дней)
