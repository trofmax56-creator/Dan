## 🥇 ИТОГ 21 — Кандидаты в следующий BIZ_RECIPE

### AI-воркфлоу: воронка + отчёты + превью (JCfD-gJHsA8)
- **Суть:** Один AI-workflow автоматизирует три задачи: управление sales-воронкой, генерация отчётов и создание превью контента.
- **Применение:** Маркетинговые агентства и команды продаж.
- **Стек:** n8n + Claude + CRM API + Canva/DALL-E

### 3000 готовых AI-агентов n8n — архив (Lox-nmxNwyk)
- **Суть:** Коллекция из 3000 предобученных n8n-шаблонов для автоматизации. Развернуть готовый агент за минуты.
- **Применение:** Быстрый старт для клиентов без разработки с нуля.
- **Действие:** Скачать архив → сделать каталог топ-20 шаблонов для нашего стека

### HubSpot CRM очистка дублей (P6x8EmrfvX8)
- **Суть:** n8n ежедневно находит и удаляет дубликаты контактов в HubSpot, валидирует данные, нормализует поля.
- **Применение:** Компании с HubSpot — аналог рецепта для Bitrix24.
- **Стек:** n8n + HubSpot API

### 5000 шаблонов n8n на GitHub (TFY5Z-52yjQ)
- **Суть:** Публичный GitHub-репозиторий с 5000+ готовыми n8n-шаблонами — бесплатная библиотека.
- **Действие:** Изучить, отобрать 20–30 шаблонов под CRM-задачи, добавить в наш каталог.
- **Ссылка:** Искать в GitHub по тегу `n8n-workflows`

### Лендинг за 7 минут: Claude Design + Perplexity (rcDsMsU_W9Q)
- **Суть:** Claude Design генерирует дизайн лендинга, Perplexity находит контент — готовая страница за 7 минут.
- **Применение:** Быстрые MVP-лендинги для клиентов под новые продукты.
- **Стек:** Claude Design + Perplexity + Vercel/Netlify

## 🥈 ИТОГ 20 — Перспективные идеи

### No-code AI workflow за несколько минут (UPooW4VoXo8)
- **Суть:** Low-code платформа позволяет основателям создавать AI-workflow без разработчика за 5–10 минут.
- **Применение:** Быстрые MVP для тестирования гипотез перед полноценной автоматизацией.
- **Стек:** n8n / Make.com + Claude API

### Локальный AI-сервер n8n + DeepSeek (fCLSH21s2cs)
- **Суть:** Замена ChatGPT/Claude подпискам — DeepSeek на локальном сервере через Ollama. n8n обращается к нему вместо облачного API. Экономия до ₽20k/мес.
- **Применение:** Клиенты с большим объёмом запросов, требования конфиденциальности.
- **Стек:** n8n + Ollama + DeepSeek R1

### 5 open-source агентов заменяющих SaaS (laxRbudJRIg)
- **Суть:** Пять бесплатных AI-агентов, которые заменяют платные SaaS-подписки: SEO-аудит, email-рассылки, CRM, аналитика, генерация контента.
- **Действие:** Собрать сравнительную таблицу open-source vs SaaS, предложить клиентам аудит расходов.

### n8n скрытые техники масштабирования (oi7JtLjUapo)
- **Суть:** Продвинутые паттерны n8n для крупных workflow: параллельное выполнение, очереди, обработка ошибок, retry-логика.
- **Применение:** Когда базовый n8n не справляется с нагрузкой клиента.
- **Действие:** Выписать топ-10 техник → добавить в CLAUDE.md для использования при разработке

### ETL автоматизация с AI-агентами (xwI6hP67ZAE)
- **Суть:** AI-агенты заменяют ручные ETL-процессы: автоматически трансформируют и грузят данные из разных источников (БД, API, CSV) без написания скриптов.
- **Применение:** Аналитические компании, бизнес с множеством разрозненных источников данных.
- **Стек:** n8n + Claude + PostgreSQL
