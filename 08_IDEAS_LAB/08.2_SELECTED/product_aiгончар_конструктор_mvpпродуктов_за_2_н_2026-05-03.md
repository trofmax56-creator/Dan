## 1. Название
**AI-Гончар** — Конструктор MVP-продуктов за 2 недели

## 2. Суть
Студия берёт бизнес-задачу клиента, Claude Code проектирует архитектуру и генерирует рабочий веб-инструмент или Telegram-бот с базой данных — без штатных разработчиков, за 10–14 рабочих дней.

## 3. Бизнес-сценарий
ДО: Владелец сети автосервисов хочет внутренний портал учёта заказов — штатный разработчик просит 500 000 руб. и 3 месяца. Или берут готовый SaaS за 25 000 руб./мес, но он не подходит под процессы. ИИ делает: клиент приходит в Студию с брифом: «Нужен инструмент: мастер вносит работы, клиент видит статус через Telegram-бот, владелец видит выручку по точкам.» За 2 дня Claude проектирует архитектуру и UI. За 10 дней — рабочий MVP с авторизацией, базой данных и Telegram-ботом. Демо клиенту на 12-й день. Клиент получает: собственный продукт под свои процессы за 150 000 руб. вместо 500 000 + владеет исходниками.

## 4. Логика
```
Бриф клиента → Claude Code проектирует архитектуру + генерирует код → ручная проверка инженером Студии → деплой на VPS клиента → сдача
```

## 5. Технический стек
- claude-code
- Python/FastAPI или Node.js
- PostgreSQL
- Telegram Bot API
- Nginx
- VPS (Selectel/Timeweb)

## 6. Связки инструментов
```
Claude Code (генерация кода) → GitHub (версионирование) → VPS Selectel (деплой) → Telegram Bot API (пользовательский интерфейс) → PostgreSQL (хранение данных)
```

## 7. Алгоритм реализации
1. Провести 2-часовую сессию с клиентом: зафиксировать 5–7 ключевых пользовательских сценариев, составить техническое задание на 1–2 страницы
2. Claude Code проектирует схему базы данных, API-эндпоинты и UI-компоненты — инженер Студии проверяет архитектуру и утверждает с клиентом за 1 день
3. Генерировать код итерациями по модулям: сначала авторизация и БД, затем основная логика, затем Telegram-бот / веб-интерфейс — каждый модуль тестируется сразу
4. Инженер проводит code review каждого модуля, исправляет ошибки Claude, тестирует на реальных данных клиента (импорт тестовой выборки)
5. Задеплоить на VPS клиента, настроить бэкапы раз в сутки, обучить 1–2 администраторов клиента за 2 часа, передать документацию и исходники

## 8. Почему купят (ROI)
Кастомная разработка у классических студий стоит 400 000–1 500 000 руб. и занимает 3–6 месяцев. Готовые SaaS-решения не закрывают специфику бизнеса и стоят 15 000–40 000 руб./мес бессрочно (180 000–480 000 руб./год). AI-Гончар даёт собственный продукт за 120 000–200 000 руб. и 2 недели — клиент владеет кодом и не платит подписку. Крючок: «Раньше своё ПО было только у корпораций. Теперь — у любого МСБ.»
