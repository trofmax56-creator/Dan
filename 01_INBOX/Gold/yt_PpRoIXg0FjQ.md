## Суть
Создание полностью автоматизированного AI-отдела продаж для B2B-агентств, который генерирует и квалифицирует горячих лидов без привлечения живых сотрудников. Система использует AI для анализа целевой аудитории, поиска контактов и отправки персонализированных предложений в LinkedIn или Email.

## Бизнес-сценарий
B2B-агентства, фрилансеры и владельцы SaaS-продуктов, которые хотят масштабировать генерацию лидов без найма sales-менеджеров. Система автоматически ищет потенциальных клиентов в целевой нише, анализирует их профили и отправляет персонализированные сообщения для холодных контактов.

## Алгоритм реализации
1. Шаг 1: Настройка данных о целевой аудитории — в Google Sheets создаёте лист с параметрами: названия компаний из целевой ниши, количество сотрудников, регион, область деятельности, сумма контракта
2. Шаг 2: Подключение API для поиска контактов (RocketReach, Hunter.io или similar) — настраиваете параметры поиска по фильтрам компании и позиции (Decision Maker)
3. Шаг 3: Отправка данных в AI-модель (Claude/ChatGPT через API) — система анализирует найденные контакты и на основе AI генерирует персонализированные сообщения для каждого лида
4. Шаг 4: Логирование результатов в базу данных — создаёте запись о статусе лида (найден контакт, сообщение отправлено, получен ответ)
5. Шаг 5: Интеграция с CRM (Bitrix24/amoCRM) — автоматически создаются карточки контактов и сделок на основе найденных лидов
6. Шаг 6: Отслеживание результатов и метрик — система мониторит открытия писем, клики по ссылкам, ответы и автоматически обновляет статусы в CRM

## Технический стек
- n8n (оркестрация workflow)
- Claude API или ChatGPT API (генерация персонализированных сообщений)
- RocketReach API или Hunter.io API (поиск контактов)
- Google Sheets API (хранение данных о целевых компаниях)
- LinkedIn API или Email API (отправка сообщений)
- Bitrix24 API или amoCRM API (интеграция CRM)
- Google Mail API или SendGrid (отправка email)
- Node.js или Python (для обработки логики если используется кастомный код)
- Webhook (для двусторонней синхронизации с внешними сервисами)
- PostgreSQL или MongoDB (логирование и хранение истории контактов)

## Связки инструментов
- Google Sheets → n8n → Claude API → Генерация сообщения → Hunter.io API → Поиск email → SendGrid → Отправка email → Bitrix24 API → Создание сделки
- LinkedIn → Webhook → n8n → Parser → Claude → Анализ профиля → Генерация сообщения → LinkedIn DM
- RocketReach API → n8n → Фильтрация по параметрам → Claude API → Персонализация → Email API → SendGrid → Отправка

## Конфигурация и параметры
- Узел 'HTTP Request' в n8n: метод POST к Claude API endpoint https://api.anthropic.com/v1/messages с headers Authorization: Bearer {API_KEY}, Content-Type: application/json
- Body параметры для Claude: model = 'claude-3-sonnet-20240229', max_tokens = 1000, messages = [{role: 'user', content: 'Напиши персонализированное письмо для {имя} из компании {название}'}]
- Узел 'Google Sheets': настройка доступа через OAuth, выбрать таблицу с ID, диапазон A1:E1000 для чтения списка компаний
- Узел 'HTTP Request' для Hunter.io: GET https://api.hunter.io/v2/domain-search?domain={домен}&type=personal с параметром limit=10
- Узел 'Set variable': сохранение найденных email-ов в переменные для дальнейшего использования
- Узел 'SendGrid' или 'SMTP': настройка From address (sender@domain.com), To address ({{email}}), Subject ({{subject_ai}}), Body ({{message_ai}})
- Узел 'Bitrix24': метод 'crm.contact.add' с параметрами NAME, PHONE, EMAIL, COMPANY_ID; затем 'crm.deal.add' с TITLE, CONTACT_ID, OPPORTUNITY (сумма сделки), STAGE_ID
- Условие 'IF': проверка найден ли email (email != empty), если нет — переход к альтернативному способу поиска
- Webhook input: URL вида https://n8n.yourdomain.com/webhook/linkedin_response для приёма данных о ответах на сообщения

## Ключевые инсайты
- Персонализация через AI экономит 80% времени на написание писем вручную — одна ноида Claude API генерирует 100+ уникальных сообщений в час
- Двухуровневая фильтрация (сначала по компании, потом по должности Decision Maker) повышает конверсию холодных писем на 40-60%
- Использование RocketReach более дорогое (~$100/месяц) но на 80% точнее Hunter.io для B2B контактов крупных компаний
- Интеграция ответов обратно в workflow (Webhook) создаёт замкнутый цикл feedback для улучшения генерируемых сообщений
- Кэширование результатов поиска контактов в базе данных (не запрашивать API дважды на одну компанию) экономит 40% расходов на API
- Стоимость автоматизации на n8n: $40/месяц (Basic) + Hunter/RocketReach ($100-300) + Claude/ChatGPT API (~$50 за 100k запросов) = ~$200-400/месяц, что дешевле одного junior sales-менеджера в 5-10 раз
- Настройка Rate Limit в n8n (параметр 'Wait') между запросами к API Hunter предотвращает блокировку на 429 ошибку
- Subject-line генерируется отдельно через prompt который ссылается на индустрию и боль компании для повышения open rate писем
- Логирование всех попыток отправки (успешно/ошибка/no-email) позволяет отслеживать качество источника контактов и ROI автоматизации
