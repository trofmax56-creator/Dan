## 1. Название
**AI-Летописец** — Автоматический База-Знаний из YouTube и видео

## 2. Суть
Агент смотрит YouTube-ролики и видеофайлы, вытаскивает структурированные знания (инструкции, кейсы, факты) и автоматически пополняет корпоративную базу знаний в Notion или Google Docs — без ручного конспектирования.

## 3. Бизнес-сценарий
ДО: Методолог учебного центра смотрит 10 видео с конференции, на каждое пишет конспект 1,5 часа. Итого 15 часов ручного труда за неделю. Половина знаний оседает только в голове и теряется. ИИ делает: методолог скидывает в Telegram-бот ссылку на YouTube-плейлист или папку с видео. Агент скачивает транскрипции, Claude анализирует каждое видео и пишет структурированную карточку: «Тема: Работа с возражениями. Ключевые тезисы: 5 штук. Применимые инструменты: 3. Цитата спикера: "Клиент говорит нет — значит не понимает ценность." Связанные материалы: [ссылка].» Клиент получает: базу знаний, которая пополняется сама, пока команда работает.

## 4. Логика
```
Telegram Bot (ссылка на видео/плейлист) → n8n скачивает транскрипцию через YouTube API / Whisper → Claude структурирует знания → Notion API создаёт карточку
```

## 5. Технический стек
- n8n
- claude-sonnet-4-5
- Telegram Bot API
- YouTube Data API
- OpenAI Whisper (для не-YouTube)
- Notion API

## 6. Связки инструментов
```
Telegram Bot → n8n Webhook → YouTube Transcript API или Whisper STT → Claude (извлечение тезисов, тегов, цитат) → Notion API (создание страницы в нужной БД) → Telegram-подтверждение автору
```

## 7. Алгоритм реализации
1. Настроить Telegram-бот: принимать YouTube-ссылки, плейлисты и видеофайлы до 500 МБ, маршрутизировать в n8n Webhook
2. Для YouTube: забирать автоматическую транскрипцию через youtube-transcript-api; для видеофайлов: транскрибировать через Whisper API (точность >90% для русского)
3. Claude получает транскрипт и системный промпт: извлечь тему, 5–7 ключевых тезисов, практические инструменты, 1–2 цитаты, теги по предметной области клиента
4. n8n создаёт страницу в Notion через API в нужной базе данных, заполняет поля: Title, Tags, Summary, Key Points, Source URL, Date Added
5. Отправить в Telegram автору ссылку на созданную карточку Notion с кнопкой «Редактировать» — цикл замыкается за 3–5 минут после отправки ссылки

## 8. Почему купят (ROI)
Компания накапливает знания из видео со скоростью 1,5 часа конспекта на 1 час видео — при 40 часах обучающего контента в год это 60 часов ручного труда (≈ 36 000 руб. по ставке 600 руб./час). 80% этих знаний не доходит до команды и теряется. AI-Летописец обрабатывает 1 час видео за 5 минут и сразу кладёт в общую базу. Крючок: «Знания компании не должны умирать вместе с конспектом одного сотрудника.»
