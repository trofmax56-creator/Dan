# 🗂️ Паспорт идеи: ИИ-агент контент-маркетинга на Claude+n8n

**Источник:** [How to INSTANTLY Build AI Agents in N8N Using Claude](https://youtube.com/watch?v=uAtSMEBosGU) — Nolan Harper | Ai Automation  (2025-05-23)
**Категория:** Замена должности
**Заменяет:** Контент-менеджер / копирайтер

---

## 💡 Что это даёт бизнесу
Агент автоматически генерирует, редактирует и публикует контент (посты, статьи, описания) по расписанию без участия человека. Клиент получает стабильный поток материалов в соцсети, блог и рассылки за 5–10% стоимости штатного сотрудника. Интеграция с CRM (Bitrix24/amoCRM) позволяет персонализировать контент под сегменты базы.

---

## ✅ Плюсы
- Боль массовая: малый бизнес не может позволить штатного контент-менеджера, но контент нужен всем
- Стек n8n + Claude API — знакомый для Студии, быстрый старт без новых компетенций
- Легко упаковать в тиражируемый продукт с ежемесячной подпиской

## ❌ Минусы / Риски
- Claude API дорожает при большом объёме генерации — нужно закладывать лимиты и мониторинг токенов
- Качество контента требует начальной настройки промптов под нишу клиента, иначе текст 'пластиковый' — репутационный риск

## 🚀 Шаги запуска
1. Шаг 1: Собрать типовой контент-план клиента (рубрики, тон, ЦА) → зашить в системный промпт Claude
2. Шаг 2: Построить n8n-workflow: триггер по расписанию → Claude генерирует пост → модерация (авто или ручная кнопка в Telegram) → публикация в канал/соцсеть
3. Шаг 3: Подключить CRM-сегменты для персонализации (например, разные тексты для новых лидов и существующих клиентов)
4. Шаг 4: Настроить мониторинг токенов и алерты, упаковать в коммерческое предложение с пилотом на 30 дней
