## 1. Название
**AI-Голос** — Автоответчик в мессенджерах

## 2. Суть
ИИ-агент на базе Claude мгновенно отвечает клиентам в WhatsApp/Telegram 24/7, квалифицирует их по 5 параметрам и сам заполняет карточку сделки в Битрикс24 или amoCRM — без участия менеджера.

## 3. Бизнес-сценарий
ДО: Клиент пишет в WhatsApp в 22:47: «Сколько стоит сделать сайт под ключ?» Менеджер видит сообщение утром, клиент уже ушёл к конкурентам. ИИ делает: через 4 секунды после сообщения бот отвечает: «Привет! Я помогу рассчитать стоимость. Подскажи, это для какого бизнеса? Уже есть структура сайта или начинаем с нуля?» После 3–4 вопросов Claude извлекает бюджет, срок, тип задачи, ЛПР ли собеседник — и создаёт сделку в CRM со скором квалификации. Менеджер утром видит: «Горячий лид, бюджет 150к, нужен лендинг к пятнице, ЛПР.» Клиент получает: ощущение живого общения ночью и уверенность, что его услышали.

## 4. Логика
```
WhatsApp/Telegram сообщение → Wazzup Webhook → n8n → Claude (квалификация по 5 критериям) → Bitrix24 REST API (создание сделки + заполнение полей)
```

## 5. Технический стек
- n8n
- claude-opus-4-7
- Wazzup
- Bitrix24
- amoCRM

## 6. Связки инструментов
```
Wazzup → n8n Webhook → Claude Sonnet (извлечение параметров из диалога) → Bitrix24 REST API (crm.deal.add + crm.deal.update)
```

## 7. Алгоритм реализации
1. Подключить Wazzup к n8n через Webhook, настроить фильтр входящих сообщений по номеру канала
2. Создать системный промпт Claude с ролью «менеджер по продажам», прописать 5 квалификационных критериев (бюджет, срок, тип задачи, ЛПР, контакт) и сценарий разговора на 3–5 шагов
3. Настроить хранение истории диалога в Redis или PostgreSQL (session_id = номер телефона), чтобы Claude помнил контекст в рамках одного чата
4. По завершении квалификации (получены 4 из 5 параметров) автоматически создать сделку в Bitrix24 через REST API с заполнением кастомных полей и тегом «AI-квалифицирован»
5. Отправить уведомление ответственному менеджеру в Telegram с карточкой лида и ссылкой на сделку в CRM; при скоре < 3 из 5 — уведомить РОПа отдельно

## 8. Почему купят (ROI)
Менеджер обрабатывает лид в среднем через 4,5 часа после обращения — конверсия падает в 10 раз по сравнению с ответом за 5 минут (данные InsideSales). При 80 лидах в месяц и потере 30% из-за долгого ответа бизнес теряет 24 потенциальных клиента. При среднем чеке 50 000 руб. = 1 200 000 руб./год уходит молча. Крючок: «Ваш конкурент ответил на 4 секунды быстрее — и забрал клиента, пока ваш менеджер спал.»
