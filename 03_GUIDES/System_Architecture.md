# 🏛 Архитектура Системы "Dan | Второй Мозг"

## 📂 Описание папок и процессов

### 1. 📥 00_RAW (Сырые данные)
- **Цель:** Сбор всей входящей информации.
- **Автоматизация:** 
  - `parser.py`: 17 TG-каналов.
  - `youtube_parser.py`: 10 YouTube-каналов.
- **Инструкция для Claude:** "Анализируй только свежие файлы (последние 24ч). Ищи практические кейсы".

### 2. 📬 01_INBOX (Входящие)
- **Цель:** Хранение итоговых дайджестов и отфильтрованных идей.
- **Процесс:** Данила формирует дайджест дважды в день (12:00 и 22:00).

### 3. 🛠 02_TOOLS (Инструменты)
- **Цель:** Хранение промптов и настроек.
- **Ключевой файл:** `Bitrix_Prompts.md` (шаблоны ответов для CRM).

### 4. 💡 05_BIZ_RECIPES (Бизнес-рецепты)
- **Цель:** Библиотека проверенных решений n8n + Bitrix24 + AI.

### 5. 🏗 07_CONSTRUCTION (Инженерная зона)
- **Цель:** Проектирование и расчеты (навес, баня, беседка).

## 🤖 Роли Ассистента (Данилы)
1. **Интегратор:** Работает с папками 00, 01, 02, 05. Фокус на бизнесе.
2. **Инженер:** Работает с папкой 07. Фокус на расчетах и стройке.

## 📜 Главные правила
- Общение только на русском языке.
- Порог 5 кредитов: предупреждать о дорогих операциях.
- Приоритет API (n8n/Gmail) над браузером.
