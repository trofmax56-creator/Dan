## 1. Название
**AI-Диспетчер** — Автоматический Маршрутизатор Заявок

## 2. Суть
ИИ-агент круглосуточно принимает входящие сообщения в WhatsApp/Telegram, квалифицирует запрос по типу задачи и срочности, затем автоматически создаёт задачу в Битрикс24 и уведомляет нужного сотрудника — без участия администратора.

## 3. Бизнес-сценарий
ДО: клиент пишет в WhatsApp компании в 22:00 «Нужна срочная доставка завтра утром, 50 позиций». Администратор увидит сообщение только утром, менеджер узнает ещё позже — заявка уже у конкурента. ИИ делает: мгновенно отвечает «Привет! Принял заявку. Уточните адрес доставки и время?», получает ответ, парсит параметры (срочность, объём, контакт), создаёт сделку в Битрикс24 со статусом «Срочно» и отправляет пуш ответственному менеджеру. Клиент получает: подтверждение через 2 минуты после сообщения, ощущение что компания работает 24/7.

## 4. Логика
```
WhatsApp-сообщение → Wazzup Webhook → n8n → Claude (извлечение параметров + классификация) → Bitrix24 REST API (создание сделки) → Telegram-уведомление менеджеру
```

## 5. Технический стек
- n8n

## 6. Связки инструментов
```
Wazzup → n8n Webhook → Claude (классификация по 4 типам + срочность) → Bitrix24 REST API (сделка + задача) → Telegram Bot (пуш менеджеру)
```

## 7. Алгоритм реализации
1. Подключить Wazzup webhook к n8n, настроить фильтр входящих сообщений по номеру WhatsApp компании
2. Написать системный промпт для Claude: извлекать 6 параметров (тип запроса, срочность 1-3, контактное лицо, суть задачи, предпочтительное время связи, источник)
3. Создать в Bitrix24 сделку через REST API с заполненными полями, присваивать теги 'срочно' при приоритете ≥ 2
4. Отправлять Telegram-уведомление ответственному менеджеру с карточкой: имя клиента, суть, ссылка на сделку в Битриксе — в течение 90 секунд после входящего
5. Настроить fallback: если Claude не смог извлечь параметры с уверенностью > 80% — уведомить дежурного администратора с оригинальным текстом

## 8. Почему купят (ROI)
Компания теряет 15-25% заявок, пришедших вне рабочего времени или в пиковую нагрузку — при среднем чеке 30 000 руб. и 20 заявках в месяц это 90 000–150 000 руб. упущенной выручки ежемесячно. AI-Диспетчер обрабатывает 100% входящих за 90 секунд в любое время суток. Крючок: «Ваш лучший администратор не болеет, не спит и не уходит в отпуск.»
