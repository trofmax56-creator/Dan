## Суть
Видео рассказывает о миграции бизнес-клиента из AmoCRM в Bitrix24 из-за высоких затрат на интеграции. Клиент устал платить за каждую новую функциональность и ищет более универсальное решение с встроенными возможностями.

## Бизнес-сценарий
Владельцы и менеджеры CRM-систем, которые работают с AmoCRM и ищут альтернативу из-за дорогостоящих интеграций. Задача — понять, почему Bitrix24 может быть экономичнее и какие процессы автоматизируются в обеих системах без дополнительных платежей.

## Алгоритм реализации
1. Шаг 1: Анализ текущей архитектуры в AmoCRM — определение всех используемых интеграций и их стоимости (платежные системы, маркетинговые сервисы, системы аналитики, коммуникационные каналы)
2. Шаг 2: Оценка бизнес-задач, для которых требуются платные интеграции в AmoCRM (отправка уведомлений, сбор данных из внешних источников, синхронизация с другими сервисами)
3. Шаг 3: Исследование встроенных возможностей Bitrix24 — какие из необходимых функций уже реализованы нативно без дополнительных платежей
4. Шаг 4: Создание плана миграции данных из AmoCRM в Bitrix24 (экспорт контактов, сделок, задач, истории коммуникаций)
5. Шаг 5: Настройка автоматизаций и workflow в Bitrix24 для замены платных интеграций из AmoCRM
6. Шаг 6: Тестирование всех процессов в Bitrix24 и проверка сохранения функциональности после миграции

## Технический стек
- AmoCRM — исходная CRM-система с платными интеграциями
- Bitrix24 — целевая CRM с более широким функционалом из коробки
- API AmoCRM — для экспорта данных
- API Bitrix24 — для импорта данных и настройки автоматизаций
- Платежные шлюзы (если используются в интеграциях)
- Email-сервисы и SMS-провайдеры для коммуникаций
- Системы аналитики для отслеживания метрик

## Связки инструментов
- AmoCRM API → Экспорт контактов и сделок
- Экспортированные данные → Битрикс24 API → Импорт в новую систему
- Битрикс24 встроенные интеграции → Замена платных сервисов из AmoCRM
- Workflow Bitrix24 → Автоматизация бизнес-процессов без дополнительных платежей

## Конфигурация и параметры
- AmoCRM: Интеграции, требующие оплаты (Telegram, Whatsapp, Email, SMS, платежные системы, аналитика)
- Bitrix24: Встроенный функционал — телефония, email, Telegram, Whatsapp (через встроенные каналы), задачи, календарь, документооборот
- Битрикс24 тарифные планы — различные уровни с разным функционалом
- API ключи для подключения интеграций в обеих системах
- Параметры синхронизации между системами — сопоставление полей и типов данных

## Ключевые инсайты
- AmoCRM требует платежей за интеграции, которые в Bitrix24 встроены в систему по умолчанию
- Bitrix24 имеет в стандартной комплектации: телефонию, видеоконференции, встроенные мессенджеры (Telegram, Whatsapp через витрину приложений)
- Экономия на интеграциях в Bitrix24 может составить 50-70% от текущих расходов на AmoCRM
- При миграции критично правильно сопоставить все кастомные поля и автоматизации из AmoCRM
- Bitrix24 позволяет создавать более сложные workflow с использованием JavaScript и встроенного конструктора
- Стоимость интеграций в AmoCRM растёт пропорционально количеству подключаемых сервисов
- Битрикс24 — более локализованное решение для русского рынка с поддержкой русских платежных систем
- Время внедрения Bitrix24 может быть дольше, но долгосрочные затраты ниже

## Подводные камни
- Экспорт из AmoCRM может быть ограничен — проверьте лимиты API на выгрузку контактов и сделок
- Bitrix24 имеет другую логику работы с сделками и воронками — требуется переучивание команды
- Некоторые специализированные интеграции, которые были в AmoCRM, могут отсутствовать в стандартном Bitrix24 (нужно смотреть витрину приложений)
- При миграции могут потеряться некоторые данные, если неправильно настроены соответствия полей
- Telemetry и аналитика в Bitrix24 работает иначе, чем в AmoCRM — отчеты нужно переделывать
- Интеграция с внешними сервисами в Bitrix24 может потребовать кодирования (JavaScript, REST API)
- Пиковые нагрузки на API Bitrix24 при импорте больших объёмов данных — нужно импортировать поэтапно
- Необходимо проверить совместимость существующих интеграций третьих сторон с Bitrix24
