## 1. Название
**AI-Некромант** — Реаниматор спящей базы

## 2. Суть
Автоматическая серия персонализированных сообщений в WhatsApp для клиентов, которые были в CRM, но не покупали более 60 дней — Claude пишет каждое сообщение с учётом истории сделки, менеджер не тратит ни минуты.

## 3. Бизнес-сценарий
ДО: В amoCRM автосалона «АвтоПлюс» 340 контактов со статусом «не дозвонились» или «думает» — и все они мертвы уже 3–6 месяцев. Менеджер Катя говорит: «Я туда не звоню, они всё равно не ответят». РОП знает, что в базе деньги, но нет инструмента. ИИ делает: n8n раз в неделю выгружает из amoCRM контакты без активности 60+ дней, Claude для каждого пишет персональное сообщение: «Иван, добрый день. Три месяца назад вы смотрели Kia Sportage в комплектации Luxe. Сейчас у нас появилась похожая позиция с пробегом 12 000 км, цена на 180 000 ниже нового. Хотите — пришлю фото?» Wazzup отправляет сообщения по 20 в час. Клиент получает: личное сообщение, а не спам-рассылку.

## 4. Логика
```
n8n Cron (еженедельно) → amoCRM API (выгрузка спящих 60+ дней) → Claude (персональный текст по истории сделки) → Wazzup API (отправка 20 сообщений/час) → amoCRM (обновление статуса)
```

## 6. Связки инструментов
```
n8n Cron → amoCRM REST API (фильтр: нет активности 60 дней) → Claude API (персонализация по полям сделки) → Wazzup API (очередь отправки) → amoCRM REST API (смена этапа на 'реанимация')
```

## 7. Алгоритм реализации
1. Настроить n8n Cron на понедельник 10:00, выгружать из amoCRM сделки: статус 'не дозвонились' / 'думает', дата последнего изменения > 60 дней
2. Для каждого контакта передать в Claude поля: имя, интересующий продукт, последнее действие, дата — промпт: напиши WhatsApp-сообщение до 3 предложений, без шаблонности, с конкретным предложением
3. Сформировать очередь отправки: не более 20 сообщений в час через Wazzup API для обхода ограничений платформы
4. Через 48 часов проверить статус: если нет ответа — отправить второе сообщение с другим углом (например, упомянуть акцию или ограниченный остаток)
5. Тех, кто ответил, автоматически перевести в этап 'повторный интерес' в amoCRM и поставить задачу менеджеру с дедлайном 2 часа

## 8. Почему купят (ROI)
Средняя стоимость нового лида в авто/недвижимости/строительстве — 3 000–15 000 руб. В спящей базе из 300 контактов уже потрачено 900 000–4 500 000 руб. на привлечение — и эти деньги просто лежат мёртвым грузом. Реанимация 5% базы = 15 сделок из воздуха. При среднем чеке 200 000 руб. — это 3 000 000 руб. выручки. Крючок: «Вы уже заплатили за этих клиентов. Осталось только добудиться.»
