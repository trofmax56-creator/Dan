# 🗂️ Паспорт идеи: ИИ-автоматизация строительного бизнеса

**Источник:** [ИИ в строительстве увеличивает прибыль. Автоматизация строительства](https://youtube.com/watch?v=wuUwDziAwn8) — Виталий Грицаль (2025-12-22)
**Категория:** Автоматизация отдела
**Заменяет:** Сметчик, менеджер по заявкам, прораб-диспетчер (частично)

---

## 💡 Что это даёт бизнесу
Строительная компания получает ИИ-ассистента, который обрабатывает входящие заявки, составляет предварительные сметы и контролирует статусы объектов без участия офисных сотрудников. Сокращение времени на обработку одного клиента с 2–4 часов до 10–15 минут. Экономия 1–2 ставок административного персонала при обороте от 5 млн руб./мес.

---

## ✅ Плюсы
- Строительный рынок в России — миллионы малых подрядчиков, острая боль с хаосом в заявках и сметах
- Легко упаковывается в типовой продукт: CRM (Bitrix24) + ИИ-чат + шаблонный сметный модуль
- Клиенты видят ROI уже в первый месяц — легко продавать через кейс экономии

## ❌ Минусы / Риски
- Строительная смета требует глубокой отраслевой кастомизации под каждый вид работ — высок риск ошибок ИИ, нужна верификация человеком
- Консервативная аудитория: строители медленно принимают цифровые решения, цикл сделки длинный

## 🧰 Артефакты для реализации

**Инструменты:** Bitrix24 или amoCRM — CRM-ядро, n8n — оркестрация сценариев, Claude API или GPT-4o — обработка заявок и генерация смет, Telegram Bot API — входящий канал заявок, Google Sheets / Notion — база нормативов и расценок
**API:** Anthropic Claude API, OpenAI GPT-4o API, Telegram Bot API, Bitrix24 REST API

## 🚀 Шаги запуска
1. Шаг 1: Провести CustDev с 5–7 малыми строительными подрядчиками — зафиксировать топ-3 боли (заявки, сметы, контроль объектов)
2. Шаг 2: Собрать MVP — Telegram-бот принимает заявку, n8n передаёт в Claude для квалификации и предварительной сметы, результат падает в Bitrix24 как сделка
3. Шаг 3: Пилот с 1–2 клиентами за символическую цену (30–50 тыс. руб.), снять метрики экономии времени
4. Шаг 4: Упаковать в тиражируемый продукт с кейсом ROI, запустить продажи через отраслевые Telegram-каналы и партнёров
