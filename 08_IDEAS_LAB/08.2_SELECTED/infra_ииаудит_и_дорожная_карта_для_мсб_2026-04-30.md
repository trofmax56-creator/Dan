# 🗂️ Паспорт идеи: ИИ-аудит и дорожная карта для МСБ

**Источник:** [AI для бизнеса в 2025: жизнь или смерть?](https://youtube.com/watch?v=7-mIu2XeCOo) — НЕЙРОЭРА | Абрамова (2025-08-24)
**Категория:** ROI/Экономика
**Заменяет:** Бизнес-консультант по цифровизации

---

## 💡 Что это даёт бизнесу
Клиент получает автоматизированный аудит своих бизнес-процессов с приоритизированным планом внедрения ИИ и конкретными ROI-расчётами. Студия продаёт не «внедрение ради внедрения», а упакованный продукт с понятной окупаемостью — это снижает барьер входа для МСБ. Владелец бизнеса за 3–5 дней получает документ с конкретными шагами вместо месяцев раздумий.

---

## ✅ Плюсы
- Низкий порог входа для клиента (50–80к руб.) — легко продаётся как первый шаг
- Создаёт pipeline: после аудита клиент естественно покупает внедрение у Студии
- Можно частично автоматизировать сам аудит через анкету + Claude/GPT — высокая маржа

## ❌ Минусы / Риски
- Видео слабое по содержанию — нет конкретных кейсов, только тезисы про 'выживание', идея генерируется из контекста, а не из уникального инсайта видео
- Рынок перегрет консультантами с похожими предложениями — нужна чёткая дифференциация через технологичность и автоматизацию самого аудита

## 💰 Экономика

| Параметр | Значение |
|---|---|
| Стоимость разработки | 30 000 – 60 000 руб. (шаблон аудита + автоматизация анкеты + генерация отчёта через Claude) |
| Инфраструктура/мес | 3 000 – 8 000 руб./мес (Claude API + Make.com/n8n + хостинг) |
| Поддержка/мес | 5 000 – 10 000 руб./мес |
| **Цена для клиента** | **60 000 – 120 000 руб. за аудит + дорожную карту** |
| Потенциал выручки Студии | 6 продаж в месяц × 80 000 руб. × 6 мес = ~2 880 000 руб. (реалистично при активных продажах); плюс конверсия 40% в полноценные проекты внедрения по 150–300к руб. |

## 🧰 Артефакты для реализации

**Инструменты:** Claude API (генерация отчёта), n8n или Make.com (оркестрация), Typeform / Tally (анкета), Notion / Google Docs (шаблон отчёта)
**API:** Claude API (Anthropic), OpenAI API (опционально), Notion API

## 🚀 Шаги запуска
1. Шаг 1: Разработать структурированную анкету из 30–40 вопросов по отделам (продажи, маркетинг, HR, финансы, логистика) — на выходе данные для анализа
2. Шаг 2: Настроить автоматический pipeline: анкета → n8n → Claude API → генерация отчёта с приоритетами и ROI-расчётами по каждому процессу
3. Шаг 3: Упаковать в продукт: лендинг, типовой договор, шаблон отчёта в фирменном стиле Студии, скрипт продажи
4. Шаг 4: Запустить на 5 пилотных клиентах (можно со скидкой 50%), собрать кейсы, масштабировать через партнёров и таргет
