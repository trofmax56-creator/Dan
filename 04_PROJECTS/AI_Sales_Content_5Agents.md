# 🤖×5 Система из 5 AI-агентов: отдел продаж + контент

> n8n + Claude + Bitrix24/amoCRM — пять специализированных агентов, которые работают 24/7 и экономят 30–60% времени менеджеров.

## Боль клиента

- Отдел продаж перегружен рутиной: квалификация, переписка, отчёты
- Контент для маркетинга создаётся медленно, дорого, нерегулярно
- Нет системы: каждый менеджер работает по-своему

## Пять агентов

| # | Агент | Функция | Триггер |
|---|---|---|---|
| 1 | Квалификатор | Оценивает входящие заявки по критериям ЦА | Новый лид в CRM |
| 2 | Персонализатор | Пишет персональное письмо под компанию клиента | Квалифицированный лид |
| 3 | Контент-мейкер | Генерирует посты, статьи, письма из brief | По расписанию / запросу |
| 4 | Аналитик воронки | Анализирует KPI и выявляет узкие места | Ежедневно в 9:00 |
| 5 | Реаниматор | Находит «заснувшие» сделки и готовит касание | Еженедельно |

## Технический стек

```
n8n (оркестрация всех 5 агентов)
  → Claude API 3 (claude-sonnet для анализа, haiku для генерации)
  → Bitrix24 API / amoCRM API
  → Webhook / Schedule Trigger
  → Google Sheets (логирование)
  → Telegram Bot (уведомления)
```
