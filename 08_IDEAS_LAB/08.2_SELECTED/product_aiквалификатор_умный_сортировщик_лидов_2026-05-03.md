## 1. Название
**AI-Квалификатор** — Умный Сортировщик Лидов

## 2. Суть
n8n-агент на базе Claude принимает входящие заявки через WhatsApp/Telegram, задаёт 5 квалификационных вопросов в диалоге и автоматически создаёт карточку сделки в Битрикс24/amoCRM с оценкой лида от 1 до 10.

## 3. Бизнес-сценарий
ДО: Менеджер получает 30 заявок в день из WhatsApp. Каждую нужно обзвонить, выяснить бюджет, срок, задачу — 40 минут на лид. Половина — нецелевые. ИИ делает: Клиент пишет: «Хочу автоматизацию, сколько стоит?» ИИ отвечает: «Привет! Я помогу подобрать решение. Сколько человек в отделе продаж и какую CRM используете сейчас?» Клиент отвечает — ИИ задаёт следующий вопрос про бюджет и срок. После 5 ответов ставит оценку 8/10, создаёт сделку в Битрикс24 с тегом «горячий» и уведомляет менеджера в Telegram: «Новый горячий лид — ООО Альфа, бюджет 200 тыс., срок — месяц. Карточка: [ссылка]». Клиент получает: менеджер звонит только тем, кто уже квалифицирован.

## 4. Логика
```
Wazzup Webhook → n8n (диалог-менеджер с памятью сессии) → Claude (квалификация по 5 критериям, оценка 1-10) → Битрикс24 REST API (создание сделки + тег) → Telegram Bot (уведомление РОПу)
```

## 5. Технический стек
- n8n
- claude-3-5-sonnet
- Wazzup
- Bitrix24
- Telegram Bot API

## 6. Связки инструментов
```
Wazzup → n8n Webhook → Claude API (5-шаговый диалог, оценка лида) → Bitrix24 REST API crm.deal.add → Telegram sendMessage (РОПу при score ≥ 7)
```

## 7. Алгоритм реализации
1. Подключить Wazzup Webhook к n8n, принимать входящие WhatsApp-сообщения, хранить session_id по номеру телефона в n8n Static Data
2. Настроить Claude с system prompt: роль квалификатора, 5 критериев (бюджет, срок, ЛПР, CRM, масштаб), инструкция задавать по одному вопросу
3. После 5 ответов Claude возвращает JSON: {score: 8, budget: '200k', crm: 'Битрикс24', tag: 'горячий'} — парсить через n8n Set-ноду
4. Создать сделку через Bitrix24 REST crm.deal.add с полями: TITLE, OPPORTUNITY, UF_CRM_SCORE, тег из ответа Claude, SOURCE = WhatsApp
5. Отправить уведомление в Telegram РОПу если score ≥ 7: имя клиента, бюджет, ссылка на сделку в CRM; если score < 4 — поставить задачу 'холодный nurturing'

## 8. Почему купят (ROI)
Менеджер тратит 40 минут на квалификацию одного лида вручную. При 30 заявках в месяц — это 20 рабочих часов, из которых половина уходит на нецелевые запросы. При среднем окладе менеджера 80 000 руб. — 40 000 руб./мес просто на анкетирование. AI-Квалификатор закрывает весь диалог за 3 минуты без участия человека. Крючок: «Менеджер должен продавать, а не анкетировать».
