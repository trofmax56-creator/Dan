# 🗂️ Паспорт идеи: ИИ-рекрутер для массового найма

**Источник:** [Как нанимать с помощью ИИ в 2025? Секреты найма и рекрутинга с конверсией х4 и скоростью х10](https://youtube.com/watch?v=i2PjLC-D37E) — Иван Малахов | AI (2025-05-23)
**Категория:** Замена должности
**Заменяет:** HR-рекрутер / ресёрчер (1–2 ставки)

---

## 💡 Что это даёт бизнесу
Система автоматически парсит резюме с hh.ru/SuperJob, проводит первичный скрининг через ИИ по заданным критериям и отправляет кандидатам персонализированные сообщения с приглашением на интервью. Клиент получает конверсию в 4 раза выше при скорости обработки в 10 раз быстрее ручного рекрутинга. HR-менеджер работает только с финальными кандидатами, прошедшими автоотбор.

---

## ✅ Плюсы
- Острая боль: малый бизнес тратит 40–120 ч в месяц на скрининг резюме — это прямые деньги
- Быстрый ROI для клиента: экономия 1 ставки рекрутера (50–80 тыс. руб./мес.) окупает проект за 2–3 месяца
- Масштабируемый продукт: одна кодовая база продаётся разным клиентам с минимальной кастомизацией под вакансию

## ❌ Минусы / Риски
- Зависимость от API hh.ru — платный доступ и лимиты могут ограничить автоматизацию
- Высокий риск негативной реакции кандидатов на массовые автосообщения — возможные жалобы и блокировки аккаунтов работодателя

## 🚀 Шаги запуска
1. Шаг 1: Собрать требования клиента — типовые вакансии, критерии отбора, тон коммуникации с кандидатами
2. Шаг 2: Настроить парсинг резюме через hh.ru API + автоматический скрининг через LLM с оценкой соответствия по критериям
3. Шаг 3: Подключить CRM (Bitrix24/amoCRM) для хранения воронки кандидатов и уведомлений HR через Telegram
4. Шаг 4: Запустить пилот на 1–2 вакансиях, замерить конверсию, передать клиенту дашборд с метриками и провести обучение
