# 🗂️ Паспорт идеи: ИИ-ассистент руководителя на Claude

**Источник:** [How To Use Claude As Your Executive Assistant (10-min to setup)](https://youtube.com/watch?v=6H8lqlPqCcQ) — Marketing Against the Grain (2025-01-16)
**Категория:** Замена должности
**Заменяет:** Личный помощник / Executive Assistant

---

## 💡 Что это даёт бизнесу
Владелец малого бизнеса получает персонального ИИ-ассистента, который знает контекст компании, отвечает на вопросы, готовит письма, резюмирует встречи и управляет задачами — без найма отдельного сотрудника. Настройка занимает менее дня, ассистент доступен 24/7 и не требует онбординга. Экономия на зарплате помощника от 60 000–120 000 руб./мес при стоимости внедрения от 80 000 руб.

---

## ✅ Плюсы
- Быстрый старт: минимальная техническая сложность, понятный результат уже через 1–2 дня
- Высокая воспринимаемая ценность для собственника — экономия на штатной единице очевидна и измерима
- Легко масштабировать: один шаблон продаётся 10–50 клиентам с минимальной кастомизацией

## ❌ Минусы / Риски
- Низкий барьер копирования: конкуренты могут повторить решение без привлечения Студии
- Зависимость от API Anthropic (Claude) — цены и доступность могут меняться для российских клиентов

## 🚀 Шаги запуска
1. Шаг 1: Собрать контекст клиента — описание бизнеса, частые задачи, шаблоны писем, FAQ → сформировать System Prompt для Claude
2. Шаг 2: Настроить канал взаимодействия (Telegram-бот или веб-интерфейс) через n8n/Make.com с подключением Claude API
3. Шаг 3: Интегрировать с рабочими инструментами клиента: Google Calendar, почта, CRM (Bitrix24/amoCRM) для чтения и записи задач
4. Шаг 4: Провести тест-драйв с клиентом (3–5 дней), скорректировать промпты, передать инструкцию по использованию
