# Пост: 25 нод n8n, которые нужны для 99% задач

---

Когда говорят «n8n сложный» — обычно имеют в виду: «я не знаю, какие ноды выбрать».

Вот реальность: 99% всех автоматизаций строится из одних и тех же 25 нод.

---

**Точки входа (триггеры):**
— Webhook — самая частая. Любой внешний сервис может запустить workflow
— Schedule — cron-расписание. Каждое утро в 9:00
— Email Trigger — новое письмо в ящике
— Telegram Trigger — сообщение боту

---

**Обработка данных:**
— HTTP Request — вызов любого API. Заменяет 90% встроенных нод
— Code — JavaScript прямо внутри схемы. Для сложной логики
— Set — задать/переписать значения переменных
— If / Switch — ветвление по условию
— Merge — объединить потоки данных
— Loop — повторить для каждого элемента массива

---

**Хранение и интеграции:**
— Google Sheets — универсальная база данных без SQL
— Gmail — отправить письмо
— Telegram — отправить сообщение
— Notion / Airtable — структурированные данные

---

**AI-блок:**
— HTTP Request к Claude/GPT API — это всё, что нужно
— AI Agent (встроен в n8n 1.x) — агент с инструментами
— OpenAI нода — готовая обёртка для GPT

---

**Главный инсайт:**

HTTP Request нода — это универсальный переходник. Если сервис имеет API — вы можете подключить его к n8n без специальной ноды. Bitrix24, amoCRM, СБИС, Диадок — всё через HTTP Request.
