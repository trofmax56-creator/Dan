# 🎯 AI-Квалификатор лидов в amoCRM

> Webhook amoCRM → n8n → ChatGPT/Claude → автоматическая оценка лида, простановка статуса и тегов в CRM без участия менеджера.

## Боль клиента

- Новый лид попадает в amoCRM — менеджер вручную оценивает, квалифицирован ли он
- 60–70% входящих лидов — нецелевые, но менеджер тратит время на каждого
- Разные менеджеры квалифицируют по-разному — нет стандарта

## Решение

При создании нового лида в amoCRM вебхук запускает n8n-воркфлоу: данные о лиде уходят в ChatGPT/Claude, который оценивает его по критериям идеального клиента (бюджет, индустрия, размер компании) и возвращает JSON с оценкой. n8n обновляет статус, теги и приоритет лида в amoCRM автоматически.

## Технический стек

```
amoCRM Webhook (новый лид) → n8n
  → HTTP Request: ChatGPT API (промпт с критериями ЦА)
  → JSON парсинг: оценка качества (hot/warm/cold), теги
  → HTTP Request: amoCRM API PATCH /api/v4/leads/{id}
    (обновление статуса, тегов, приоритета)
  → Telegram: уведомление менеджеру (только hot)
```

**Ключевой промпт (System):**
```
Ты — эксперт по квалификации B2B-лидов.
Оцени лида по критериям: бюджет (>500k руб), индустрия (из списка),
должность (решающий лицо). Верни JSON:
{quality: "hot|warm|cold", tags: [], priority: 1-10, reason: "..."}
```
