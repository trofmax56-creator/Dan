## 1. Название
**AI-Прораб** — Автоматизатор бизнес-процессов

## 2. Суть
Claude анализирует все отделы компании через структурированное интервью, составляет карту автоматизируемых задач с ROI-оценкой каждой и выдаёт приоритизированный план внедрения — за 20 минут вместо недельного консалтинга.

## 3. Бизнес-сценарий
ДО: Владелец кофейной сети из 7 точек жалуется: «Мои администраторы тратят полдня на сведение отчётов вручную, менеджер переписывает заявки поставщикам из таблицы в мессенджер, а я вообще не понимаю с чего начать автоматизацию.» Консультант берёт 50 000 руб. за аудит и приходит через неделю с презентацией. → ИИ делает: Клиент открывает Telegram-бота, отвечает на 15 вопросов по каждому отделу («Сколько заявок в день?», «Откуда приходят лиды?», «Как контролируете склад?»). Claude классифицирует каждый процесс по матрице «боль × сложность автоматизации», считает потенциальную экономию ФОТ. → Клиент получает: PDF-отчёт на 8 страниц: топ-5 процессов для автоматизации, ожидаемая экономия 180 000 руб./мес, готовое ТЗ для разработчика. Через 20 минут — то, за что консультант берёт 50 000 руб. и неделю.

## 4. Логика
```
Telegram-бот (вопросы по отделам) → n8n собирает ответы → Claude анализирует и ранжирует процессы → n8n генерирует PDF-отчёт → отправляет клиенту в Telegram
```

## 5. Технический стек
- n8n
- claude-sonnet-4-6
- Telegram Bot API
- PDFMonkey или HTML-to-PDF

## 6. Связки инструментов
```
Telegram Bot → n8n Webhook → Claude API (анализ + расчёт ROI) → PDFMonkey API → Telegram sendDocument
```

## 7. Алгоритм реализации
1. Создать Telegram-бота через BotFather, настроить webhook в n8n; бот задаёт 15 вопросов по 5 отделам (продажи, склад, производство, маркетинг, финансы) — по 3 вопроса на отдел
2. n8n собирает все ответы в JSON-структуру и передаёт одним запросом в Claude с системным промптом: 'Выступи экспертом по автоматизации МСБ России, оцени каждый процесс по шкале Pain 1-10 / Dev 1-10 / ROI в руб./мес'
3. Claude возвращает структурированный JSON с топ-5 процессами, экономией ФОТ, рекомендованным стеком (n8n / Bitrix24 / 1С) и сроками внедрения
4. n8n формирует HTML-шаблон отчёта, конвертирует в PDF через PDFMonkey API (или wkhtmltopdf на VPS)
5. Бот отправляет PDF клиенту + предлагает записать звонок с командой Студии Трофимов — лид квалифицирован и прогрет

## 8. Почему купят (ROI)
Боль: владелец МСБ не знает с чего начать автоматизацию — тратит 50 000–150 000 руб. на консалтинг или не делает ничего годами. 40% задач в среднем бизнесе можно убрать прямо сейчас (по данным разбора Гришина). Экономия от внедрения топ-5 процессов — в среднем 120 000–250 000 руб./мес ФОТ. Крючок: «За 20 минут в Telegram — план автоматизации, который консультант готовит неделю за 50 000 рублей.»
