## 1. Название
**AI-Коммерсант** — Генератор точных КП

## 2. Суть
Claude получает описание задачи клиента, стек технологий и риски — и за 2 минуты генерирует структурированное коммерческое предложение с точной ценой, этапами и обоснованием, которое не занижает стоимость под давлением.

## 3. Бизнес-сценарий
ДО: Менеджер или владелец студии тратит 2–3 часа на составление КП, угадывает цену «под рынок», потом смотрит на итог и понимает: «схалтурил на собственном ценнике» — взял на 30% дешевле реального объёма работ. ИИ делает: менеджер заполняет форму на 7 полей (тип задачи, стек, интеграции, количество сущностей, срок, риски, клиент), нажимает «Сгенерировать» — Claude возвращает КП через 90 секунд: структура работ, оценка в часах по каждому этапу, итоговая цена, 3 варианта пакетов (минимум / оптимум / максимум), раздел «риски и допущения». Клиент получает: профессиональный PDF-документ с логотипом, который не стыдно отправить заказчику.

## 4. Логика
```
Форма с параметрами задачи → n8n → Claude (расчёт по шаблону + обоснование цены) → Google Docs / PDF-генератор → отправка менеджеру + запись в Bitrix24
```

## 6. Связки инструментов
```
Tilda/Google Form (ввод параметров) → n8n Webhook → Claude (генерация структуры КП + расчёт стоимости) → Google Docs API (создание документа по шаблону) → Bitrix24 (прикрепление к сделке) → Telegram Bot (уведомление менеджеру со ссылкой)
```

## 7. Алгоритм реализации
1. Создать форму на 7 полей: тип задачи, технологический стек, количество интеграций, ожидаемый срок, бюджетный диапазон клиента, известные риски, имя клиента и компания
2. Написать системный промпт Claude с базой расценок Студии, нормативами часов на типовые задачи (n8n-интеграция = 8–16ч, CRM-настройка = 12–24ч) и инструкцией формировать 3 варианта пакетов
3. Запретить в промпте снижать итоговую цену при повторном запросе — фиксировать первый расчёт как финальный (инсайт из adept_ecommerce_356: первая цифра Claude — самая точная)
4. Сгенерировать Google Doc по шаблону через Docs API: вставить разделы «Состав работ», «Этапы и сроки», «Стоимость», «Риски и допущения», «Условия оплаты»
5. Прикрепить документ к сделке в Bitrix24 и отправить Telegram-уведомление менеджеру со ссылкой на КП и итоговой суммой

## 8. Почему купят (ROI)
Менеджер студии автоматизации в среднем недооценивает проект на 25–35% из-за спешки и давления клиента — при 10 проектах в месяц со средним чеком 100 000 руб. это 250 000 – 350 000 руб. недополученной выручки ежемесячно. AI-Коммерсант считает без эмоций и не даёт заднюю под давлением. Крючок: «Claude не боится назвать правильную цену — в отличие от вашего менеджера.»
