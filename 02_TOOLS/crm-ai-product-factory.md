# 🧠 Skill: CRM + AI Product Factory (v2.0)

Этот скилл превращает технические статьи и видео из папки `01_INBOX/Gold` в готовые коммерческие офферы и технические ТЗ.

## 📋 Стандарт описания продукта (9 пунктов)
Каждая сгенерированная идея ДОЛЖНА содержать:
1. **Суть:** Краткое техническое описание решения.
2. **Бизнес-сценарий:** Реальная ситуация клиента, которую решает продукт.
3. **Логика:** Путь данных (Trigger → Process → Action).
4. **Технический стек:** Список инструментов (n8n, Claude Sonnet, Whisper, API и т.д.).
5. **Связки инструментов:** Как именно сервисы общаются между собой.
6. **Алгоритм реализации:** Пошаговая инструкция для инженера n8n.
7. **Почему купят (ROI):** Экономия ФОТ в рублях, срок окупаемости.
8. **Экономика:** Разработка для Студии / инфраструктура клиента в месяц / цена продажи клиенту.
9. **Ссылка на материал:** Прямая ссылка на исходный файл в `Gold`.

## 🔍 Алгоритм отбора Gold → Продукт

### Шаг 1 — Скоринг по Матрице Востребованности
Каждый Gold-файл оценивается по трём критериям:

| Критерий | Вопрос | Диапазон |
|---|---|---|
| **Pain** | Клиент теряет деньги/время прямо сейчас без этого? | 1–10 |
| **Dev** | Сложность реализации (инверсия: 1 = легко, 10 = сложно) | 1–10 |
| **Profit** | Потенциал выручки Студии (10 = продаём за 300к+) | 1–10 |
| **ИТОГ** | Pain + (10 − Dev) + Profit | max 30 |

### Шаг 2 — Роутинг по score
```
ИТОГ > 24  → 05_BIZ_RECIPES/                  (берём в производство)
ИТОГ 20–24 → 08_IDEAS_LAB/08.2_SELECTED/      (следующий спринт)
ИТОГ 18–19 → 08_IDEAS_LAB/08.1_RAW_IDEAS/     (идея, не срочно)
ИТОГ < 18  → пропускаем
```

### Шаг 3 — Фильтры качества (до синтеза)
Отсеиваем файл если:
- нет конкретной боли клиента (абстрактный контент, «вода»)
- решение не реализуемо на n8n + Claude + CRM за 2–4 недели
- нет понятного способа продать за ≥50 000 руб.

### Шаг 4 — Синтез продукта
Заполнить все 9 пунктов стандарта по материалу из Gold-файла.
Добавить тег `#action_needed` если ИТОГ > 24.

## 🛠 Алгоритм работы скилла
1. **Сбор:** Прочитать файл(ы) из `01_INBOX/Gold`.
2. **Скоринг:** Оценить по Матрице (шаг 1–3 выше).
3. **Синтез:** Заполнить все 9 пунктов стандарта.
