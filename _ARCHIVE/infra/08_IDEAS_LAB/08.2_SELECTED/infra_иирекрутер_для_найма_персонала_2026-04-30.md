# 🗂️ Паспорт идеи: ИИ-рекрутер для найма персонала

**Источник:** [Как ИИ меняет подбор персонала | Разговор с аналитиком Т1 - HR AI](https://youtube.com/watch?v=T44phiZgfRg) — MasListo Official (2026-03-19)
**Категория:** Замена должности
**Заменяет:** HR-менеджер / рекрутер (первичный скрининг и обработка кандидатов)

---

## 💡 Что это даёт бизнесу
Автоматизирует первичный скрининг резюме, общение с кандидатами и назначение собеседований — без участия человека. Клиент получает отфильтрованный шортлист из 5–10 подходящих кандидатов вместо ручного разбора 200+ откликов. Сокращает время закрытия вакансии с 3–4 недель до 5–7 дней.

---

## ✅ Плюсы
- Острая боль у малого бизнеса: нет штатного HR, найм съедает время руководителя
- Легко упаковать в повторяемый продукт с ежемесячной подпиской
- Интегрируется в существующие CRM (Bitrix24, amoCRM) — стек Студии совпадает

## ❌ Минусы / Риски
- Российский рынок труда требует русскоязычной тонкой настройки моделей и специфических job-бордов (hh.ru, SuperJob)
- Клиенты малого бизнеса часто боятся отдавать найм 'роботу' — нужна педагогика продаж

## 🧰 Артефакты для реализации

**Инструменты:** n8n (оркестрация воронки кандидатов), Bitrix24 / amoCRM (карточки кандидатов, воронка найма), Claude API (анализ резюме, генерация вопросов), Телеграм-бот (коммуникация с кандидатами)
**API:** Anthropic Claude API (скрининг резюме), hh.ru API (парсинг вакансий и откликов), Telegram Bot API (общение с кандидатами), Bitrix24 REST API / amoCRM API

**Репозитории:**
- n8n-io/n8n (github.com/n8n-io/n8n)
- Поиск: 'AI recruiter n8n workflow' на n8n.io/workflows

**Документация:**
- docs.anthropic.com
- dev.hh.ru — документация hh.ru API
- core.telegram.org/bots/api

## 🚀 Шаги запуска
1. Шаг 1: Описать воронку найма клиента (источники резюме → скрининг → приглашение → собеседование), собрать требования к 2–3 типовым вакансиям
2. Шаг 2: Настроить автопарсинг откликов с hh.ru через API или n8n, загрузку в CRM-воронку кандидатов
3. Шаг 3: Подключить Claude для скоринга резюме по критериям вакансии + Telegram-бот для авто-переписки с кандидатами (уточнение деталей, назначение слота)
4. Шаг 4: Пилот на одной вакансии → замер метрик (время на скрининг, % прошедших в шортлист) → упаковка в тиражируемый продукт со стандартными шаблонами
