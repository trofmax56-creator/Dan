## 1. Название
**AI-Диспетчер** — Автопилот WhatsApp-продаж

## 2. Суть
Claude-агент ведёт полноценный диалог с клиентами в WhatsApp 24/7: отвечает на вопросы по продукту, выявляет потребности, обрабатывает возражения и при готовности клиента к покупке — передаёт живому менеджеру с готовым досье.

## 3. Бизнес-сценарий
ДО: Клиент пишет в WhatsApp в 22:30: «У вас есть интеграция с 1С? И сколько стоит под ключ?» Менеджер увидит сообщение завтра в 9 утра. К тому времени клиент уже договорился с конкурентом. ИИ делает: Claude отвечает через 10 секунд: «Да, интеграцию с 1С делаем — работаем с УТ, УПП и Бухгалтерией. Под ключ от 80 тысяч. Расскажите, какие процессы хотите автоматизировать — заказы, склад или финансы?» Клиент вовлекается, диалог идёт. Когда клиент говорит «окей, хочу обсудить детали» — агент фиксирует всё в Битрикс24 и пишет менеджеру: «Горячий лид Алексей, ООО СтройГрупп, хочет интеграцию 1С+CRM, бюджет ~150к, ждёт звонка». Клиент получает: мгновенный отклик в любое время, ощущение живого общения.

## 4. Логика
```
Wazzup Webhook → n8n (сессия диалога с историей) → Claude (RAG по базе знаний продуктов + промпт продавца) → Wazzup (ответ клиенту) → при триггере готовности: Bitrix24 сделка + Telegram менеджеру
```

## 5. Технический стек
- n8n
- claude-3-5-sonnet
- Wazzup
- Bitrix24
- Telegram Bot API
- Google Sheets (база знаний)

## 6. Связки инструментов
```
Wazzup Webhook → n8n (хранение истории чата в Static Data по phone_id) → Claude API (system prompt: роль менеджера + база знаний из Google Sheets) → Wazzup sendMessage → при intent='ready_to_buy': Bitrix24 crm.deal.add + Telegram уведомление
```

## 7. Алгоритм реализации
1. Настроить Wazzup Webhook в n8n; при каждом входящем сообщении загружать историю диалога (последние 20 сообщений) из n8n Static Data по ключу phone_number
2. Загрузить базу знаний компании (прайс, FAQ, кейсы) из Google Sheets в system prompt Claude при каждом запросе — обновлять автоматически каждые 6 часов через Cron
3. Claude получает: history + новое сообщение + база знаний; возвращает JSON {reply: 'текст ответа', intent: 'browsing|interested|ready_to_buy|human_needed', lead_data: {...}}
4. Отправлять reply через Wazzup API; если intent = 'human_needed' или вопрос вне компетенции — добавлять префикс '⚡ Передаю менеджеру' и уведомлять в Telegram
5. При intent = 'ready_to_buy' создавать сделку в Bitrix24 crm.deal.add с полями: имя, телефон, потребность, история диалога как комментарий; отправлять менеджеру досье в Telegram

## 8. Почему купят (ROI)
Бизнес теряет 35-40% лидов из-за долгого ответа — если не ответить в течение 5 минут, конверсия падает в 4 раза. При 50 лидах в месяц и среднем чеке 100 тыс. — это 1,5-2 млн руб. упущенной выручки ежемесячно. AI-Диспетчер отвечает за 10 секунд в любое время суток. Крючок: «Ваши конкуренты уже спят, а вы ещё продаёте».
