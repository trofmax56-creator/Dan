### 1. Описание идеи

Система автоматически обзванивает базу лидов: AI генерирует персонализированный скрипт под каждый контакт, TTS (Text-to-Speech) озвучивает его через телефонию, ответы клиентов распознаются и передаются в CRM. Горячие лиды — мгновенно к живому менеджеру.

Результат: 100+ звонков в день без одного SDR.

### 2. Технический стек

- n8n (оркестрация workflow)
- Twilio Voice API (звонки + TTS + запись) — или UIS, МТС Exolve
- Claude/GPT API (генерация скрипта + анализ ответа)
- Google Sheets (база лидов: телефон, имя, компания, контекст)
- Bitrix24 / amoCRM API (создание сделок по горячим лидам)
- Telegram Bot (алерты по горячим лидам)

### 3. Пошаговый план внедрения

**Шаг 1 — Подготовка базы лидов**
```
Google Sheets структура:
phone | name | company | context | status | call_date | result

Источники: экспорт из CRM, парсинг 2GIS, купленные базы
```

**Шаг 2 — Генерация скрипта через Claude**
```javascript
// HTTP Request → Claude API
const script = await claude({
  system: `Ты — опытный менеджер по продажам компании [Название].
  Звонишь потенциальному клиенту. Цель: назначить встречу или демо.
  Скрипт: приветствие (5 сек) → суть предложения (15 сек) → вопрос-крючок.
  Максимум 30 секунд. Естественно, без слов-паразитов.`,
  user: `Клиент: ${name}, компания: ${company}, контекст: ${context}`
})
// Вернёт: "Добрый день, ${name}! Это [Имя] из [Компания]..."
```

**Шаг 3 — Звонок через Twilio**
```javascript
// HTTP Request → Twilio API
POST https://api.twilio.com/2010-04-01/Accounts/{{AccountSid}}/Calls.json
Auth: Basic (AccountSid, AuthToken)
Body:
  To: +7{{phone}}
  From: +7{{twilio_number}}
  Twiml: <Response>
    <Say language="ru-RU" voice="Polly.Tatyana">{{script}}</Say>
    <Record maxLength="30" transcribe="true"
            transcribeCallback="{{n8n_webhook_url}}/recording"/>
  </Response>
```

**Шаг 4 — Анализ ответа клиента**
```javascript
// Когда приходит расшифровка записи:
// HTTP Request → Claude API
const analysis = await claude({
  system: "Определи реакцию клиента на звонок. Верни JSON: {interest: high|medium|low|no, reason: '...'}",
  user: transcription_text
})

// Обновить в Google Sheets: статус + результат
// Если interest === 'high': создать сделку в CRM + алерт менеджеру
```

**Шаг 5 — Расписание и лимиты**
```
Schedule Trigger: пн-пт 10:00–18:00, каждые 5 минут
Лимит: не более 20 звонков/час (во избежание блокировки)
Пауза между звонками: 30–60 сек (рандомная)
```

### 4. Сценарии использования

| Цель | Скрипт фокус | KPI |
|---|---|---|
| Назначить встречу | Уникальное предложение + вопрос | Встречи/100 звонков |
| Квалифицировать базу | Короткий вопрос о потребности | % целевых |
| Реактивация | Спецпредложение для старых клиентов | Конверсия в оплату |
| Напоминание | Статус заявки / оплаты | % успешных |

### 5. Нюансы

- Скрипт до 30 секунд — иначе клиент вешает трубку. Проверить на живых звонках
- Twilio в России требует верификации номера и может блокироваться. Альтернатива: UIS, МТС Exolve, Zadarma
- Законодательство РФ: холодные звонки разрешены, но запрещены по ночам и выходным (ФЗ о связи)
- Голос Polly.Tatyana (AWS) качественнее, чем Google TTS для русского языка
