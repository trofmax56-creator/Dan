# 🎯 Стратегия партнерского маркетинга V2: Россия + Обогащение данных

## 1. Целевая матрица партнеров (18 сегментов)

### Группа A: Digital-производство
- **Web-студии:** Разработка сайтов (Tilda, Bitrix, WP).
- **SEO-агентства:** Продвижение в поиске.
- **SMM-студии:** Ведение соцсетей.
- **PR-агентства:** Работа со СМИ и репутацией.
- **Контекстная реклама:** Директ и Google Ads.
- **Мобильная разработка:** iOS/Android приложения.

### Группа B: Бизнес-консалтинг
- **Продажники:** Бизнес-тренеры, построение отделов продаж.
- **Фин-директора:** Аутсорс финансового учета.
- **HR-консультанты:** Наем и адаптация персонала.
- **Систематизаторы:** Коучи по регламентам и процессам.
- **Налоговые консультанты:** Оптимизация налогов.

### Группа C: Бэк-офис сервисы
- **Бухгалтерия:** Аутсорсинг учета для ООО и ИП.
- **Юристы:** B2B сопровождение, договорное право.
- **Аутсорс продаж:** Колл-центры и удаленные отделы.

### Группа D: Технологические вендоры
- **Телефония:** МТС, UIS, Мегафон (региональные дилеры).
- **Чат-боты:** Разработчики простых ботов без CRM.
- **Платежные системы:** Интеграторы эквайринга.
- **BI-аналитики:** Специалисты по дашбордам.

## 2. Спецификация поиска (Яндекс-Разведчик)

### Инструмент: Firecrawl
Логика: Поиск по всей РФ с исключением конкурентов-партнеров.

**Примеры запросов для скрапинга:**
1. `site:*.ru "разработка сайтов" "для бизнеса" -site:bitrix24.ru/partners/`
2. `intitle:"бухгалтерские услуги" "прайс-лист" "Россия" -amoCRM`
3. `"бизнес-тренер" "консалтинг" "отдел продаж" -"внедрение Битрикс"`
4. `site:vk.com "руководитель digital-агентства" (Россия|Москва|Курск)`

## 3. Модуль обогащения (Data Enrichment)

### Инструмент 1: Dadata API
- **Метод:** `clean` (физические лица) — для правильного склонения имени ЛПР.
- **Метод:** `findById` (организации) — получение ИНН по названию и поиск актуальных КПП/ОГРН.

### Инструмент 2: Контур.Фокус API
- **Задача:** Фильтрация по выручке.
- **Логика:** Если выручка за прошлый год < 20 млн руб. -> Лид в "Low Priority".
- **Контакты:** Извлечение телефонов из реестров (если есть открытые данные по закупкам или лицензиям).

---

## 4. Архитектура n8n (Конвейер Агентов)

1. **Agent SCOUT:** Мониторит Firecrawl -> Собирает URL сайтов.
2. **Agent ENRICHER:** Скрапит сайт -> Ищет ИНН -> Идет в Dadata/Контур -> Заполняет CRM.
3. **Agent WRITER:** Анализирует услуги партнера -> Пишет персонализированный оффер (Вариант B/C из A/B теста).
4. **Agent SENDER:** Отправляет сообщение в Wazzup API (если есть моб. номер) или на почту.

- **План поиска:** 50 контактов в неделю.
- **Целевая конверсия:** 20% в ответ ЛПР.
