## 1. Название
**AI-Ревизор** — Контроль 100% Диалогов

## 2. Суть
Claude автоматически прослушивает и читает 100% диалогов менеджеров с клиентами (записи звонков + переписка WhatsApp), выставляет оценку по скрипту продаж и формирует персональный отчёт по каждому менеджеру с конкретными ошибками — без ОКК-отдела.

## 3. Бизнес-сценарий
ДО: РОП Сергей раз в неделю вручную слушает 5 звонков из 200 — это 2% контроля. Менеджер Антон систематически не закрывает на следующий шаг, но Сергей об этом не знает — видит только результат в цифрах продаж. ИИ делает: каждый звонок через UIS автоматически транскрибируется, текст уходит в n8n → Claude оценивает по 8 критериям скрипта (приветствие, выявление потребности, презентация, работа с возражением, закрытие на следующий шаг) → выставляет оценку 1–5 по каждому → если итоговый балл < 3, РОП получает уведомление в Telegram: «Антон, звонок 14:32, балл 2.1/5 — не выявил бюджет, не предложил следующий шаг, клиент ушёл думать». Клиент получает: еженедельный автоотчёт по каждому менеджеру с топ-3 слабых мест и динамикой улучшения.

## 4. Логика
```
UIS/МТС Exolve webhook (завершение звонка) → n8n → транскрибация (Whisper API) → Claude (оценка по скрипту, JSON с баллами) → IF балл < 3 → Telegram уведомление РОПу → еженедельный отчёт в Google Sheets
```


## 7. Алгоритм реализации
1. Подключить UIS или МТС Exolve к n8n webhook: при завершении звонка передавать запись MP3 и метаданные (менеджер, клиент, длительность, номер телефона)
2. Настроить транскрибацию через OpenAI Whisper API — разделение на спикеров (диаризация), передача текста как JSON с ролями «менеджер» и «клиент»
3. Написать Claude-промпт с встроенным скриптом продаж клиента: оценить 8 критериев по шкале 1–5, вернуть структурированный JSON с баллами, цитатами нарушений и рекомендациями
4. Настроить IF-ноду: при среднем балле < 3 — немедленный алерт в Telegram РОПу с цитатой и временной меткой; при балле 3–4 — добавить в еженедельный дайджест
5. Формировать еженедельный отчёт через Google Sheets API: топ-3 ошибки по каждому менеджеру, динамика среднего балла за 4 недели, лидерборд команды

## 8. Почему купят (ROI)
Отдел ОКК из 2 человек стоит 160 000–200 000 руб./мес ФОТ и всё равно слушает только 5–10% звонков. AI-Ревизор проверяет 100% за 3 000–5 000 руб./мес на API. Рост конверсии на 15–25% при системном контроле скрипта подтверждён в 80% внедрений. Крючок: «Вы платите ОКК за 5% контроля. Мы даём вам 100% за цену одного обеда».
